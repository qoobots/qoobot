## 项目地图（8 大子项目）

### 核心层

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qoobody](./qoobody/)** | 硬件本体 | 📱 iPhone 硬件 | 传感器、执行器、计算平台、机械结构、能源系统 |
| **[qoobrain](./qoobrain/)** | 大脑操作系统 | ⚙️ iOS | 感知、认知、决策、规划、运动控制、通信 |
| **[qoocore](./qoocore/)** | 芯片与加速框架 | 🧠 Apple Silicon + Core ML | 端侧推理运行时、模型编译、硬件加速 |

### 平台层

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qooeco](./qooeco/)** | 应用生态 | 🏪 App Store | 技能市场、开发者工具、第三方算法集成 |
| **[qoocloud](./qoocloud/)** | 云端服务 | ☁️ iCloud | 云端推理、数据同步、远程协作、多机管理 |
| **[qoosvc](./qoosvc/)** | 系统服务 | 🎙️ Siri + MapKit + ARKit | 语音助手、导航、空间理解、多机互联、自诊断 |

### 保障层

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qoocode](./qoocode/)** | 开发者工具链 | 🛠️ Xcode | IDE、仿真器、调试器、性能剖析、数据标注 |
| **[qooauth](./qooauth/)** | 账号与安全 | 🔐 Apple ID + Secure Enclave | 统一身份、设备认证、权限管理、隐私合规 |

## 依赖关系

```
qooauth ──► 所有服务（认证/授权/加密）

qoocode ──► 开发/调试/部署 qoobrain + qoobody

qoocore ──► qoobrain（模型推理加速）
        ──► qoosvc（视觉/语音推理加速）

qoosvc  ──► qoobrain（提供系统服务 API）

qooeco  ──► qoobrain SDK（技能开发）
        ──► qoocloud（技能分发）

qoocloud ──► qoobrain（云端推理/同步）
         ──► qooeco（技能市场后端）
```

## 快速开始

```bash
# 大脑操作系统（目前唯一可运行的项目）
cd qoobrain
pip install -e brain_sdk/ -e brain_ai/
python -m pytest brain_ai/tests brain_sdk/tests -v
```
