## 项目地图（12 子项目）

### 🧬 核心层 — 机器人本体能力

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qoobody](./qoobody/)** | 硬件本体 | 📱 iPhone 硬件 | 传感器、执行器、计算平台、机械结构、能源 |
| **[qoobrain](./qoobrain/)** | 大脑操作系统 | ⚙️ iOS | 感知、认知、决策、规划、运动控制、通信 |
| **[qoocore](./qoocore/)** | 芯片与加速 | 🧠 Apple Silicon + Core ML | 端侧推理、模型编译、硬件加速 |

### 🔌 平台层 — 开发者与用户服务

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qooeco](./qooeco/)** | 应用生态 | 🏪 App Store | 技能市场、开发者工具、第三方算法集成 |
| **[qoocloud](./qoocloud/)** | 云端服务 | ☁️ iCloud | 云端推理、数据同步、远程协作、多机管理 |
| **[qoosvc](./qoosvc/)** | 系统服务 | 🎙️ Siri + MapKit + ARKit | 语音助手、导航、空间理解、多机互联 |

### 🛡️ 保障层 — 质量与安全

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qoocode](./qoocode/)** | 开发者工具 | 🛠️ Xcode | IDE、仿真器、调试器、性能剖析、数据标注 |
| **[qooauth](./qooauth/)** | 账号与安全 | 🔐 Apple ID + Secure Enclave | 身份认证、权限管理、隐私合规 |
| **[qooregs](./qooregs/)** | 法规合规 | 📋 FCC/CE/全球准入 | 安全标准、无线认证、隐私法规、出口管制 |

### 🌍 生态层 — 产业与社区

| 项目 | 定位 | iPhone 类比 | 说明 |
|------|------|-------------|------|
| **[qoogear](./qoogear/)** | 配件生态 | 🎧 AirPods + MFi | 第三方配件认证、末端执行器、可穿戴设备 |
| **[qoocommunity](./qoocommunity/)** | 品牌社区 | 🎪 WWDC + 开发者关系 | 年度大会、开发者计划、高校合作 |
| **[qoochain](./qoochain/)** | 供应链制造 | 🏭 富士康 + 全球供应链 | 生产标准、出厂标定、质量检测、BOM 设计 |

## 依赖关系

```
qooauth ────► 所有服务（认证/授权/加密）
qooregs ────► qoobody + qoobrain + qoocloud（合规约束）

qoocore ────► qoobrain（模型推理加速）
          ──► qoosvc（视觉/语音推理加速）

qoosvc  ────► qoobrain（系统服务 API）

qoogear ────► qoobody（配件 HAL 接口）
         ──► qooeco（配件技能分发）

qooeco  ────► qoobrain SDK（技能开发）
         ──► qoocloud（技能市场后端）

qoocloud ────► qoobrain（云端推理/同步）
           ──► qooeco（市场后端）
           ──► qoochain（设备追溯）

qoocode ────► qoobrain + qoobody（开发/调试/部署）

qoocommunity ► 所有项目（人才培养、生态运营）
qoochain ────► qoobody（生产制造）
```
