## 二、技术栈总结

| 组件 | 技术 | 端口 |
|------|------|------|
| qoostore-cloud | Java 21 + Spring Boot 3.x + Spring Cloud | 8209 |
| qoostore-edge | C++17 + Python 3.11 | 端侧本地 |
| 数据库 | PostgreSQL 16 | 5432 |
| 缓存 | Redis 7 | 6379 |
| 对象存储 | MinIO | 9000 |
| 消息队列 | RocketMQ | 9876 |
| 搜索引擎 | Elasticsearch 8 | 9200 |
| 服务注册 | Nacos | 8848 |
