# qoostore — 交互设计

> 最后更新：2026-06-28

---

## 一、用户角色

| 角色 | 描述 | 交互界面 |
|------|------|----------|
| 普通用户 | 浏览/搜索/购买/安装技能 | Web Portal / Mobile App / Mini Program |
| 开发者 | 上传技能/查看分析/管理收益 | Web Portal / qoodev CLI |
| 审核员 | 审核技能提交/处理举报 | Web Portal (管理后台) |
| 运营管理员 | 编辑精选/管理分类/查看报表 | Web Portal (管理后台) |
| 机器人 | 自动安装/更新/卸载技能 | qoostore-edge (端侧 Agent) |

## 三、技能包格式 (.qooskills)

```
skill-name-v1.2.3.qooskills  (ZIP 格式)
│
├── manifest.json           # 技能元数据
│   {
│     "skillId": "com.example.cleaning",
│     "name": "智能清洁",
│     "version": "1.2.3",
│     "minQOSVersion": "0.3.0",
│     "category": "home",
│     "permissions": ["camera", "motor.arm"],
│     "sandboxLevel": "restricted",
│     "entryPoint": "main.py",
│     "runtime": "python3.11",
│     "maxMemoryMB": 512,
│     "maxCPUPercent": 30
│   }
│
├── skill/
│   ├── main.py             # 技能入口
│   ├── models/             # AI 模型文件
│   ├── config/             # 配置文件
│   └── resources/          # 资源文件
│
├── icons/                  # 图标资源
│   ├── icon_512.png
│   └── icon_128.png
│
├── screenshots/            # 截图
│   ├── screenshot_1.png
│   └── screenshot_2.png
│
├── privacy_label.json      # 隐私标签
└── signature.sig           # 开发者签名
```

## 五、端侧 CLI 命令 (qoodev 集成)

```bash
# 开发者上传技能
qoodev store publish ./my-skill

# 查看审核状态
qoodev store status

# 查看收益
qoodev store revenue --period month

# 用户安装技能
qoodev store install com.example.cleaning

# 用户搜索技能
qoodev store search "清洁"

# 查看已安装技能
qoodev store list

# 卸载技能
qoodev store uninstall com.example.cleaning
```
