# qooremote 交互设计

## 1. 用户角色

| 角色 | 描述 | 权限 |
|------|------|------|
| 主操作员 (Primary Operator) | 拥有完全控制权 | 接管/交还/急停/参数调整/会话管理 |
| 观察员 (Observer) | 仅查看权限 | 视频查看/状态监控/语音对讲 |
| 示教员 (Instructor) | 示教录制权限 | 运动控制+数据录制+技能创建 |
| 管理员 (Admin) | 系统管理权限 | 所有权限+操作员管理+审计 |

### 3.2 输入设备映射

| 输入设备 | 映射到机器人 | 交互方式 |
|----------|-------------|----------|
| 动捕手套/服 | 全身关节角度 | 1:1 运动映射 |
| VR 手柄 (左手) | 基座移动 (x/y/θ) | 摇杆/触控板 |
| VR 手柄 (右手) | 手臂末端位姿 | 6-DOF 跟踪 |
| VR 头显 | 头部/云台方向 | 头部追踪 |
| 键盘 WASD | 基座移动 | 按键 |
| 键盘 Q/E | 基座旋转 | 按键 |
| 键盘 方向键 | 头部控制 | 按键 |
| 键盘 Space | 紧急停止 | 按键 |
| 游戏手柄 左摇杆 | 基座移动 | 摇杆 |
| 游戏手柄 右摇杆 | 头部/手臂 | 摇杆 |
| 游戏手柄 L1/R1 | 末端开/合 | 扳机 |

## 4. 移动端遥控 (Mobile App)

移动端提供精简版遥控面板，支持：
- 单路视频回传 + 触摸摇杆控制
- 语音对讲（对讲机模式）
- 急停按钮
- 状态监控（电池/温度/通信）

## 5. 桌面端遥控 (Desktop Console)

桌面端提供完整遥控能力：
- 多路视频同时显示（最多 4 路）
- 动捕/VR 设备接入
- 示教录制与回放
- 3D 机器人模型实时姿态显示
- 操作日志与分析

## 6. 异常处理交互

| 异常场景 | 系统行为 | 操作员提示 |
|----------|----------|------------|
| 网络延迟 >500ms | 自动降速至安全速度 | 🔴 高延迟警告 + 延迟数值 |
| 网络断开 | 机器人自动 STOP | 🔴 连接丢失 + 倒计时重连 |
| 机器人碰撞检测 | 自动 ProtectiveStop | 🟡 碰撞告警 + 位置信息 |
| 机器人急停触发 | 立即 EmergencyStop | 🔴 急停触发 + 原因 |
| 电池低电量 (<15%) | 警告 + 建议结束遥控 | 🟡 低电量警告 |
| 电机过热 | 限制功率输出 | 🟡 温度警告 + 建议降速 |
