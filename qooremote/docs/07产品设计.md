## 1. 产品定位

qooremote 是 QooBot 的**人类兜底安全网**和**技能示教系统**。它使人类能够在机器人自主智能无法胜任时远程接管，同时采集的操作数据可反哺 AI 训练，形成"自主→人工→学习→更自主"的飞轮。

## 4. 核心功能流程

### 4.1 远程接管流程

```
机器人: 遇到障碍/长尾场景
    │
    ▼
qoobrain: 置信度低于阈值
    │
    ▼
发起接管请求 → 通知操作员
    │
    ▼
操作员: 查看视频/传感器 → 确认接管
    │
    ▼
建立 WebRTC 连接 (P2P 优先，TURN 兜底)
    │
    ▼
人工遥控运行
    │
    ├── 场景解决 → 释放控制 → 恢复自主
    └── 断线/超时 → 安全停止 → 等待重连/自主恢复
```

### 4.2 示教流程

```
操作员执行任务 (遥控)
    │
    ▼
录制器采集多模态数据:
  · 关节轨迹 (100Hz)
  · 视频流 (30fps)
  · 力/触觉 (100Hz)
  · 操作员指令 (事件)
    │
    ▼
数据自动标注 + 分段
    │
    ▼
上传到 qoocloud 训练管道
    │
    ▼
更新模型 → OTA 下发
```

## 5. 产品路线图

| 阶段 | 重点 | 里程碑 |
|:-----|------|------|
| 🚀 MVP | Web Portal + 手柄操控 + 单路视频 | 100ms 内操控延迟 |
| 📦 增强 | VR 客户端 + 多路视频 + 示教录制 | 动捕遥操作 |
| 🎯 完整 | 力反馈 + 3D场景重建 + 多操作员 | 触觉临场感 |
