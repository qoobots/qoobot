# qooremote — 远程遥控子系统 功能清单完成进度

> 最后更新：2026-06-28 | **✅ v0.3 全部完成 (10/10)**

## 项目概述

qooremote 是 QOS 两大核心子系统之一（与 qoobrain 并列），负责**远程人工遥控**。当自主大脑无法应对长尾场景时，人类操作员可远程接管机器人的感知、决策与控制。

**对标 Android**：无障碍服务 (Accessibility Service) + 设备管理 (Device Policy) + 远程桌面

**技术栈**：Java 17 + Spring Boot 3 (cloud_service) | C++17 (onboard_core) | TypeScript + Vue3 (web_portal) | Protobuf (协议)

**平台覆盖**：🖥️ onboard_core | ☁️ cloud_service | 🖱️ desktop_console | 🌐 web_portal | 📱 mobile_app

## 技术实现概要

| 属性 | 值 |
|------|------|
| 架构模式 | 跨子项目架构概念，代码分布在 qoosvc/qoocloud/qooweb/qoobody |
| 端侧 C++ 栈 | C++17 + libdatachannel (WebRTC) + FFmpeg (视频编码) + Protobuf |
| 云端 Java 栈 | Java 17 + Spring Boot 3 + WebRTC (mediasoup) |
| 前端栈 | TypeScript + Vue3 + Element Plus + WebRTC API |
| 协议定义 | `qoocloud/proto/teleop/` (Protobuf v3) |
| 视频编码 | H.264/H.265 硬件编码 (VAAPI/NVENC/VideoToolbox) |
| 音频编码 | Opus 编码 + WebRTC AEC/ANS/AGC |
| 信令协议 | WebRTC DataChannel + 自定义控制协议 |
| 会话管理 | 基于 Redis 的会话状态存储 + 心跳检测 |
| API 端点 | `/api/v1/teleop/*` (18+ 个端点) |
