# qoobot-desktop — 桌面软件套件

> QooBot 生态的桌面端工具集，包含开发者工具与远程遥控控制台。

本目录包含两个独立的桌面应用子项目：

| 子项目 | 定位 | 用户 | 优先级 |
|--------|------|------|--------|
| [qoodev](./qoodev) | 仿生人开发者工具链 | 第三方开发者 | P0 |
| [qooremote](./qooremote) | 远程机器人监控遥控工具 | 操作员 / 机器人管理者 | P1 |

## 子项目关系

```
                    ┌─────────────────────────┐
                    │    qoobot-desktop       │
                    │                         │
  qoodev ──────────►│  开发/调试/打包/发布      │────── qoobrain (大脑 OS)
                    │  仿真/性能剖析/数据标注   │────── qoobody (硬件)
                    │                         │────── qoostore (技能市场)
                    │                         │
  qooremote ───────►│  远程操控/实时监控        │────── qoobrain (接收指令)
                    │  视频回传/语音对讲        │────── qoosvc (系统服务)
                    │  示教录制/动捕遥操作      │────── qoocloud (信令中继)
                    └─────────────────────────┘
```

## 技术栈

| 组件 | 技术 |
|------|------|
| CLI / 工具 | Python 3.11+ |
| GUI 控制台 | PySide6 |
| LSP 服务器 | Python |
| IDE 插件 | TypeScript (VS Code Extension API) |
