# qooremote — 远程机器人监控遥控工具

> 人在回路：当自主大脑无法应对长尾场景时，人类操作员可远程接管机器人的感知、决策与控制。
>
> **对标**：Tesla Optimus 遥控示教 + Boston Dynamics 远程操作台。
>
> **当前版本**：v0.1.0

## 定位

qooremote 是 QooBot 生态的远程遥控控制台，提供从视频回传、遥操作控制、
语音对讲到示教录制的一站式远程操作体验。它是机器人管理员的日常操作入口。

**依赖**：qoobrain（接收控制指令）、qoosvc（系统状态服务）、qoocloud（信令中继）
**被依赖**：无

## 核心能力

| 能力 | 说明 | 平台 |
|------|------|------|
| 🎥 视频回传 | 多路摄像头实时低延迟回传 | desktop + web + mobile |
| 🎮 遥操作控制 | 动捕/VR/手柄三种操控方式 | desktop |
| 🎤 语音对讲 | 双向实时语音通信 | desktop + mobile |
| 🖐️ 远程接管 | 半自主→全手动模式无缝切换 | desktop + web |
| 📝 示教录制 | 操作数据采集 → 训练数据生成 | desktop |
| 📊 实时仪表盘 | 机器人状态/传感器/电量的可视化监控 | desktop + web + mobile |
| 🗺️ 3D 数字孪生 | 机器人实时 3D 姿态同步显示 | desktop + web |
| 🔔 告警通知 | 异常事件实时推送 | desktop + web + mobile |

## 平台覆盖

| 平台 | 技术栈 | 说明 |
|------|--------|------|
| 🖥️ desktop_console | Python / PySide6 | 主操作台，支持动捕/VR/手柄 |
| 🌐 web_portal | TypeScript / Vue 3 / Element Plus | 浏览器端监控面板 |
| 📱 mobile_app | TypeScript / Vue 3 / UniApp | 移动端监控与对讲 |
