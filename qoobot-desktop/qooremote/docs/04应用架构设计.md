### 2.2 模块职责

| 模块 | 层 | 职责 |
|------|:--:|------|
| MainWindow | UI | 主窗口框架、菜单栏、工具栏、工作区布局管理 |
| VideoPanel | UI | 多路视频渲染（OpenGL 纹理）、布局切换（1/4/9 宫格）、截图 |
| DashPanel | UI | 仪表盘组件容器、自定义布局拖拽、数据图表 |
| CtrlPanel | UI | 操控模式选择、手柄设置、键盘映射配置 |
| VoicePanel | UI | PTT 按钮、音量指示器、通话状态 |
| RobotVM | ViewModel | 机器人状态数据绑定、3D 姿态模型驱动 |
| VideoVM | ViewModel | 视频流生命周期管理、码率/分辨率切换 |
| TeleopVM | ViewModel | 输入设备→控制指令映射、指令平滑与限幅 |
| RecordingVM | ViewModel | 录制状态管理、回放控制、数据导出 |
| SignalingClient | Service | WebSocket 连接管理、心跳、信令收发 |
| WebRTCManager | Service | PeerConnection 管理、音视频轨管理、ICE/STUN/TURN |
| InputManager | Service | 手柄/键盘/鼠标输入事件采集与标准化 |
| RecordingService | Service | 多通道数据同步录制、HDF5/SQLite 存储 |

## 3. Web 监控面板架构 (web_portal)

### 3.1 前端技术栈

| 层级 | 技术 | 版本 |
|------|------|:----:|
| 框架 | Vue 3 (Composition API) | 3.4+ |
| UI 库 | Element Plus | 2.5+ |
| 状态管理 | Pinia | 2.1+ |
| 路由 | Vue Router | 4.3+ |
| 3D 渲染 | Three.js | 0.160+ |
| 图表 | ECharts | 5.5+ |
| WebSocket | 原生 WebSocket + 自动重连 |
| WebRTC | 浏览器原生 RTCPeerConnection |
| 构建 | Vite | 5+ |
| 语言 | TypeScript | 5.3+ |

### 3.2 页面路由

```
/                          → 机器人列表页 (RobotList)
/robot/:id                 → 机器人监控详情页 (RobotDetail)
/robot/:id/video           → 视频查看页 (VideoView)
/robot/:id/3d              → 3D 姿态查看页 (3DView)
/alerts                    → 告警中心 (AlertCenter)
/settings                  → 设置页 (Settings)
```

## 5. 跨平台共享层

### 5.1 共享核心模块

位于 `qooremote/core/` 下的 Python SDK，提供桌面端使用（Web/移动端使用独立 TypeScript 实现相同的协议交互）：

```
qooremote/
└── console/
    └── core/                     # 核心共享层
        ├── signaling/            # WebSocket 信令客户端
        │   ├── client.py         # 连接/重连/心跳/消息收发
        │   ├── messages.py       # 信令消息定义 (JSON Schema)
        │   └── heartbeat.py      # 心跳管理
        ├── webrtc/               # WebRTC 管理
        │   ├── peer.py           # PeerConnection 封装
        │   ├── video_track.py    # 视频轨管理
        │   ├── audio_track.py    # 音频轨管理
        │   └── data_channel.py   # DataChannel 封装
        ├── teleop/               # 遥操作核心
        │   ├── controller.py     # 控制指令基类
        │   ├── gamepad.py        # 手柄映射
        │   ├── keyboard.py       # 键盘映射
        │   ├── mocap.py          # 动捕接口
        │   └── vr.py             # VR 接口
        ├── recording/            # 录制核心
        │   ├── recorder.py       # 多通道录制器
        │   ├── player.py         # 回放器
        │   └── exporter.py       # .qoodata 导出
        ├── models/               # 数据模型
        │   ├── robot_state.py    # 机器人状态模型
        │   ├── joint_state.py    # 关节状态
        │   ├── sensor_data.py    # 传感器数据
        │   └── alert.py          # 告警模型
        └── utils/                # 工具类
            ├── units.py          # 单位转换
            ├── interpolation.py  # 插值/平滑
            └── timer.py          # 高精度计时
```

## 6. 通信架构

### 6.1 协议分层

```
┌────────────────────────────────────────────┐
│              应用层协议                      │
│  · 控制指令 (JSON over DataChannel)         │
│  · 状态上报 (JSON over DataChannel)         │
│  · 告警通知 (JSON over DataChannel)         │
├────────────────────────────────────────────┤
│              传输层协议                      │
│  · 信令: WebSocket (wss://)                 │
│  · 媒体: WebRTC (SRTP/SCTP)                │
│  · 数据: DataChannel (SCTP over DTLS)      │
├────────────────────────────────────────────┤
│              安全层                         │
│  · TLS 1.3 加密                            │
│  · JWT Token 认证                          │
│  · mTLS 双向证书 (可选)                     │
└────────────────────────────────────────────┘
```

### 6.2 连接生命周期

```
操作端 (qooremote)              云中继 (qoocloud)              机器人端 (qoobot)
      │                              │                              │
      │  ① WebSocket Connect         │                              │
      │─────────────────────────────►│                              │
      │  ② Auth (JWT Token)         │                              │
      │─────────────────────────────►│                              │
      │  ③ List Robots              │                              │
      │─────────────────────────────►│                              │
      │  ④ Select Robot             │                              │
      │─────────────────────────────►│  ⑤ Relay Connect Request    │
      │                              │─────────────────────────────►│
      │                              │  ⑥ Robot Accept             │
      │                              │◄─────────────────────────────│
      │  ⑦ Session Established      │                              │
      │◄─────────────────────────────│                              │
      │                              │                              │
      │  ⑧ WebRTC Offer (SDP)       │                              │
      │─────────────────────────────►│─────────────────────────────►│
      │  ⑨ WebRTC Answer (SDP)      │                              │
      │◄─────────────────────────────│◄─────────────────────────────│
      │                              │                              │
      │  ⑩ ICE Candidate Exchange   │                              │
      │◄────────────────────────────►│◄────────────────────────────►│
      │                              │                              │
      │ ════════ P2P WebRTC Connection Established ═══════════      │
      │  · Video Track               │                              │
      │  · Audio Track               │                              │
      │  · DataChannel (Control)     │                              │
      │  · DataChannel (State)       │                              │
```
