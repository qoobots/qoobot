# 05 — 交互设计

> 版本：v0.1.0 | 最后更新：2026-06-30 | 状态：Draft
> 子项目：qooremote（远程机器人监控遥控工具）

---

## 1. 交互设计原则

| 原则 | 说明 | 设计体现 |
|------|------|----------|
| 低认知负荷 | 操作员注意力应集中在机器人任务上，而非 UI | 减少弹窗、自动化常用操作 |
| 即时反馈 | 任何操作必须有 < 100ms 的视觉/触觉反馈 | 按钮按下效果、状态变化动画 |
| 错误预防 | 关键操作需确认，不产生不可逆后果 | 紧急制动除外（一键触发） |
| 一致可预测 | 跨平台相同的操作产生相同的结果 | 统一快捷键、统一手势 |
| 无障碍 | 支持键盘完整操作，色盲友好配色 | 高对比度主题、键盘导航 |

## 2. 全局交互规范

### 2.1 连接状态指示

```
🟢 Online        信号良好，延迟 < 100ms，全功能可用
🟡 Degraded      信号中等，延迟 100-500ms，视频降质
🔴 Offline       断开连接，所有控制功能锁定
⚪ Connecting    正在建立连接中
```

| 状态 | 图标 | 颜色 | 可用操作 |
|------|:----:|------|----------|
| Online | 🟢 | #22C55E | 全功能 |
| Degraded | 🟡 | #EAB308 | 全功能（视频自动降质提示） |
| Offline | 🔴 | #EF4444 | 仅查看历史数据、连接操作 |
| Connecting | ⚪ | #94A3B8 | 无操作，显示加载动画 |

### 2.2 告警分级

| 级别 | 颜色 | 图标 | 通知方式 | 示例 |
|:----:|------|:----:|----------|------|
| 🔴 紧急 | #EF4444 | ⚠️ | 弹窗 + 声音 + 闪烁 | 碰撞检测、紧急制动触发 |
| 🟠 严重 | #F97316 | ⚡ | 横幅 + 声音 | 电机过温、通信中断 |
| 🟡 警告 | #EAB308 | 📢 | 通知栏 | 电量低于20%、高延迟 |
| 🔵 信息 | #3B82F6 | ℹ️ | 静默记录 | 任务完成、模式切换 |

### 2.3 快捷键体系

| 快捷键 | 功能 | 作用域 |
|--------|------|--------|
| `Ctrl+N` | 新建连接 | 全局 |
| `Ctrl+D` | 断开连接 | 全局 |
| `Space` | 紧急制动（需二次确认） | 全局 |
| `F11` | 全屏切换 | 全局 |
| `Ctrl+R` | 开始/停止录制 | 连接状态下 |
| `Ctrl+M` | 标记录制点 | 录制中 |
| `Tab` | 视频布局切换 | 视频面板聚焦时 |
| `1-9` | 切换摄像头1-9 | 视频面板聚焦时 |
| `W/A/S/D` | 机器人移动控制 | 手动操控模式下 |
| `Q/E` | 机器人旋转 | 手动操控模式下 |
| `↑/↓/←/→` | 末端执行器平移 | 末端控制模式下 |
| `PgUp/PgDn` | 末端执行器升降 | 末端控制模式下 |
| `T` | PTT 语音（按住说话） | 连接状态下 |

## 6. 异常状态交互

### 6.1 连接中断

```
场景: WebSocket / WebRTC 连接意外中断

交互流程:
1. 所有操控功能立即锁定（变灰不可操作）
2. 视频画面冻结，叠加半透明遮罩 + "连接中断"文字
3. 状态指示变为 🔴 Offline
4. 自动开始重连倒计时: "正在重连... (3/2/1)"
5. 重连成功后自动恢复之前状态（视频流/操控模式）
6. 30s 内未重连成功，弹窗提示 "无法连接到机器人，是否重试？"
   [重试] [切换机器人] [离线查看]
```

### 6.2 紧急制动

```
场景: 操作员触发紧急制动（Space 键 / 紧急制动按钮）

交互流程:
1. 屏幕四角红色闪烁边框（0.5s）
2. 中央显示 "⚠️ 紧急制动已激活" 
3. 状态栏显示 "🛑 紧急制动" 红色文字
4. 所有控制指令停止发送
5. 解除: 需点击 "解除制动" 按钮并确认
6. 制动操作完整记录到审计日志
```

### 6.3 低电量警告

```
场景: 机器人电量降至 20% 以下

交互流程:
1. 电量图标变为橙色闪烁
2. 顶部横幅提示 "⚠️ 电量过低 (18%)，预估剩余 12 分钟"
3. 视频码率自动降至最低
4. 电量降至 10%: 红色弹窗 "🔴 电量严重不足，建议立即返航充电"
5. 电量降至 5%: 自动触发安全待机模式
```
