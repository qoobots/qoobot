> 版本：v0.1.0 | 最后更新：2026-06-30 | 状态：Draft

## 1. 数据域总览

qooremote 涉及五大数据域：

```
┌─────────────────────────────────────────────────┐
│                  qooremote 数据域                  │
│                                                  │
│  ┌───────────┐ ┌───────────┐ ┌───────────┐      │
│  │ 机器人状态 │ │ 控制指令  │ │  媒体流   │       │
│  │ Robot     │ │ Command   │ │ Media     │       │
│  │ State     │ │           │ │ Stream    │       │
│  └─────┬─────┘ └─────┬─────┘ └─────┬─────┘      │
│        │             │             │            │
│  ┌─────┴─────────────┴─────────────┴─────┐      │
│  │           录制数据 (Recording)         │      │
│  └──────────────────┬───────────────────┘      │
│                     │                           │
│  ┌──────────────────┴───────────────────┐      │
│  │           配置与偏好 (Config)          │      │
│  └──────────────────────────────────────┘      │
└─────────────────────────────────────────────────┘
```

对于高频变化数据（关节角度等），使用增量更新减少带宽：

```json
{
  "type": "joint_delta",
  "timestamp": 1719700000050,
  "updates": [
    {"id": 0, "p": 0.525, "v": 0.15},
    {"id": 1, "p": -0.312, "v": -0.08}
  ]
}
```

## 4. 媒体流数据

### 4.1 视频流配置

| 参数 | 默认值 | 可选值 |
|------|--------|--------|
| 编码格式 | H.264 | H.264 / H.265 / VP9 / AV1 |
| 分辨率 | 1280×720 | 640×480 / 1280×720 / 1920×1080 |
| 帧率 | 30 fps | 15 / 20 / 30 |
| 码率 | 2 Mbps | 0.5-8 Mbps (自适应) |
| 关键帧间隔 | 2s | 1-5s |
| 传输协议 | WebRTC (SRTP) | WebRTC / RTSP |

### 4.2 音频流配置

| 参数 | 默认值 | 可选值 |
|------|--------|--------|
| 编码格式 | Opus | Opus / G.711 |
| 采样率 | 48000 Hz | 8000 / 16000 / 48000 |
| 声道 | 单声道 | 单声道 / 立体声 |
| 码率 | 32 kbps | 16-128 kbps |
| 帧长 | 20ms | 10 / 20 / 40 / 60 ms |

## 5. 录制数据 (Recording)

### 5.1 .qoorlog 录制格式

采用 HDF5 容器，多通道同步存储：

```
recording_20260630_143022.qoorlog (HDF5)
├── /metadata/
│   ├── version: "1.0"
│   ├── robot_id: "qoobot-01"
│   ├── operator_id: "op-42"
│   ├── start_time: 1719700222000
│   ├── end_time: 1719700552000
│   ├── duration_ms: 330000
│   ├── sample_rate_hz: 100
│   └── channels: ["joints", "imu", "forces", "video_0", "control"]
├── /joints/                      # shape: (N, 28)
│   ├── positions
│   ├── velocities
│   └── torques
├── /imu/                         # shape: (N, 9)
│   ├── orientation
│   ├── angular_velocity
│   └── linear_acceleration
├── /forces/                      # shape: (N, 4)
│   └── values
├── /control/                     # shape: (N, 7)
│   ├── mode
│   ├── target_pose
│   └── gripper_command
├── /video_0/                     # H.264 编码视频
│   └── frames
├── /events/                      # 标记点
│   ├── timestamps
│   └── labels
└── /timestamps/                  # 统一时间戳 (微秒)
    └── utc_us
```

```yaml
connections:
  - id: "default"
    name: "本地机器人"
    cloud_url: "wss://cloud.qoobot.dev/signaling"
    token: "eyJ..."

  - id: "lab-02"
    name: "实验室机器人 02"
    cloud_url: "wss://lab.qoobot.local/signaling"
    token: "eyJ..."
```

### 6.2 用户偏好

```yaml
# $HOME/.qooremote/preferences.yaml
ui:
  theme: "dark"                    # dark | light
  language: "zh-CN"               # zh-CN | en-US
  layout_preset: "standard"       # standard | teleop | recording | immersive

video:
  default_resolution: "1280x720"
  default_fps: 30
  adaptive_bitrate: true
  max_bitrate_mbps: 4

teleop:
  control_mode: "end_effector"    # end_effector | joint | gamepad
  gamepad_mapping: "xbox_default"
  sensitivity: 0.7                # 0.1 - 1.0
  deadzone: 0.05                  # 摇杆死区
  smoothing_factor: 0.3           # 指令平滑系数

voice:
  input_device: "default"
  output_device: "default"
  ptt_mode: true                  # true = 按键说话, false = 免提
  volume_input: 0.8
  volume_output: 0.7

alerts:
  sound_enabled: true
  critical_rules:
    - type: "collision_detected"
    - type: "motor_overtemp"
    - type: "communication_lost"
  warning_rules:
    - type: "battery_low"
      threshold: 20

recording:
  default_path: "~/qooremote_recordings"
  auto_record_on_connect: false
  channels:
    joints: true
    imu: true
    forces: true
    video: true
    control: true
    audio: false
```

## 7. 本地存储结构

```
$HOME/.qooremote/
├── connections.yaml              # 连接配置

├── logs/                         # 应用日志
│   └── qooremote_20260630.log
├── cache/                        # 缓存
│   ├── robot_models/             # URDF 模型缓存
│   └── thumbnails/               # 视频缩略图缓存
├── recordings/                   # 录制数据（默认路径）
│   └── recording_20260630_143022.qoorlog
└── certs/                        # TLS 证书
    └── ca.pem
```

## 8. 数据安全

| 数据类型 | 传输加密 | 存储加密 | 访问控制 |
|----------|:------:|:------:|:------:|
| 机器人状态 | TLS 1.3 | 不存储（实时流） | 操作员 Token |
| 控制指令 | TLS 1.3 + HMAC | 不存储（实时流） | 操作员 Token + 权限 |
| 视频流 | SRTP (WebRTC) | 录制数据 AES-256 可选 | 操作员 Token |
| 音频流 | SRTP (WebRTC) | 录制数据 AES-256 可选 | 操作员 Token |
| 录制数据 | — | AES-256-GCM（可选加密） | 文件系统权限 |
| 配置/偏好 | — | 文件系统权限 | 操作系统用户 |
| Token | TLS 1.3 | 操作系统密钥链 | 操作系统用户 |
