# qooweb — 功能清单与完成进度

> **项目定位**：QooBot 官方品牌网站（web_portal），纯静态 HTML 网站，承载品牌展示、产品介绍、生态项目详情、开发者入口等功能。
> **技术栈**：HTML5 + CSS3 + Vanilla JavaScript（无框架依赖）
> **平台覆盖**：🌐 web_portal
> **国际化**：中文版（根目录）+ 英文版（en/ 子目录）
