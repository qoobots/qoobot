Project Overview

The Swiss Army Knife for Robot Development

qoodev is the developer toolchain project of the QooBot ecosystem, providing robot developers with complete tool support from coding and simulation testing to performance optimization.

Tool 01
🔌

IDE Plugin Suite

Robot development extensions for VS Code, PyCharm, CLion, and other mainstream IDEs.

Tool 03

Tool 04
📊

Performance Profiler

CPU/GPU usage, memory footprint, power analysis, and latency measurement tools.

Tool 05

Tool 06

Automated testing, code quality checks, build and release, and deployment monitoring tools.
