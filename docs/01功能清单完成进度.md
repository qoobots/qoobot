# QooBot 功能清单与开发进度

> 最后更新：2026-06-26
>

## 🧠 qoobrain — 大脑操作系统 &nbsp;&nbsp; `P0 最高优先` `✅ 已完成`

> **定位**：机器人软件中枢。感知、认知、决策、规划、控制、通信全部就绪，是其他所有子项目的运行基础。
> **依赖**：无
> **被依赖**：qoosvc、qoocloud、qoocore

| 功能模块 | 描述 | 状态 | 优先级 |
|----------|------|------|--------|
| 多模态感知 | 视觉、听觉、触觉、力觉传感器融合 | ✅ 已完成 | P0 |
| 环境认知 | SLAM、物体识别、场景理解、语义地图 | ✅ 已完成 | P0 |
| 任务决策 | 基于 LLM 的任务分解与推理 | ✅ 已完成 | P0 |
| 运动规划 | 全身运动规划、避障、抓取轨迹生成 | ✅ 已完成 | P0 |
| 实时控制 | 力控/位控、阻抗控制、全身协调 | ✅ 已完成 | P0 |
| 实时通信 | 内部总线通信、多进程 IPC、DDS | ✅ 已完成 | P1 |
| 行为树引擎 | 可编排任务流执行框架 | ✅ 已完成 | P1 |
| 安全监护 | 碰撞检测、力限监控、紧急停止 | ✅ 已完成 | P0 |

| 子系统 | 功能数 | 说明 |
|--------|--------|------|
| 🧠 计算平台 | 6 | SoC选型、异构拓扑、HSM安全模块 |
| 📷 感知系统 | 7 | 视觉/LiDAR/IMU/麦克风阵列接口与时间同步 |
| 💪 执行系统 | 6 | 关节电机、末端执行器、紧急制动硬件 |
| ⚡ 能源系统 | 5 | BMS、电源树、充电系统、低功耗模式 |
| 🔗 通信与互联 | 5 | 内部总线、高速链路、Wi-Fi/BLE/UWB/5G |
| 🏗️ 结构与形态 | 6 | 形态因子、关节模组、EMC、DFM |
