## 3. 包装工程方案

### 3.1 包装分级体系

| 等级 | 名称 | 适用场景 | 防护能力 |
|:----:|------|----------|----------|
| **P1** | 工业运输箱 | 整机长途运输/国际海运 | IP54 + 抗震 G ≤ 15 |
| **P2** | 开发套件箱 | 套件国内运输 | 防尘 + 抗震 G ≤ 10 |
| **P3** | 零件防护包 | 单件备件/零件 | 防静电 + 缓冲 |

### 4.3 WMS 系统功能

| 模块 | 功能 | 技术方案 |
|------|------|----------|
| **入库管理** | 收货、质检、上架、SN 绑定 | PDA + 条码/RFID |
| **库存管理** | 库位管理、盘点、效期预警 | 实时库存 + 循环盘点 |
| **订单管理** | 订单接收、波次分配、拣货路径优化 | 智能波次算法 |
| **出库管理** | 拣货、复核、打包、称重、出库 | PDA + 电子秤 |
| **物流追踪** | 承运商对接、轨迹查询、签收确认 | API 对接主流物流商 |
| **报表分析** | 库存周转、订单时效、异常统计 | BI 看板 |

## 6. 逆向物流方案

### 6.1 逆向物流场景

| 场景 | 触发条件 | 物流方向 | 责任方 |
|------|----------|----------|--------|
| **退货** | 7 天无理由 / 质量问题 | 用户→中心仓 | QooBot / 用户 |
| **保修维修** | 保修期内故障 | 用户→维修中心→用户 | QooBot |
| **以旧换新** | 升级换代 | 用户→回收中心 | QooBot |
| **召回** | 安全/质量缺陷 | 用户→指定点 | QooBot |
| **核心部件更换** | 故障/升级 | 工程师上门/双向 | QooBot |
| **电池回收** | 寿命到期 | 用户→回收站 | QooBot |

### 6.2 逆向物流操作流程

```
用户报修 ──▶ 远程诊断 ──▶ 判定处理方式
                              │
              ┌───────────────┼───────────────┐
              ▼               ▼               ▼
         CRU 寄送         上门维修         返厂维修
              │               │               │
              ▼               ▼               ▼
        用户自换         工程师更换      物流取件→工厂
              │               │               │
              ▼               ▼               ▼
        旧件回收          旧件回收        维修/更换
              │               │               │
              └───────────────┼───────────────┘
                              ▼
                        检验→入库/报废
```

## 7. 物流成本控制

### 7.1 降本策略

| 策略 | 措施 | 预期效果 |
|------|------|----------|
| **运输整合** | 批量集拼、回程配载 | 运费降低 15-25% |
| **包装优化** | 轻量化设计、可回收材料 | 包装成本降低 20% |
| **仓储共享** | 与合作伙伴共用仓配 | 仓储成本降低 30% |
| **路线优化** | 多式联运、中欧班列替代空运 | 国际运费降低 40% |
| **数字化** | WMS/TMS 系统、自动化分拣 | 人工成本降低 25% |
| **本地化** | 目标市场本地组装/包装 | 关税+运费降低 30% |

### 7.2 物流费用占营收目标

| 阶段 | 目标占比 | 行业平均 |
|------|:------:|:------:|
| 一期（< 500 台/年） | ≤ 8% | 8-12% |
| 二期（< 2,000 台/年） | ≤ 6% | 6-10% |
| 三期（> 10,000 台/年） | ≤ 4% | 5-8% |

## 9. 应急预案

### 9.1 风险与应对

| 风险 | 影响 | 应急预案 |
|------|------|----------|
| **运输延误** | 交付超期 | 备选物流商、多模式切换、提前预警 |
| **货物损坏** | 客户投诉 | 保险全覆盖、快速理赔、备件应急 |
| **海关扣押** | 无法交付 | 合规预审、专业报关行、目的国代理 |
| **仓储爆仓** | 无法入库 | 弹性仓储合作、临时扩容 |
| **极端天气** | 运输中断 | 多线路预案、安全库存前置 |
| **疫情/封控** | 物流瘫痪 | 多仓布局、无接触配送 |

### 9.2 保险方案

| 险种 | 覆盖范围 | 保额 |
|------|----------|------|
| 运输险 | 在途损坏/丢失 | 货值 110% |
| 仓储险 | 仓库火灾/水损/盗窃 | 库存总值 |
| 产品责任险 | 产品缺陷致第三方损失 | ¥1,000 万/次 |
