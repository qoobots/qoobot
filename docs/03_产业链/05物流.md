# QooBot 产业链 — ④ 物流（Logistics）

> 最后更新：2026-06-29
>
> 本文档定义 QooBot 双足人形机器人产业链第四要素：物流。

---

## 1. 物流模式

| 模式 | 适用场景 | 时效 | 覆盖区域 |
|------|----------|------|----------|
| **整机直发** | 成品机器人交付 | 3-7 天（国内）/ 7-21 天（国际） | 全球 |
| **套件分发** | DIY 套件/开发套件 | 3-5 天（国内）/ 5-14 天（国际） | 全球 |
| **零件供应** | 维修备件/升级组件 | 1-3 天（国内）/ 3-10 天（国际） | 全球 |
| **数字交付** | 软件/固件/图纸下载 | 即时 | 全球（GitHub/CDN） |

## 3. 包装标准

| 包装等级 | 内容 | 防护标准 |
|----------|------|----------|
| **工业运输箱** | 整机 + 配件 + 文档 | IP54 防尘防水，定制 EVA 内衬，抗震 |
| **开发套件箱** | 核心组件 + 工具 + 快速指南 | 定制泡沫内衬，双层瓦楞纸 |
| **零件包装** | 独立零件/模块 | 防静电包装 + 气泡膜 |
| **数字交付** | GitHub Release / 官方网站下载 | CDN 加速，SHA256 校验 |

### 3.1 整机运输箱规格

| 参数 | 规格 |
|------|------|
| 外形尺寸 | 1200 × 800 × 600 mm |
| 材质 | 铝合金框架 + ABS 面板 |
| 防护等级 | IP54 |
| 内衬 | CNC 定制 EVA 泡棉 |
| 重量（含机器人） | ~80-100 kg |
| 堆叠层数 | ≤ 3 层 |

## 5. 逆向物流（退换/维修）

| 场景 | 流程 | 时效 |
|------|------|------|
| **用户退货** | 用户申请 → 审核 → 上门取件 → 入库检验 | 3-7 天 |
| **保修维修** | 用户报修 → 诊断 → 寄回/上门 → 维修 → 寄回 | 5-10 天 |
| **以旧换新** | 评估 → 折价 → 发新机 → 回收旧机 | 7-14 天 |
| **核心部件升级** | 通知 → 预约 → 上门更换 → 旧件回收 | 3-5 天 |
