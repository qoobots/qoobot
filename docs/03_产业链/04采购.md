# QooBot 产业链 — ③ 采购（Procurement）

> 最后更新：2026-05-26
>
> 本文档定义 QooBot 双足人形机器人产业链第三要素：采购。

---

## 1. 采购策略

QooBot 的采购策略以**开源 BOM 清单**为基础，实现供应链透明化：

- **标准化优先**：优先选用通用工业标准件，降低采购门槛
- **多源供应**：关键器件至少 2 个可替代供应商
- **社区团购**：通过 qoochain 组织社区批量采购，降低成本
- **本地化替代**：提供国产替代方案，降低进口依赖

## 4. 供应商管理

### 4.1 供应商分级

| 等级 | 定义 | 管理方式 | 示例 |
|------|------|----------|------|
| **战略供应商** | 核心器件独家/主力供应 | 长期协议、技术协同 | NVIDIA、绿的谐波 |
| **优选供应商** | 稳定供应、质量可靠 | 年度框架协议 | T-Motor、奥比中光 |
| **合格供应商** | 通过认证、可替代 | 批次订单 | 多家 CNC 厂 |
| **候选供应商** | 审核中/备用 | 样品验证中 | 新引入供应商 |

### 4.2 供应商准入流程

```
┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│ 资质审核   │──▶│ 样品验证   │──▶│ 小批量试用 │──▶│ 批量评估   │──▶│ 正式准入   │
│ 企业资质   │   │ 性能/尺寸 │   │ 100台级   │   │ 质量/交期 │   │ 列入合格   │
│ 质量体系   │   │ /可靠性   │   │ 验证      │   │ /服务     │   │ 供应商名录  │
└──────────┘   └──────────┘   └──────────┘   └──────────┘   └──────────┘
```

## 5. 供应链风险管理

| 风险类型 | 具体风险 | 缓解措施 |
|----------|----------|----------|
| **芯片断供** | 高端 GPU 出口管制 | 多架构适配（华为昇腾/地平线/高通），qoocore 硬件抽象层 |
| **电机缺货** | 特种电机交期长 | 标准化接口，支持多品牌替换；建立安全库存 |
| **减速器垄断** | 谐波减速器产能有限 | 支持行星减速器替代方案；培育国内供应商 |
| **价格波动** | 原材料涨价 | 长期协议价；BOM 多方案设计 |
| **地缘政治** | 贸易壁垒 | 本地化采购方案；多区域供应链布局 |

## 6. 采购执行流程

```
需求计划 → 询价比价 → 订单下达 → 跟催到货 → 来料检验 → 入库结算
```

| 环节 | 负责 | 周期 |
|------|------|------|
| 需求计划 | 项目/生产 | 提前 4-8 周 |
| 询价比价 | 采购 | 1 周 |
| 订单下达 | 采购 | 1-3 天 |
| 跟催到货 | 采购 | 按交期 |
| 来料检验 | 质量 | 1-3 天 |
| 入库结算 | 仓储/财务 | 1 周 |
