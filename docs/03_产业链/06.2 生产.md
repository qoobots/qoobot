# QooBot 产业链 — ② 生产（Production）

> 最后更新：2026-06-29
>
> 本文档定义 QooBot 双足人形机器人产业链第二要素：生产。

---

## 1. 生产模式

QooBot 支持**多级生产模式**，适配不同规模的制造需求：

| 模式 | 规模 | 适用场景 | 生产周期 |
|------|------|----------|----------|
| **DIY 自造** | 单台 | 个人开发者/研究实验室 | 2-4 周 |
| **小批量代工** | 10-100 台 | 初创团队/高校实验室 | 4-8 周 |
| **中批量制造** | 100-1000 台 | 商业部署/系统集成商 | 8-16 周 |
| **大规模量产** | 1000+ 台 | OEM/ODM 合作伙伴 | 16-24 周 |

### 2.1 各工艺环节说明

| 工艺环节 | 关键工序 | 质量控制点 |
|----------|----------|------------|
| **CNC 加工** | 编程→粗加工→精加工→去毛刺 | 尺寸公差 ±0.02mm，表面粗糙度 Ra ≤ 1.6 |
| **表面处理** | 清洗→阳极氧化/喷漆→固化→检验 | 膜厚均匀性，色差 ΔE ≤ 1.5 |
| **组件装配** | 轴承压装→齿轮啮合→预紧力调节 | 间隙 ≤ 0.05mm，转动顺滑无异响 |
| **电气集成** | 布线→端子压接→焊接→导通测试 | 100% 导通，绝缘电阻 > 10MΩ |
| **整机总装** | 模块对接→紧固→线束连接→外观检查 | 全螺栓扭矩校验，接口防呆确认 |
| **质量检测** | 功能测试→性能测试→安全测试 | 全功能 PASS，无异常 |
| **软件烧录** | OS 安装→固件刷写→模型部署→激活 | 版本校验，启动自检 PASS |
| **标定校验** | 关节零位→IMU→相机→力传感器 | 各传感器精度达标 |
| **整机老化** | 48h 连续运行→高低温循环→振动 | 零故障，性能衰减 < 5% |
| **包装出货** | 清洁→附件装箱→封箱→标签 | 包装完整性，清单核对 |

## 3. 质量管控

| 阶段 | 检测项目 | 标准/方法 | 负责 |
|------|----------|-----------|------|
| **来料检验** | 电机性能、传感器精度、结构件公差 | 批次抽检 AQL 2.5 | qoochain |
| **过程检验** | 装配扭矩、焊接质量、线束导通 | 全检 | 生产方 |
| **标定校验** | 关节零位、IMU/相机标定、力传感器校准 | 自动化标定工装 | qoochain |
| **整机测试** | 行走稳定性、操作精度、续航测试、安全功能 | 标准化测试用例 | qoochain/测试团队 |
| **老化测试** | 连续 48 小时运行、高低温循环、振动测试 | 企业标准 | 生产方 |
| **出厂终检** | 全功能复核、外观检查、包装完整性 | 全检 | 生产方 |

## 4. 生产资源需求

| 资源类型 | DIY 模式 | 小批量 | 大规模量产 |

| **CNC 加工中心** | 外包 | 1-2 台 | 5-10 台 |

| **装配工位** | 个人工作台 | 5-10 工位 | 20-50 工位 |
| **测试设备** | 基础仪表 | 专用测试台 | 自动化测试线 |
| **仓储面积** | — | 100-300 ㎡ | 1000+ ㎡ |

## 5. 生产合作伙伴体系

| 合作类型 | 角色 | 要求 | 认证流程 |
|----------|------|------|----------|
| **认证加工伙伴** | CNC/3D 打印/表面处理 | 通过样品试制验证 | 送样→检验→小批量→认证 |
| **认证装配伙伴** | 组件装配/整机总装 | 通过装配工艺审核 | 工艺评审→试装→批量→认证 |
| **认证测试伙伴** | 标定/测试/老化 | 具备专用测试设备 | 设备审核→对标测试→认证 |
| **OEM/ODM 伙伴** | 整机代工 | 通过全流程审核 | 全面审核→试产→量产→认证 |
