## 📦 三级产品树：各子项目 × 多平台展开

### 1. qoobrain（自主意识大脑）

| 平台 | 部署形态 | 说明 |
|------|----------|------|
| **onboard_core** | Python 3.11 AI 引擎 + C++17 实时引擎 | 端侧自主运行，感知/认知/决策/规划/控制/安全 |
| **cloud_service** | 云端推理网关 | 长尾任务云端大模型推理增强 |
| **web_portal** | Web 状态面板 | 机器人实时状态仪表盘、行为树可视化编辑器 |
