## 2. 服务等级协议（SLA）

| 服务项 | 社区版（免费） | 专业版 | 企业版 |
|--------|:------------:|:------:|:------:|
| **响应时间** | 最佳努力 | 8 小时内 | 4 小时内 |
| **解决时间** | 社区驱动 | 48 小时内 | 24 小时内 |
| **支持渠道** | Discord / 论坛 | Discord + 工单 + 邮件 | 专属通道 + 电话 |
| **远程诊断** | — | ✅ | ✅ |
| **OTA 升级** | 手动下载 | ✅ 自动推送 | ✅ 自动推送 + 回滚 |
| **现场服务** | — | 按次计费 | ✅ 含 2 次/年 |
| **备件保障** | 社区互助 | 5 个工作日内 | 48 小时内 |
| **培训服务** | 在线文档 | 在线培训 2 次/年 | 现场培训 + 定制 |
| **软件更新** | 社区版更新 | 专业版功能 | 企业版全功能 |

## 3. 维修与备件

### 3.1 维修分级

| 维修类型 | 处理方式 | 周期 |
|----------|----------|------|
| **用户可更换单元（CRU）** | 用户自行更换（提供视频教程） | 即时 |
| **现场可更换单元（FRU）** | 授权技术人员上门更换 | 48h-72h |
| **返厂维修** | 寄回维修中心 | 5-10 工作日 |
| **核心部件更换** | 更换整机模块 | 3-5 工作日 |

### 3.2 备件库配置

| 备件类别 | 库存策略 | 覆盖率 |
|----------|----------|--------|
| 电机/驱动模块 | 安全库存 5% | 常用型号 100% |
| 传感器模块 | 安全库存 3% | 主要型号 100% |
| 结构件 | 按需 3D 打印 | 全部可打印 |
| 电子板卡 | 安全库存 5% | 核心板卡 100% |
| 线束/连接器 | 安全库存 10% | 标准件 100% |
| 电池 | 安全库存 3% | 当前型号 100% |

### 3.3 维修流程

```
用户报修 → 远程诊断 → 判定级别 → CRU: 寄备件 + 视频指导
                               → FRU: 派单上门
                               → 返厂: 寄回 → 维修 → 检验 → 寄回
                               → 核心: 模块更换
```

## 4. 软件维护与升级

| 维护类型 | 内容 | 频率 |
|----------|------|------|
| **安全补丁** | 漏洞修复、安全更新 | 发现即发布 |
| **功能更新** | 新功能、性能优化 | 每月 |
| **大版本升级** | 架构升级、重大功能 | 每季度 |
| **固件更新** | MCU 固件、驱动更新 | 随需发布 |
| **模型更新** | AI 模型优化、新能力 | 每月 |
| **社区贡献合并** | 合并社区 PR | 持续 |

### 4.1 OTA 升级机制

| 阶段 | 操作 | 回滚策略 |
|------|------|----------|
| 灰度推送 | 5% → 25% → 100% | 异常自动暂停 |
| A/B 分区 | 保留上一版本 | 启动失败自动回退 |
| 校验验证 | SHA256 + 签名 | 校验失败拒绝安装 |
| 用户确认 | 通知 → 下载 → 确认 → 安装 | 用户可推迟 |
