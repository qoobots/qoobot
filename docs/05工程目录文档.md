### 按平台归属分类

```
onboard_core（机器人端侧）
  ├── qoobrain/        ← Python 3.11 + C++17 + ROS 2
  ├── qoobody/         ← C++ (HAL .so) + 设计文档
  ├── qoocore/         ← C++17 推理引擎
  └── qoosvc/          ← C++/Python 系统服务

cloud_service（云端服务）
  ├── qoobot-cloud/    ← Java + Spring Boot（统一后端仓库）
  ├── qoostore/        ← 技能市场后端 + Web 前端
  ├── qoochain/        ← 供应链后端 + Web 管理
  ├── qoocompliance/   ← 合规后端 + Web 管理
  ├── qoogear/         ← 认证后端 + Web 管理
  └── qoocommunity/    ← 社区后端 + Web 前端

desktop_console（PC 桌面）
  └── qoodev/          ← Python 3.10+ + PySide6

web_portal / mobile_app / mini_program / mobile_h5（前端全域）
  └── qooweb/          ← TypeScript + Vue3 + Element Plus（前端主栈基准）
```

### 架构概念：qooremote（远程遥控子系统）

> `qooremote` **不是独立子项目目录**，而是跨现有子项目的**架构概念**。它是 QOS 两大核心子系统之一（与 `qoobrain` 并列），代码分布在多个目录中：

| 涉及功能 | 分布目录 | 说明 |
|----------|----------|------|
| 视频流采集/编码 | `qoosvc/` | 端侧视频流推送服务 |
| 控制指令中继 | `qoobot-cloud/` → `qoocloud/teleop/` | 云端控制中继服务 |
| 遥控会话管理 | `qoobot-cloud/` → `qoocloud/teleop/` | 会话生命周期管理 |
| 前端遥控面板 | `qooweb/` → `src/modules/cloud/` | Web/移动端遥控 UI |
| HAL 层操控接口 | `qoobody/include/qoobody/hal/` | 底层执行指令接口 |

> 协议定义统一在 `qoobot-cloud/proto/teleop/` 下，跨端共享。

## 🏪 qoostore 技能市场（端云协同）

`qoostore/` 采用端云协同架构，对标 Apple App Store。云端负责商店运营，机器人端负责技能运行：

```
qoostore/
├── README.md
├── docs/                             ← 共享设计文档
│
├── qoostore-cloud/                   ← ☁️ 云端技能商店（Java 21 + Spring Cloud）
│   ├── pom.xml
│   └── src/main/java/com/qoobot/store/cloud/
│       ├── skill/                    ← 技能管理（发布/审核/上下架）
│       ├── cdn/                      ← 技能包分发
│       ├── review/                   ← 内容审核
│       └── commerce/                 ← 计费/分成/结算
│
└── qoostore-edge/                    ← 🤖 机器人端运行环境（C++17 + Python 3.11）
    ├── CMakeLists.txt
    ├── include/                      ← 公共头文件
    ├── src/
    │   ├── installer/                ← 技能安装器
    │   ├── sandbox/                  ← 安全沙箱
    │   ├── permission/               ← 权限管控
    │   └── monitor/                  ← 运行时监控
    └── config/                       ← 运行配置
```

### 模块职责

| 模块 | 运行位置 | 职责 | 技术栈 |
|------|---------|------|--------|
| `qoostore-cloud/` | 云端 | 技能商店/发布审核/计费/分发/运营分析 | Java 21 + Spring Cloud |
| `qoostore-edge/` | 机器人端 | 技能安装/沙箱隔离/权限管控/运行时监控 | C++17 + Python 3.11 |
