## 📱 产品定位：QooBotOS (简称 QOS)

**QooBotOS**（简称 **QOS**）是双足人形机器人的操作系统，定位为**开源版 iPhone**——类比 iOS 之于智能手机，QOS 之于人形机器人：

> iOS 让智能手机从"功能机"进化为"应用生态平台"；
> QOS 让人形机器人从"单一功能机器"进化为"技能生态平台"。

| 维度 | iPhone (iOS) | QOS |
|------|-------------|-----|
| **硬件** | iPhone 设备 | 双足人形机器人（qoobody 参考设计） |
| **操作系统** | iOS | QOS（qoobrain 大脑 OS） |
| **芯片** | A 系列 / M 系列 | qoocore 推理芯片 |
| **账号体系** | Apple ID | QooIDaaS（qooauth） |
| **应用商店** | App Store | QooStore（qooeco） |
| **开发者工具** | Xcode + Swift | qoodev + Python/C++/TypeScript |
| **默认 App** | 电话/短信/相机/地图/Safari/健康... | 见下文 |
| **配件生态** | MFi 认证 | MFQ 认证（qoogear） |
| **云端服务** | iCloud | qoocloud |
| **隐私安全** | 隐私标签 + ATT | qooregs 合规 + 隐私保护 |
| **社区** | Apple Developer | qoocommunity |

## 🎯 产品哲学

与 iPhone 对标的核心逻辑：

```
iPhone = 硬件(iPhone) + OS(iOS) + 账号(Apple ID) + 商店(App Store) + 云(iCloud) + 开发者(Xcode)

QooBot = 硬件(qoobody) + OS(qoobrain) + 账号(qooauth) + 商店(qooeco) + 云(qoocloud) + 开发者(qoodev)
```

> **QOS 不是做一个机器人，而是做一套让机器人"活"起来的操作系统。**
> 正如 iPhone 重新定义了手机，QOS 致力于重新定义人形机器人——从一个执行固定程序的机器，变成一个可安装技能、可扩展能力、可不断进化的智能体。
