# QooBrain 功能清单与开发进度

> 最后更新：2026-06-26 | Phase 2 工程化与真机验证 | Sprint 11 进行中

---

## 项目概览

**QooBrain** 是 QooBot 的大脑操作系统，面向具身智能的端到端机器人软件栈。采用 monorepo 架构，集成大语言模型、计算机视觉、运动规划与实时控制。

| 指标 | 数据 |
|------|------|
| 版本 | `1.0.0-alpha` |
| 子模块 | 9 个 |
| 源代码文件 | ~500 |
| 单元/集成测试 | 301 个 (全部通过) |
| 技术栈 | C++17/ROS2 + Python 3.11/gRPC + TypeScript/Next.js/Three.js |
| 许可证 | Apache 2.0 |
