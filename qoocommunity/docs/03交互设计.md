> 版本：v0.1 | 更新：2026-04-23

---

## 一、用户角色

| 角色 | 权限范围 | 说明 |
|------|----------|------|
| **访客 Visitor** | 浏览文档/论坛/博客/活动页面 | 未登录用户 |
| **社区成员 Member** | 发帖/评论/提问/回答/报名/投稿 | 已注册用户 |
| **贡献者 Contributor** | 提交 PR / 参与 Code Review / 维护文档 | Member + CLA 签署 |
| **版主 Moderator** | 管理论坛/审核内容/处理举报 | 社区任命 |
| **管理员 Admin** | 全局管理/系统配置/数据分析 | 核心团队 |
| **TSC 成员** | 技术决策/投票/RFC 审批 | 技术指导委员会 |

## 四、核心交互流程

### 4.1 用户提问-回答流程

```
用户提问
    │
    ▼
Markdown 编辑器 (支持代码高亮/图片上传)
    │
    ▼
选择标签 (最多5个)
    │
    ▼
提交 → 敏感词过滤 → 发布
    │
    ▼
其他用户看到问题 → 回答
    │
    ▼
提问者收到通知 → 查看回答
    │
    ├── 投票(Upvote/Downvote)
    ├── 评论讨论
    └── 标记最佳答案
         │
         ▼
    提问者+最佳回答者获得声望
```

### 4.2 贡献者晋升流程

```
用户注册 → 签署 CLA → Contributor
    │
    ▼
提交 10+ PR → 3 个月活跃 → 自动推荐 Maintainer
    │
    ▼
社区投票(2/3) + TSC 审批 → Committer
    │
    ▼
年度选举 → TSC Member
```

### 4.3 活动参与流程

```
浏览活动列表 → 点击活动详情
    │
    ▼
查看议程/嘉宾/地点
    │
    ▼
点击"报名" → 填写信息(姓名/公司/职位)
    │
    ▼
收到确认通知(邮件/App推送/小程序消息)
    │
    ▼
活动当天 → 扫码签到 → 参与活动
    │
    ▼
活动后 → 查看录播 → 下载资料 → 填写反馈
```

## 五、CLI 命令 (qoodev 集成)

社区相关命令集成到 qoodev CLI 中：

```bash
# 文档相关
qoodev docs open              # 打开文档站点
qoodev docs search <query>    # 搜索文档
qoodev docs version <ver>     # 切换文档版本

# 贡献相关
qoodev community cla sign     # 签署 CLA
qoodev community cla status   # 查看 CLA 状态

# 发布展示
qoodev showcase publish       # 发布项目到 Showcase
qoodev showcase list          # 查看展示列表

# 学院
qoodev academy courses        # 课程列表
qoodev academy learn <id>     # 开始学习
qoodev academy cert list      # 认证列表
```
