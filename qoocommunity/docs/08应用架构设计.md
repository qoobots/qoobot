## 2. 文档站点架构

### 2.1 技术选型

| 组件 | 技术 | 说明 |
|:-----|------|------|
| 静态生成 | MkDocs Material / Docusaurus | 文档站点框架 |
| API 文档 | Sphinx + Breathe (C++) + autodoc (Python) | 自动 API 文档生成 |
| 搜索 | Algolia / Meilisearch | 全文搜索 |
| 多语言 | Crowdin / GitLocalize | 翻译管理 |
| 托管 | GitHub Pages / Vercel | 静态托管 + CDN |
| Playground | WebAssembly (Pyodide) + Monaco Editor | 浏览器内运行 Python |

### 2.2 文档构建流水线

```
源代码 (Python/C++)
    │
    ▼
Sphinx/Breathe 提取 docstring
    │
    ▼
MkDocs 集成 → 生成静态 HTML
    │
    ▼
多语言翻译 (Crowdin)
    │
    ▼
部署到 CDN
```

### 2.3 版本化文档

```
docs.qoobot.ai/
├── latest/           → 最新稳定版 (默认)
├── v2.1/             → v2.1 文档
├── v2.0/             → v2.0 文档
├── v1.x/             → v1.x 文档
└── dev/              → 开发中版本
```

## 3. 社区平台集成

### 3.1 GitHub 集成

```
GitHub Org: github.com/qoobot
├── qoobot/qoobrain (核心仓库)
├── qoobot/qoobody  (硬件设计)
├── qoobot/qoosvc   (系统服务)
├── qoobot/qoocloud (云端服务)
├── qoobot/docs     (文档源码)
├── qoobot/examples (示例代码)
└── qoobot/rfcs     (RFC 提案)
```

### 3.2 自动化

```
GitHub Event → GitHub Actions
    ├── PR 提交 → CLA 检查 + CI + Code Review 分配
    ├── Issue 创建 → 自动标签 + 社区机器人回复
    ├── PR 合并 → 更新贡献者统计 + 生成 Changelog
    └── Release → 自动发布 + 通知社区
```
