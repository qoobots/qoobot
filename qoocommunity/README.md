# qoocommunity — 人形机器人品牌与社区

> 机器人的"WWDC + 开发者关系 + Today at Apple"：
> 年度开发者大会、开发者计划、技术社区、高校合作、内容生态。

## 定位

qoocommunity 是 QooBot 生态的品牌与社区平台，通过开发者大会、培训认证、
社区运营和高校合作，构建全球人形机器人开发者生态。

## 模块

| 模块 | 说明 | 状态 |
|------|------|------|
| `conference/` | QooBot 全球开发者大会（年度） | 📋 规划中 |
| `dev_program/` | 开发者计划（注册、认证、等级、权益） | 📋 规划中 |
| `academy/` | QooBot 学院（在线课程、认证考试、实训） | 📋 规划中 |
| `forums/` | 开发者社区论坛（问答、展示、协作） | 📋 规划中 |
| `evangelist/` | 技术布道（博客、视频、演讲、workshop） | 📋 规划中 |
| `university/` | 高校合作（课程、实验室、竞赛、奖学金） | 📋 规划中 |
| `showcase/` | 作品展示平台（案例库、demo 视频） | 📋 规划中 |
| `hackathon/` | 黑客马拉松（线上/线下、主题赛、奖金） | 📋 规划中 |
| `ambassador/` | 社区大使计划（地区代表、用户组） | 📋 规划中 |

## 年度开发者大会

| 环节 | 内容 |
|------|------|
| Keynote | 年度技术路线图发布 |
| Sessions | 技术深度分享（感知、规划、控制、仿真） |
| Labs | 动手实验（端到端机器人开发） |
| Expo | 合作伙伴与初创公司展示 |
| Awards | 年度最佳技能/应用/论文颁奖 |
| Networking | 开发者社交晚宴 |
