## 2. Maven 模块依赖关系

```
qoocloud (父 POM)
├── qoocloud-common          ← 公共模块
│
├── qoocloud-inference       → common + qooauth-client
├── qoocloud-device          → common + qooauth-client
├── qoocloud-ota             → common + device (Feign)
├── qoocloud-data            → common
├── qoocloud-orchestra       → common + device (Feign)
├── qoocloud-twin            → common
├── qoocloud-observability   → common
└── qoocloud-infra           → common
```

## 3. Package 职责一览

| Package | 职责 | 关键框架 |
|---------|------|---------|
| `controller/` | REST API 控制器 | Spring MVC, Jakarta Validation |
| `service/` | 业务逻辑层 | Spring, Triton Client, vLLM Client |
| `grpc/` | gRPC 服务实现 | grpc-java |
| `websocket/` | WebSocket 连接管理 | Spring WebSocket |
| `entity/` | JPA 实体 | Jakarta Persistence |
| `repository/` | 数据访问层 | Spring Data JPA, MyBatis-Plus |
| `config/` | Bean 配置 | Spring Boot |
| `dto/` | 数据传输对象 | Lombok, Jackson |

## 4. 构建与运行

### 4.1 本地启动

```bash
# 启动基础设施
docker compose up -d postgres redis minio nacos rocketmq

# 编译所有模块
mvn clean compile -pl qoocloud-common
mvn clean compile

# 按依赖顺序启动
mvn spring-boot:run -pl qoocloud-inference -Dspring-boot.run.profiles=dev
mvn spring-boot:run -pl qoocloud-device -Dspring-boot.run.profiles=dev
mvn spring-boot:run -pl qoocloud-ota -Dspring-boot.run.profiles=dev
# ... 其他服务
```

### 4.2 Docker 构建

```bash
# 构建单个服务
cd qoocloud-inference
mvn package -DskipTests
docker build -t qoocloud-inference:latest .

# 构建所有服务
./scripts/build_all.sh
```

### 4.3 K8s 部署

```bash
kubectl apply -f deploy/kubernetes/infra/   # 先部署基础设施
kubectl apply -f deploy/kubernetes/inference/
kubectl apply -f deploy/kubernetes/device/
kubectl apply -f deploy/kubernetes/ota/
# ... 其他服务
```
