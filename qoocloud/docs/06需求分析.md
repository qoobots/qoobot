# qoocloud — 需求分析

> 版本：v0.1 | 最后更新：2026-06-29 | 状态：Draft | 子项目：qoocloud（云端服务）
> 对标：AWS IoT Core + OpenAI API + GitHub Actions OTA | 技术栈：Java 21 + Spring Cloud + MySQL + Redis + Kafka + K8s

---

## 1. 需求概述

qoocloud 是 QooBot 的云端大脑，连接每一台 QooBot 机器人与云端智能。提供远程大模型推理、设备全生命周期管理、OTA 升级、多机器人编排、数字孪生等云端核心能力。端云协同架构中，重型推理上云、轻量推理在端。

---
