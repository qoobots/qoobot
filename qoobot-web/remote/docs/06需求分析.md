# qooremote — 需求分析

> 版本：v0.1 | 最后更新：2026-06-29 | 状态：Draft | 子项目：qooremote（远程遥控）
> 对标：无障碍服务 (Accessibility Service) + 设备管理 (Device Policy) + 远程桌面

---

## 1. 需求概述

qooremote 是 QooBot 远程人工遥控子系统。当自主大脑 qoobrain 无法应对长尾场景时，人类操作员可远程接管机器人的感知、决策与控制。它是自主智能的"安全网"和"学习源"——既提供人类兜底，又采集示教数据反哺 AI 训练。

---

## 2. 功能需求 (FR)

### FR-100：遥操作控制
| 编号 | 功能需求 | 描述 | 优先级 |
|:-----|---------|------|:------:|
| FR-101 | 动作捕捉遥操作 | 动捕设备→机器人运动映射，全身运动实时同步（延迟<100ms） | P1 |
| FR-102 | VR 沉浸式遥控 | VR 头显第一人称遥控，头部追踪+手柄操控+立体视觉回传 | P2 |
| FR-103 | 手柄/键盘遥控 | 常规输入设备遥控，基础移动与操作，支持自定义按键映射 | P1 |
| FR-104 | 触屏遥控 | 移动端触屏遥控，虚拟摇杆+手势操作 | P1 |

### FR-200：远程感知与通信
| 编号 | 功能需求 | 描述 | 优先级 |
|:-----|---------|------|:------:|
| FR-201 | 远程视频回传 | 多路视频实时编码与传输，H.264/H.265 硬件编码，分辨率自适应 | P0 |
| FR-202 | 远程音频对讲 | 双向低延迟语音通信（<150ms），Opus 编码+AEC 回声消除 | P1 |
| FR-203 | 传感器数据回传 | 力觉/触觉/IMU 等多模态传感器数据实时回传 | P2 |
| FR-204 | 3D 场景重建回传 | 机器人端实时 3D 场景重建并低带宽传输到操控端 | P2 |

### FR-300：控制模式管理
| 编号 | 功能需求 | 描述 | 优先级 |
|:-----|---------|------|:------:|
| FR-301 | 远程接管 | 自主↔遥控模式无缝切换，保持运动连续性，双向确认 | P0 |
| FR-302 | 遥控会话管理 | 遥控会话生命周期管理：建立/鉴权/断线重连/超时释放 | P0 |
| FR-303 | 多操作员协作 | 支持多操作员同时观察，主控/观察者角色切换 | P2 |
| FR-304 | 紧急停止 | 任何操作员可触发紧急停止，优先级最高，延迟<50ms | P0 |
| FR-305 | 权限分级 | 操作员权限分级：观察/基础操控/高级操控/管理员 | P1 |

### FR-400：示教与诊断
| 编号 | 功能需求 | 描述 | 优先级 |
|:-----|---------|------|:------:|
| FR-401 | 示教录制与回放 | 操作数据采集→技能训练，关节轨迹+传感器+视频多模态录制 | P1 |
| FR-402 | 远程诊断 | 远程查看机器人系统状态、日志、传感器数据、健康报告 | P1 |
| FR-403 | 控制信令中继 | 云端低延迟指令转发（WebRTC DataChannel / 自定义 UDP） | P1 |

## 3. 非功能需求 (NFR)

| 类别 | 编号 | 需求 | 目标值 |
|:-----|:-----|------|:------:|
| ⚡ 延迟 | NFR-101 | 视频回传端到端延迟 | <200ms (局域网), <500ms (互联网) |
| ⚡ 延迟 | NFR-102 | 控制指令延迟 | <50ms (局域网), <150ms (互联网) |
| ⚡ 延迟 | NFR-103 | 动捕→运动执行延迟 | <100ms |
| ⚡ 带宽 | NFR-201 | 视频带宽 | 2-20Mbps (自适应) |
| ⚡ 带宽 | NFR-202 | 控制信令带宽 | <100Kbps |
| 🔒 安全 | NFR-301 | 通信加密 | DTLS/SRTP 全程加密 |
| 🔒 安全 | NFR-302 | 操作员认证 | 双因素认证 + 会话令牌 |
| 🛡️ 可靠 | NFR-401 | 断线自动安全停止 | <500ms 检测 + 立即制动 |
| 🛡️ 可靠 | NFR-402 | 连接可用性 | >99.9% |

## 4. 用户角色

| 角色 | 场景 | 需求 |
|:-----|------|------|
| 👨‍🔧 远程操作员 | 机器人卡住/长尾场景 | 低延迟视频+控制、直观操控界面 |
| 👀 监督者 | 监控多个机器人 | 多路视频、异常告警、一键接管 |
| 👨‍🏫 示教者 | 训练机器人新技能 | 精确录制、回放编辑、数据标注 |
| 🔧 运维工程师 | 远程故障排查 | 远程诊断、日志查看、传感器数据 |
