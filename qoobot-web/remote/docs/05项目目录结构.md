# qooremote 项目目录结构

## 1. 概述

qooremote 是**跨子项目架构概念**，代码分布在多个现有子项目中。本项目目录 (`qooremote/`) 承载设计文档、协调配置和协议定义。

## 3. 模块依赖关系

```
qoodev/qoodev/teleop/ (Python 遥控 SDK)
qooremote/qooremote-web/ (Vue3 遥控面板)
    ↓ 依赖 WebSocket / REST
qoocloud-teleop/ (云端中继 :8208)
    ↓ 依赖 Protobuf + WebRTC
qooremote/proto/ (协议定义)
    ↓ 被依赖
qoosvc/svc_remote/ + qoosvc/svc_video/ (端侧服务)
    ↓ 依赖 HAL
qoobody/hal/ + qoobody/src/hal/ (硬件抽象层)
```

## 4. 协议定义文件说明

| 文件 | 描述 | 使用方 |
|------|------|--------|
| `teleop_command.proto` | 控制指令：Fullbody/Joint/Gripper/Head/ESTOP | qoodev → qoocloud → qoosvc |
| `teleop_state.proto` | 状态上报：机器人姿态/关节/传感器/诊断 | qoosvc → qoocloud → qoodev/qooweb |
| `teleop_session.proto` | 会话管理：创建/连接/切换/关闭 | qoodev/qooweb ↔ qoocloud |
| `teleop_teaching.proto` | 示教数据：轨迹/传感器/视频录制 | qoosvc → qoocloud |
| `teleop_media.proto` | 媒体协商：SDP/ICE/编码参数 | qoodev/qoosvc ↔ qoocloud |

## 5. 新增 qoocloud 模块注册

在 `qoocloud/pom.xml` 中添加：

```xml
<module>qoocloud-teleop</module>
```

在 `qoocloud/pom.xml` 的 `<modules>` 中添加：

```xml
<module>qoocloud/qoocloud-teleop</module>
```
