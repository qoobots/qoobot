# qooremote 数据设计

## 1. 核心实体

### 1.1 遥控会话 (TeleopSession)

```
TeleopSession {
    session_id:       UUID            PK
    robot_id:         string          FK → device.robot_id
    operator_id:      string          FK → qooauth.user_id
    control_mode:     enum(AUTO|HYBRID|TELEOP)
    session_status:   enum(INITIATING|CONNECTING|ACTIVE|PAUSED|CLOSING|CLOSED)
    media_types:      set(VIDEO|AUDIO|DATA)
    
    -- WebRTC 连接信息
    sdp_offer:        text
    sdp_answer:       text
    ice_candidates:   json[]
    
    -- 时间戳
    created_at:       timestamp
    connected_at:     timestamp
    takeover_at:      timestamp      -- 接管发生时间
    handover_at:      timestamp      -- 交还发生时间
    closed_at:        timestamp
    last_heartbeat:   timestamp
    
    -- 统计
    command_count:    bigint         -- 指令总数
    video_bytes_sent: bigint        -- 视频流量
    audio_bytes_sent: bigint        -- 音频流量
    max_latency_ms:   int           -- 最大延迟
    avg_latency_ms:   int           -- 平均延迟
}
```

### 1.2 控制指令记录 (ControlCommand)

```
ControlCommand {
    command_id:       UUID            PK
    session_id:       UUID            FK → TeleopSession
    sequence:         bigint          -- 序列号
    
    command_type:     enum(FULLBODY|JOINT|GRIPPER|HEAD|ESTOP|MODE_CHANGE)
    command_data:     json            -- 具体指令内容
    
    -- 校验
    range_valid:      bool            -- 范围校验通过
    rate_valid:       bool            -- 速率校验通过
    
    -- 时间戳
    sent_at:          timestamp       -- 操作员发送时间
    relayed_at:       timestamp       -- 云端中继时间
    received_at:      timestamp       -- 机器人接收时间
    executed_at:      timestamp       -- 执行完成时间
    
    -- 延迟
    network_latency_ms: int          -- 网络延迟
    execution_latency_ms: int        -- 执行延迟
    total_latency_ms:  int           -- 端到端延迟
}
```

### 1.3 机器人状态快照 (RobotStateSnapshot)

```
RobotStateSnapshot {
    snapshot_id:      UUID            PK
    session_id:       UUID            FK → TeleopSession
    timestamp:        timestamp
    
    -- 全身状态
    fullbody_state:   json            -- qoo_fullbody_state_t 序列化
    
    -- 关节状态
    joint_states:     json[]          -- 每个关节角度/速度/力矩
    
    -- 安全状态
    safety_mode:      enum(NORMAL|REDUCED_SPEED|PROTECTIVE_STOP|ESTOP)
    active_events:    enum[]          -- 当前活跃安全事件
    
    -- 电池
    battery_soc:      float           -- 电量百分比
    battery_temp:     float           -- 电池温度
    
    -- 诊断
    cpu_temp:         float
    motor_temps:      float[]
    comm_quality:     enum(EXCELLENT|GOOD|FAIR|POOR|LOST)
}
```

### 1.4 示教记录 (TeachingRecord)

```
TeachingRecord {
    record_id:        UUID            PK
    session_id:       UUID            FK → TeleopSession
    operator_id:      string          FK → qooauth.user_id
    robot_id:         string          FK → device.robot_id
    
    -- 元数据
    name:             string          -- 示教名称
    description:      text            -- 描述
    tags:             string[]        -- 标签
    
    -- 录制数据
    duration_ms:      bigint         -- 录制时长
    frame_count:      int            -- 数据帧数
    data_format:      string         -- 数据格式版本
    
    -- 数据存储
    trajectory_path:  string          -- 关节轨迹文件路径
    sensor_data_path: string          -- 传感器数据文件路径
    video_path:       string          -- 视频录制文件路径
    
    -- 质量评估
    quality_score:    float           -- 示教质量评分
    is_verified:      bool            -- 是否经过验证
    
    created_at:       timestamp
}
```

## 2. 数据库设计 (PostgreSQL)

### 2.1 teleop_sessions 表

```sql
CREATE TABLE teleop_sessions (
    session_id      UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    robot_id        VARCHAR(64) NOT NULL,
    operator_id     VARCHAR(128) NOT NULL,
    control_mode    VARCHAR(16) NOT NULL DEFAULT 'AUTO',
    session_status  VARCHAR(16) NOT NULL DEFAULT 'INITIATING',
    media_types     JSONB NOT NULL DEFAULT '[]',
    
    sdp_offer       TEXT,
    sdp_answer      TEXT,
    ice_candidates  JSONB,
    
    created_at      TIMESTAMP NOT NULL DEFAULT NOW(),
    connected_at    TIMESTAMP,
    takeover_at     TIMESTAMP,
    handover_at     TIMESTAMP,
    closed_at       TIMESTAMP,
    last_heartbeat  TIMESTAMP,
    
    command_count       BIGINT DEFAULT 0,
    video_bytes_sent    BIGINT DEFAULT 0,
    audio_bytes_sent    BIGINT DEFAULT 0,
    max_latency_ms      INTEGER DEFAULT 0,
    avg_latency_ms      INTEGER DEFAULT 0,
    
    CONSTRAINT fk_robot FOREIGN KEY (robot_id) REFERENCES devices(robot_id)
);

CREATE INDEX idx_sessions_robot ON teleop_sessions(robot_id, created_at DESC);
CREATE INDEX idx_sessions_operator ON teleop_sessions(operator_id, created_at DESC);
CREATE INDEX idx_sessions_status ON teleop_sessions(session_status);
```

### 2.2 control_commands 表

```sql
CREATE TABLE control_commands (
    command_id      UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    session_id      UUID NOT NULL REFERENCES teleop_sessions(session_id),
    sequence        BIGINT NOT NULL,
    command_type    VARCHAR(16) NOT NULL,
    command_data    JSONB NOT NULL,
    
    range_valid     BOOLEAN DEFAULT true,
    rate_valid      BOOLEAN DEFAULT true,
    
    sent_at         TIMESTAMP,
    relayed_at      TIMESTAMP,
    received_at     TIMESTAMP,
    executed_at     TIMESTAMP,
    
    network_latency_ms   INTEGER,
    execution_latency_ms INTEGER,
    total_latency_ms     INTEGER
);

CREATE INDEX idx_commands_session ON control_commands(session_id, sequence);
```

### 2.3 robot_state_snapshots 表

```sql
CREATE TABLE robot_state_snapshots (
    snapshot_id     UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    session_id      UUID NOT NULL REFERENCES teleop_sessions(session_id),
    timestamp       TIMESTAMP NOT NULL DEFAULT NOW(),
    
    fullbody_state  JSONB,
    joint_states    JSONB,
    safety_mode     VARCHAR(16),
    active_events   JSONB,
    
    battery_soc     REAL,
    battery_temp    REAL,
    cpu_temp        REAL,
    motor_temps     JSONB,
    comm_quality    VARCHAR(8)
);

CREATE INDEX idx_snapshots_session ON robot_state_snapshots(session_id, timestamp);
```

### 2.4 teaching_records 表

```sql
CREATE TABLE teaching_records (
    record_id       UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    session_id      UUID NOT NULL REFERENCES teleop_sessions(session_id),
    operator_id     VARCHAR(128) NOT NULL,
    robot_id        VARCHAR(64) NOT NULL,
    
    name            VARCHAR(256) NOT NULL,
    description     TEXT,
    tags            JSONB DEFAULT '[]',
    
    duration_ms     BIGINT NOT NULL,
    frame_count     INTEGER NOT NULL,
    data_format     VARCHAR(32) DEFAULT 'v1.0',
    
    trajectory_path     VARCHAR(512),
    sensor_data_path    VARCHAR(512),
    video_path          VARCHAR(512),
    
    quality_score   REAL,
    is_verified     BOOLEAN DEFAULT false,
    
    created_at      TIMESTAMP NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_teaching_robot ON teaching_records(robot_id, created_at DESC);
CREATE INDEX idx_teaching_operator ON teaching_records(operator_id, created_at DESC);
```

## 3. Redis 缓存设计

| Key Pattern | 类型 | TTL | 说明 |
|-------------|------|-----|------|
| `teleop:session:{session_id}` | Hash | 会话期间 | 活跃会话状态 |
| `teleop:robot:{robot_id}:session` | String | 会话期间 | 机器人当前会话ID |
| `teleop:operator:{op_id}:session` | String | 会话期间 | 操作员当前会话ID |
| `teleop:session:{session_id}:heartbeat` | String | 5s | 心跳检测 |
| `teleop:session:{session_id}:commands:seq` | String | 会话期间 | 指令序列号 |
| `teleop:robot:{robot_id}:state` | Hash | 30s | 机器人最新状态缓存 |
| `teleop:latency:{session_id}` | ZSet | 1h | 延迟时间序列 |

## 4. 文件存储 (MinIO/对象存储)

| 路径模式 | 内容 | 格式 |
|----------|------|------|
| `teaching/{robot_id}/{record_id}/trajectory.bin` | 关节轨迹数据 | 二进制 |
| `teaching/{robot_id}/{record_id}/sensors.bin` | 传感器数据 | 二进制 |
| `teaching/{robot_id}/{record_id}/video.mp4` | 视频录制 | MP4 |
| `teaching/{robot_id}/{record_id}/metadata.json` | 录制元数据 | JSON |

## 5. 消息队列 (RocketMQ)

| Topic | 生产者 | 消费者 | 说明 |
|-------|--------|--------|------|
| `teleop-command-{robot_id}` | qoocloud-teleop | Robot Agent | 控制指令下发 |
| `teleop-state-{robot_id}` | Robot Agent | qoocloud-teleop | 状态上报 |
| `teleop-event` | Robot Agent | qoocloud-teleop | 事件通知(急停/碰撞等) |
| `teleop-teaching-{robot_id}` | Robot Agent | qoocloud-teleop | 示教数据采集 |
