Project Overview

The Swiss Army Knife for Robot Development

qoobot-desktop is the desktop software suite of the QooBot ecosystem, providing robot developers with complete tool support from project initialization, coding, debugging and simulation to build and deployment. It includes 4 sub-modules with all 56 features completed.

Module 01
⌨️

CLI Toolchain

Project initialization, build, debug, deploy — a one-stop command-line robot development workflow.

Module 02
🔍

LSP Server

Code completion, diagnostics, symbol navigation — intelligent editor support for robot development languages.

Module 03
🔌

VS Code Plugin

Integrated development extension — syntax highlighting, intelligent hints, simulation integration, one-click deploy to robot.

Module 04
🐍

Python SDK

Robot application development library — skill authoring, perception calls, motion control, Pythonic API ready out of the box.

Module 05
🎮

Simulator Integration

Isaac Sim + MuJoCo deep integration, physically accurate real-time robot behavior simulation.

Module 06
🚀

Deployment & Ops Tools

Remote deployment, configuration management, version rollback, monitoring — end-to-end DevOps tools.
