桌面开发工具套件

💻 qoobot-desktop

机器人开发全流程桌面工具链 — 包含 CLI 工具、LSP Server、VS Code 插件与 Python SDK，覆盖编码、调试、仿真、构建、部署的完整开发体验。

v1.0 已完成 56/56 功能完成
项目概览

机器人开发的瑞士军刀

qoobot-desktop 是 QooBot 生态的桌面软件套件，为机器人开发者提供从项目初始化、代码编写、调试仿真到构建部署的全流程工具支持。包含 4 个子模块，共 56 个功能模块全部完成。
