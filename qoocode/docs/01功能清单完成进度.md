# qoocode — 开发者工具 功能清单与开发进度

> 最后更新：2026-06-27
>
> **当前阶段**：v1.0 GA — 正式发布，稳定版本 + 完整文档 + 示例项目。详见[开发路线图](03开发路线图.md)。
>
> **定位**：开发者生产力工具链。没有仿真器/IDE 插件/调试器，第三方开发者无法高效地为 QooBot 开发应用。
> **依赖**：qoobrain（需集成其 API 和仿真环境）
> **被依赖**：qooeco、qoocommunity
>
> **设计理念**：对标 Xcode + Instruments + TestFlight 的完整工具生态。机器人开发与移动 App 开发有本质不同——开发者需要物理仿真环境、实时遥操作调试、多传感器数据可视化、以及从示教到自主的渐进式开发流程。qoocode 提供从项目脚手架到技能发布的一站式工具链。
