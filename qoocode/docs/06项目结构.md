## 模块职责

### CLI 工具包 (`qoocode/`)

| 模块 | 职责 | 入口点 |
|------|------|--------|
| `main.py` | CLI 入口，命令注册与分发 | `qoo` 命令 |
| `scaffold.py` | 项目脚手架引擎，模板渲染 | `qoo init` 内部 |
| `context.py` | 检测当前目录是否在 qoocode 项目中 | 所有命令前置检查 |
| `commands/init.py` | 项目初始化命令，支持 skill/service/model 模板 | `qoo init` |
| `commands/build.py` | 构建命令，编排构建流程 | `qoo build` |
| `commands/run.py` | 运行命令，启动技能/服务 | `qoo run` |
| `commands/test.py` | 测试命令，运行单元测试 | `qoo test` |
| `commands/doctor.py` | 环境诊断，检查依赖与配置 | `qoo doctor` |
| `commands/sim.py` | 仿真环境管理，启停/暂停/场景/监控 | `qoo sim` ← v0.5 新增 |
| `templates/` | 项目模板定义，含代码生成逻辑 | 由 `scaffold.py` 调用 |

### 仿真桥接层 (`qoocode/sim_bridge/`) ← v0.5 新增

| 模块 | 职责 | 核心类/函数 |
|------|------|-------------|
| `interface.py` | 统一仿真接口抽象基类与数据类型 | `SimBackend`, `SimConfig`, `SimScene`, `SimSensorData`, `SimControlCommand` |
| `manager.py` | 仿真生命周期管理器，多后端注册与切换 | `SimManager`, `register_backend`, `list_backends` |
| `scene_loader.py` | 场景加载器，支持预置/JSON/MJCF/USD | `SceneLoader`, `register_preset`, `list_presets` |
| `mujoco_backend.py` | MuJoCo 物理引擎后端，MJCF 动态生成 | `MuJoCoBackend` |
| `isaac_sim_backend.py` | Isaac Sim 后端，USD 场景加载 | `IsaacSimBackend` |
| `debugger.py` | 调试诊断套件 | `LogStream`, `SensorVisualizer`, `SceneVisualizer`, `VariableMonitor`, `Profiler` |
| `dashboard_server.py` | WebSocket 服务端，推送仿真数据到 Dashboard | `DashboardServer` |

### Python SDK (`qoobot-sdk/`)

| 模块 | 职责 | 核心类/函数 |
|------|------|-------------|
| `skill.py` | 技能基类，定义生命周期钩子 | `QooSkill`, `SkillConfig`, `SkillState` |
| `perception.py` | 感知数据类型与接口 | `Image`, `PointCloud`, `IMUData`, `JointStates` |
| `control.py` | 控制指令类型 | `JointCommand`, `EndEffectorTarget`, `GripperCommand` |
| `communication.py` | 通信客户端与 ROS2 桥接 | `BrainOSClient`, `ROS2Bridge` |
| `logging.py` | 结构化日志，支持分级与上下文 | `QooLogger`, `LogContext` |

### VS Code 插件 (`vscode-plugin/`)

| 模块 | 职责 | 技术 |
|------|------|------|
| `extension.ts` | 插件激活入口，初始化 LSP/CLI/视图 | TypeScript |
| `lsp/client.ts` | LSP 客户端，启动 pygls 语言服务器 | vscode-languageclient |
| `commands.ts` | 命令面板集成，CLI 命令桥接 | VS Code API |
| `editors/behaviorTreeEditor.ts` | 行为树可视化编辑，拖拽 + SVG 渲染 | WebView + CustomEditor |
| `snippets/provider.ts` | qoobot-sdk API 智能补全 | CompletionItemProvider |
| `views/projectExplorer.ts` | 项目文件树，技能/行为树分类显示 | TreeDataProvider |
| `views/skillList.ts` | 技能清单，扫描 QooSkill 子类 | TreeDataProvider |
| `views/behaviorTreeList.ts` | 行为树文件列表，快速打开编辑 | TreeDataProvider |

### LSP 语言服务器 (`qoocode-lsp/`)

| 模块 | 职责 | 协议 |
|------|------|------|
| `server.py` | LSP 服务器主体，注册所有处理器 | JSON-RPC (stdio/TCP) |
| `completion.py` | 代码补全，感知 qoobot-sdk API | textDocument/completion |
| `hover.py` | 悬停文档，显示 SDK API 说明 | textDocument/hover |
| `definition.py` | 跳转到定义、查找引用 | textDocument/definition, references |
| `diagnostics.py` | 代码检查，项目结构验证 | textDocument/publishDiagnostics |
| `symbols.py` | 文档符号，类/方法大纲视图 | textDocument/documentSymbol |
| `signature.py` | 函数签名帮助，参数提示 | textDocument/signatureHelp |

### Web Dashboard (`web-dashboard/`) ← v0.5 新增

| 模块 | 职责 | 技术 |
|------|------|------|
| `App.tsx` | 主布局，Tab 导航 (场景/日志/传感器/性能) | React |
| `Scene3D.tsx` | Three.js 3D 场景渲染，机器人/物体实时位姿 | @react-three/fiber |
| `LogPanel.tsx` | 实时日志流，分级过滤、搜索、导出 | React |
| `SensorPanel.tsx` | 传感器数据统计，频率/范围/均值可视化 | React |
| `ProfilerPanel.tsx` | 性能剖析，端到端延迟链路 P50/P95/P99 | React |
| `StatusBar.tsx` | 连接状态、仿真时间、实时因子 | React |
| `Sidebar.tsx` | 场景层次树，机器人/物体/摄像机列表 | React |
| `useWebSocket.ts` | WebSocket 钩子，自动重连，消息路由 | React Hook |
| `simStore.ts` | Zustand 全局状态 (仿真/日志/传感器/性能) | Zustand |

## 入口点

```
# CLI 命令
qoo init <name> [--type skill|service|model]
qoo build / run / test / doctor

# 仿真管理 ← v0.5 新增
qoo sim start [--backend mujoco|isaac_sim] [--scene home|factory|empty] [--headless]
qoo sim stop / pause / resume / step [N]
qoo sim monitor [--interval 0.5] [--duration 0]
qoo sim scenes / backends

# LSP 服务器
qoocode-lsp                    # stdio 模式 (IDE 集成)
qoocode-lsp --tcp --port 2087  # TCP 模式 (远程开发)

# Web Dashboard ← v0.5 新增
cd web-dashboard && npm run dev   # http://localhost:3000

# VS Code 插件
命令面板:
  qoocode: Init Project / Build / Run / Test / Doctor
  qoocode: Create New Skill / Service / Model
  qoocode: Open Behavior Tree Editor
  qoocode: Start / Stop Simulation
  qoocode: Open Dashboard

侧栏视图:
  qoocode Project Explorer / Skills / Behavior Trees
```
