qoo build / run / test / doctor
qoo sim start [--backend mujoco|isaac_sim] [--scene home|factory|empty] [--headless]
qoo sim stop / pause / resume / step [N] / monitor
qoo package build / inspect / validate / extract / sign / verify / resolve
qoo record start / inspect / replay / export / compare
qoo debug attach / server / breakpoint / backtrace
qoo ci init
qoo eco submit / status / search
qoo docs generate
qoo profile run / flame / comm / model
