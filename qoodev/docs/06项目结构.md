## 模块职责

### CLI 工具包 (`qoodev/`)

| 模块 | 职责 | 入口点 |
|------|------|--------|
| `main.py` | CLI 入口，15 子命令注册与分发 | `qoo` 命令 |
| `scaffold.py` | 项目脚手架引擎，模板渲染 | `qoo init` 内部 |
| `context.py` | 检测当前目录是否在 qoodev 项目中 | 所有命令前置检查 |
| `commands/init.py` | 项目初始化命令 | `qoo init` |
| `commands/build.py` | 构建命令 | `qoo build` |
| `commands/run.py` | 运行命令 | `qoo run` |
| `commands/test.py` | 测试命令 | `qoo test` |
| `commands/doctor.py` | 环境诊断 | `qoo doctor` |
| `commands/sim.py` | 仿真环境管理 | `qoo sim` |
| `commands/package_cmd.py` | 技能打包/验证/签名 | `qoo package` ← v0.7 |
| `commands/record_cmd.py` | 数据录制/回放/导出 | `qoo record` ← v0.7 |
| `commands/debug_cmd.py` | 远程调试/断点管理 | `qoo debug` ← v0.7 |
| `commands/ci_cmd.py` | CI/CD 工作流初始化 | `qoo ci` ← v0.7 |
| `commands/profile_cmd.py` | 性能剖析 (run/flame/comm/model) | `qoo profile` ← v1.5 |
| `commands/compile_cmd.py` | 模型编译/交叉编译/量化 | `qoo compile` ← v1.5 |
| `commands/annotate_cmd.py` | 数据标注 (create/label/review/stats/export/import) | `qoo annotate` ← v1.5 |
