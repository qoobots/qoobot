# qoodev — 开发者工具 功能清单与开发进度

> 最后更新：2026-06-29
>
> **当前阶段**：v1.7.0 — 调试器客户端完善 + 加速仿真后端集成 + 测试扩展至55个
>
> **定位**：开发者生产力工具链。没有仿真器/IDE 插件/调试器，第三方开发者无法高效地为 QooBot 开发应用。
> **依赖**：qoobrain（需集成其 API 和仿真环境）
> **被依赖**：qoostore、qoocommunity
>
> **设计理念**：对标 Xcode + Instruments + TestFlight 的完整工具生态。机器人开发与移动 App 开发有本质不同——开发者需要物理仿真环境、实时遥操作调试、多传感器数据可视化、以及从示教到自主的渐进式开发流程。qoodev 提供从项目脚手架到技能发布的一站式工具链。
>
