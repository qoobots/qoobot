## 2. 测试分层详解

### 2.1 单元测试

| 组件 | 框架 | 覆盖率目标 |
|:-----|:-----|:-----------|
| VS Code Extension (TS) | Jest + vscode-test | ≥ 80% |
| CLI 工具 (Python) | pytest | ≥ 85% |
| Web Dashboard | Jest + React Testing Library | ≥ 75% |
| 仿真接口 | pytest + mock | ≥ 80% |

```typescript
// VS Code Extension 测试示例
describe('BehaviorTreeEditor', () => {
    it('should render 100 nodes without performance drop', async () => {
        const editor = new BehaviorTreeEditor();
        const tree = generateTree(100);
        const start = performance.now();
        await editor.render(tree);
        const elapsed = performance.now() - start;
        expect(elapsed).toBeLessThan(200); // ms
    });
});
```
