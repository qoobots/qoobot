## 1. 产品概述

qoodev 是一站式机器人技能开发工具链，覆盖从项目脚手架到技能发布的全流程。它解决了机器人开发的核心痛点：物理仿真难、调试观测难、多传感器协同复杂。

## 2. 用户旅程

### 2.1 技能开发完整流程

```
┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐
│ 创建    │──>│ 编写    │──>│ 仿真    │──>│ 调试    │──>│ 发布    │
│ 项目    │   │ 代码    │   │ 测试    │   │ 优化    │   │ 上线    │
└─────────┘   └─────────┘   └─────────┘   └─────────┘   └─────────┘
     │             │             │             │             │
  qoo init    VS Code      qoo sim      qoo debug    qoo package
  模板脚手架   智能补全    物理仿真     断点调试     .qooskills
  行为树编辑  代码生成    传感器仿真   变量监控     签名发布
  项目结构    语法检查    域随机化     性能剖析     qoostore上线
```

### 2.2 新手入门体验

```
下载 VS Code 插件
    │
    ▼
qoo init --template=hello_skill  ← 一键创建
    │
    ▼
打开 behavior_tree.qoobt         ← 可视化编辑行为树
    │
    ▼
qoo sim                          ← 一键启动仿真
    │
    ▼
在仿真中实时查看机器人行为      ← Web Dashboard
    │
    ▼
qoo package                      ← 打包技能
```

## 5. 设计原则

1. **降低门槛**：从零到第一个技能 < 30 分钟。
2. **所见即所得**：仿真即真实，调试即现场。
3. **跨平台一致**：Windows/macOS/Linux 体验统一。
4. **渐进式复杂度**：简单任务简单做，复杂任务可深入。
5. **开发者优先**：API 设计优先考虑开发者体验 (DX)。
6. **开放可扩展**：插件架构，第三方可扩展工具链。
