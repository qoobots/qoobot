## 1. 安全架构总览

│  └──────────────────────────────────────────────────────────┘   │

## 3. 调试通道安全

| 措施 | 实现 |
|:-----|:-----|
| **传输加密** | WebSocket over TLS 1.3 |
| **认证** | JWT Token 验证 (qooauth签发) |
| **会话管理** | 30分钟超时, 单会话限制 |
| **命令鉴权** | 调试命令需匹配技能开发者权限 |
| **审计日志** | 所有调试命令记录 |

## 5. 威胁模型

| 威胁 | 攻击向量 | 缓解措施 | 严重度 |
|:-----|:---------|:---------|:------:|
| **恶意技能** | 上传恶意代码 | 沙箱隔离 + 权限限制 + 审核 | 高 |
| **调试劫持** | 未授权调试连接 | Token认证 + 会话管理 | 高 |
| **代码注入** | 通过断点注入 | 命令白名单 + 频率限制 | 中 |
| **依赖投毒** | 恶意依赖包 | npm/pip audit + SBOM | 中 |
| **仿真逃逸** | 从仿真攻击主机 | 容器隔离 + seccomp | 低 |
| **密钥泄露** | 签名证书泄露 | HSM存储 + 短有效期 | 高 |
