## 1. 工程总览

### 1.1 工程全景

```
brain_os/                                    ← 工程根 (monorepo)
│
├── brain_core/                              ← 项目: C++ 实时层 (brain_core)
├── brain_ai/                                ← 项目: Python AI 层 (brain_ai)
├── brain_viz/                               ← 项目: TypeScript 前端 (brain_viz)
├── brain_sdk/                               ← 项目: Python SDK (brain-sdk pip 包)
├── brain_proto/                             ← 项目: Protobuf API 定义
├── brain_sim/                               ← 项目: 仿真环境配置
├── brain_models/                            ← 项目: 模型存储 (Git LFS)
├── brain_deploy/                            ← 项目: 部署工具与配置
├── brain_docs/                              ← 项目: 文档站点 (MkDocs)
│
├── CMakeLists.txt                           ← 顶层 CMake (ament_cmake)
├── pyproject.toml                           ← Python workspace 管理
├── package.json                             ← TypeScript workspace 管理
├── docker-compose.yml                       ← 开发环境一键启动
├── Makefile                                 ← 顶层构建快捷命令
├── .gitignore
├── .gitmodules
├── README.md
└── LICENSE
```

### 1.2 项目与架构层次映射

```
工程 → 项目          → 语言/运行时     → 对应架构层        → 容器/进程
─────────────────────────────────────────────────────────────────────
brain_os
  ├── brain_core    → C++17           → Layer 0-2         → brain_core 进程
  │                                      基础设施 + 基础服务
  │
  ├── brain_ai      → Python 3.11     → Layer 1 + 3       → brain_ai 进程
  │                                      基础模型 + 业务逻辑
  │
  ├── brain_viz     → TypeScript      → Layer 4-5          → brain_viz 进程
  │                      Next.js 14      接口层 + 应用层
  │
  ├── brain_sdk     → Python 3.11     → Layer 4-5          → pip 包
  │                                      对外 SDK
  │
  ├── brain_proto   → Protobuf        → 跨层契约           → 编译产物
  │
  ├── brain_sim     → Python/URDF     → 仿真辅助           → 独立环境
  │
  ├── brain_models  → 二进制          → Layer 1 模型文件   → Git LFS
  │
  ├── brain_deploy  → Docker/YAML     → 运维              → 编排层
  │
  └── brain_docs    → Markdown/MkDocs → 文档              → 静态站点
```

### 1.3 跨进程通信矩阵

```
brain_core (C++)          brain_ai (Python)         brain_viz (TS)
     │                          │                        │
     ├──── ROS 2 Topic ────────►│◄── gRPC ──────────────┤
     │     (感知/控制数据)        │   (认知/决策服务)        │
     │                          │                        │
     ├──── gRPC Server ────────►│◄── WebSocket ──────────┤
     │     (运动执行/安全)        │   (事件推送/场景更新)    │
     │                          │                        │
     │                          ├── Envoy ──────────────►│
     │                          │   (gRPC-Web 代理)       │
     │                          │                        │
     └── ROS 2 + DDS ──────────┴─────────────────────────┘
           (共享内存同机通信)
```

#### 2.1.3 行为树 Action 节点详解

| Action 节点 | 功能描述 | 输入参数 | 输出 |
|-------------|----------|----------|------|
| `NavigateTo` | 移动底盘导航到目标坐标 | target: Pose6D | SUCCESS/FAILURE |

| `PickObject` | 抓取指定物体 | object_id: UUID | SUCCESS/FAILURE |
| `PlaceObject` | 将物体放置到目标 | object_id, target: Pose6D | SUCCESS/FAILURE |
| `AvoidObstacle` | 动态避障通过 | entity_id | SUCCESS/FAILURE |
| `Wait` | 等待指定时长 | duration: float(秒) | SUCCESS |
| `Speak` | 语音播报文本 | text: string | SUCCESS |
| `Observe` | 触发全面场景感知 | - | scene_id |
| `HITLConfirm` | 发起人在回路确认 | trajectories[], timeout_ms | selected_id |

#### 2.2.3 BrainAgent 核心接口

```python
class BrainAgent:
    """自研轻量 Agent 框架 (替代 LangChain)"""

    def recognize_intent(
        self, utterance: str, context: dict
    ) -> TaskIntent:
        """单轮意图识别，返回结构化 Intent"""

    def decompose_task(
        self, intent: TaskIntent, scene: Scene
    ) -> SubtaskList:
        """任务分解，返回有序子任务列表"""

    def generate_behavior_tree(
        self, subtasks: SubtaskList, skills: SkillRegistry
    ) -> BTDefinition:
        """生成行为树 JSON 定义"""

    def understand_scene(
        self, scene: Scene, task: Task
    ) -> SemanticContext:
        """场景语义理解"""

    def reason_common_sense(
        self, objects: list[DetectedObject]
    ) -> list[ObjectAttribute]:
        """常识推理（物体属性推断）"""
```

### 4.2 导入路径规范

```python
# Python: 绝对导入
from brain_ai.llm_agent.intent_parser import IntentParser   # ✅
from .intent_parser import IntentParser                     # ❌ 禁止跨模块相对导入
```

```cpp
// C++: 统一 include 前缀
#include <brain_core/ros2_bridge/node_manager.h>  // ✅
#include "node_manager.h"                         // ❌ 禁止裸路径
```

```typescript
// TypeScript: 路径别名
import { SceneView } from '@/components/scene-view/SceneView';  // ✅
import { SceneView } from '../../../components/scene-view/...'; // ❌ 禁止深层相对路径
```

### 4.3 模块组织原则

1. **每个模块包含**：`__init__.py` (Python) 或 `CMakeLists.txt` (C++) 或 `index.ts` (TypeScript)
2. **每个功能一个文件**：避免单个文件过大 (C++ < 500 行, Python < 300 行, TSX < 200 行)
3. **测试镜像目录结构**：`src/foo/bar.py → tests/foo/test_bar.py`
4. **配置集中管理**：每个项目的 `config/` 目录统一管理 YAML
