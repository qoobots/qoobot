## 1. 多入口交互矩阵

qoosvc 作为机器人端系统服务层，面向 3 类交互对象提供不同的 API 入口：

| 入口 | 目标用户 | 交互形式 | 技术栈 |
|------|---------|---------|--------|
| C++ Service API | 上层技能/C++ 模块 | 头文件 + 共享库 (.so/.dylib) | C++17 直接调用 |
| Python Service API | 上层技能/Python 模块 | Python 包导入 | pybind11 绑定 |
| DDS Topic | qoobrain 内部模块 / 多机器人 | 发布/订阅 | Fast DDS / Cyclone DDS |
| CLI 工具 | 开发者/运维 | 命令行 | Typer (Python) |

## 2. C++ Service API

### 2.1 API 概览

```cpp
// 语音交互服务
namespace qoosvc::voice {
    class VoiceService {
    public:
        // 唤醒词管理
        bool set_wake_word(const std::string& word, const std::string& lang);
        bool enable_wake_word(bool enable);
        
        // ASR 识别
        struct ASRResult {
            std::string text;
            std::string language;
            float confidence;
            bool is_final;
        };
        std::future<ASRResult> recognize(const AudioBuffer& audio);
        void register_asr_callback(std::function<void(const ASRResult&)> cb);
        
        // TTS 合成
        struct TTSConfig {
            std::string voice_id = "default";
            float speed = 1.0f;
            float pitch = 1.0f;
        };
        AudioBuffer synthesize(const std::string& text, const TTSConfig& config);
        
        // NLU 理解
        struct NLUResult {
            std::string intent;
            std::map<std::string, std::string> slots;
            float confidence;
        };
        NLUResult understand(const std::string& text);
        
        // 说话人识别
        struct SpeakerInfo {
            std::string id;
            std::string name;
            float confidence;
        };
        SpeakerInfo identify_speaker(const AudioBuffer& audio);
    };
}
```

### 2.2 导航引擎 API

```cpp
namespace qoosvc::navigation {
    // 导航目标
    struct NavigationGoal {
        std::string frame_id = "map";
        double x, y, z;
        double qx, qy, qz, qw = 1.0;  // 目标朝向四元数
    };
    
    // 导航结果
    struct NavigationResult {
        enum Status { SUCCESS, FAILED, CANCELED, RECOVERING };
        Status status;
        double distance_traveled;
        double duration_sec;
        std::string error_message;
    };
    
    class NavigationService {
    public:
        // 导航到目标点
        std::future<NavigationResult> navigate_to(const NavigationGoal& goal);
        bool cancel_navigation();
        
        // 路径查询
        struct PathPoint {
            double x, y, theta;
            double velocity;
        };
        std::vector<PathPoint> get_current_path() const;
        
        // 区域管理
        bool add_restricted_zone(const std::string& name, 
                                  const std::vector<Polygon>& polygons);
        bool add_speed_zone(const std::string& name, 
                             const std::vector<Polygon>& polygons, 
                             double max_speed);
        bool add_preferred_zone(const std::string& name,
                                 const std::vector<Polygon>& polygons);
        
        // 自主探索
        struct ExplorationConfig {
            double max_duration_sec = 600;
            double max_distance_m = 50;
            bool return_to_start = true;
        };
        std::future<NavigationResult> explore(const ExplorationConfig& config);
        
        // 动态避障
        void set_social_navigation(bool enable);  // 社会导航规范
        void set_personal_space(double radius_m);  // 个人空间半径
    };
}
```

```cpp
namespace qoosvc::diagnostics {

        std::vector<DiagnosticItem> items;

        std::vector<std::string> errors;
        std::vector<std::string> maintenance_suggestions;
        std::chrono::system_clock::time_point timestamp;
    };
    
    class DiagnosticsService {
```

## 3. Python Service API

### 3.1 语音交互

```python
from qoosvc.voice import VoiceService, WakeWordConfig, TTSConfig

voice = VoiceService()

# 唤醒词配置
voice.configure_wake_word(
    WakeWordConfig(word="Hey QooBot", language="zh", threshold=0.7)
)
voice.on_wake_word(lambda: print("Wake word detected!"))

# ASR 识别
async for result in voice.stream_recognize():
    print(f"ASR: {result.text} (confidence: {result.confidence:.2f})")

# NLU 理解
nlu_result = voice.understand("去厨房帮我拿一瓶水")
print(f"Intent: {nlu_result.intent}")
print(f"Slots: {nlu_result.slots}")  # {location: "厨房", object: "一瓶水"}

# TTS 合成
voice.speak("好的，正在前往厨房", 
            TTSConfig(voice_id="warm_female", speed=1.0))
```

### 3.2 导航引擎

```python
from qoosvc.navigation import NavigationService, NavigationGoal

nav = NavigationService()

# 导航到指定位置
goal = NavigationGoal(x=3.5, y=2.1, z=0.0, qz=0.707, qw=0.707)
result = await nav.navigate_to(goal)
if result.status == NavigationResult.SUCCESS:
    print(f"导航完成，行驶距离: {result.distance_traveled:.2f}m")

# 区域管理
nav.add_restricted_zone("楼梯口", polygons=[...])
nav.add_speed_zone("走廊", polygons=[...], max_speed=0.5)

# 自主探索
nav.explore(max_duration_sec=300, return_to_start=True)
```

### 3.3 人物交互

```python
from qoosvc.people import PeopleService

people = PeopleService()

# 人脸识别
@people.on_face_detected
def on_face(face):
    if face.recognized:
        print(f"识别到 {face.name} (confidence: {face.confidence:.2f})")
    else:
        print(f"检测到陌生人")

# 人物跟随
people.follow_person(name="张三", distance=1.5)

# 人物搜索
found = await people.search_person(name="李四", timeout_sec=120)
if found:
    print(f"在 {found.location} 找到李四")
```

## 4. DDS Topic 接口

### 4.1 Topic 列表

| Topic | 类型 | 方向 | QoS | 说明 |
|-------|------|------|-----|------|
| `/qoosvc/voice/asr_result` | ASRResult | 发布 | Reliable | ASR 识别结果 |
| `/qoosvc/voice/tts_audio` | AudioData | 发布 | BestEffort | TTS 合成音频 |
| `/qoosvc/nav/current_pose` | PoseStamped | 发布 | BestEffort | 导航当前位姿 (100Hz) |
| `/qoosvc/nav/current_path` | Path | 发布 | Reliable | 当前规划路径 |
| `/qoosvc/nav/goal` | PoseStamped | 订阅 | Reliable | 外部导航目标 |
| `/qoosvc/nav/cancel` | Empty | 订阅 | Reliable | 取消导航 |
| `/qoosvc/diag/health_report` | HealthReport | 发布 | Reliable | 健康报告 (1Hz) |
| `/qoosvc/diag/alert` | DiagnosticAlert | 发布 | Reliable | 诊断告警 |
| `/qoosvc/hmi/expression` | ExpressionCmd | 订阅 | Reliable | 表情指令 |
| `/qoosvc/hmi/light` | LightCmd | 订阅 | Reliable | 灯光指令 |
| `/qoosvc/hmi/gesture` | GestureEvent | 发布 | BestEffort | 手势事件 |
| `/qoosvc/people/face_detected` | FaceDetection | 发布 | BestEffort | 人脸检测结果 |
| `/qoosvc/charge/status` | ChargeStatus | 发布 | Reliable | 充电状态 (10Hz) |
| `/qoosvc/multi/discovery` | RobotInfo | 发布 | Reliable | 机器人发现广播 |
| `/qoosvc/multi/task_offer` | TaskOffer | 发布 | Reliable | 任务分配提议 |

## 5. CLI 工具

### 5.1 命令体系

```bash
# 服务管理
qoo svc list                          # 列出所有系统服务及状态
qoo svc start <service>               # 启动指定服务
qoo svc stop <service>                # 停止指定服务
qoo svc restart <service>             # 重启指定服务
qoo svc status <service>              # 查看服务详细状态

# 语音交互
qoo voice test                        # 语音交互测试（录音→识别→合成）
qoo voice asr <audio_file>            # 离线 ASR 识别音频文件
qoo voice tts "你好" --output test.wav # 文字转语音

# 导航
qoo nav go --x 3.5 --y 2.1            # 导航到指定坐标
qoo nav cancel                        # 取消当前导航
qoo nav path                          # 显示当前规划路径
qoo nav explore --duration 300        # 启动自主探索

# 诊断
qoo diag post                         # 运行开机自检
qoo diag report                       # 生成健康报告
qoo diag monitor                      # 启动实时监控（终端显示）
qoo diag calibrate --sensor imu       # 触发传感器校准

# 人机交互
qoo hmi expression --type happy       # 设置表情
qoo hmi light --color 00FF00 --pattern breathing  # 设置灯光
qoo hmi screen --show status          # 显示屏幕页面
```

## 6. 错误处理

### 6.1 错误码体系

| 错误码 | 含义 | 建议处理 |
|--------|------|---------|
| `SVC_NOT_FOUND` | 服务未安装或未启动 | 检查服务安装状态 |
| `SVC_BUSY` | 服务正在执行其他任务 | 等待或取消当前任务后重试 |
| `NAV_GOAL_UNREACHABLE` | 导航目标不可达 | 更换目标点或检查地图 |
| `NAV_PATH_BLOCKED` | 路径被障碍物阻塞 | 等待障碍物清除或重新规划 |
| `NAV_LOCALIZATION_LOST` | 定位丢失 | 触发重定位或手动接管 |
| `VOICE_NO_SPEECH` | 未检测到语音 | 靠近机器人或提高音量 |
| `VOICE_NOT_UNDERSTOOD` | 语音无法理解 | 换一种表达方式重试 |
| `DIAG_COMPONENT_FAILED` | 组件自检失败 | 查看详细诊断信息 |
| `CHARGE_DOCK_NOT_FOUND` | 未找到充电底座 | 确保底座通电且在可见范围内 |
| `CHARGE_DOCK_FAILED` | 充电对接失败 | 重试或手动放置到充电底座 |
| `PEOPLE_NOT_FOUND` | 未找到目标人物 | 检查人物是否在机器人感知范围内 |
| `MULTI_PEER_UNREACHABLE` | 协作机器人不可达 | 检查网络连接和对方状态 |
