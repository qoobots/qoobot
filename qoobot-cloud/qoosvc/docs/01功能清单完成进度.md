# qoosvc — 系统服务 功能清单与开发进度

> 最后更新：2026-06-26
>
> **定位**：机器人端系统级能力。语音助手、导航引擎、自诊断等，运行在 qoobrain 之上。
> **依赖**：qoobrain、qoobody
> **被依赖**：qooeco（技能市场可分发 qoosvc 的扩展能力）
>
> **设计理念**：qoosvc 是 QooBot 的 "Core Services" 层（对标 iOS Core Services / Android System Services）。它不是面向最终用户的技能，而是所有机器人应用共享的基础系统能力——语音交互、自主导航、空间理解、设备诊断、人机交互等。这些服务通过统一 API 暴露给上层技能使用。
