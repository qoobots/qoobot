### 1.2 核心表结构

#### devices — 设备主表

```sql
CREATE TABLE devices (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    serial_number   VARCHAR(64) NOT NULL UNIQUE,
    device_name     VARCHAR(100),
    model           VARCHAR(64) NOT NULL,
    hardware_version VARCHAR(32),
    firmware_version VARCHAR(32),
    user_id         UUID,                       -- 关联 qooauth users
    activation_status VARCHAR(16) NOT NULL DEFAULT 'pending',
    online_status   VARCHAR(16) NOT NULL DEFAULT 'offline',
    last_heartbeat  TIMESTAMPTZ,
    last_ip         INET,
    last_location   JSONB,
    config_json     JSONB DEFAULT '{}',
    metadata_json   JSONB DEFAULT '{}',
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    decommissioned_at TIMESTAMPTZ
);

CREATE INDEX idx_devices_user ON devices(user_id);
CREATE INDEX idx_devices_online ON devices(online_status);
CREATE INDEX idx_devices_model ON devices(model);
```

#### device_status — 设备实时状态 (ClickHouse)

```sql
CREATE TABLE device_status (
    device_id       UUID,
    timestamp       DateTime64(3),
    cpu_percent     Float32,
    memory_total_mb UInt32,
    memory_used_mb  UInt32,
    disk_total_gb   Float32,
    disk_used_gb    Float32,
    battery_percent Float32,
    battery_charging UInt8,
    temperature_c   Float32,
    wifi_rssi       Int16,
    network_latency_ms UInt16,
    current_task    String,
    errors_json     String DEFAULT '[]'
) ENGINE = MergeTree()
PARTITION BY toYYYYMM(timestamp)
ORDER BY (device_id, timestamp)
TTL timestamp + INTERVAL 90 DAY;
```

#### ota_packages — OTA 升级包

```sql
CREATE TABLE ota_packages (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name            VARCHAR(200) NOT NULL,
    version         VARCHAR(32) NOT NULL,
    type            VARCHAR(16) NOT NULL,       -- firmware/model/skill
    target_model    VARCHAR(64),
    file_url        VARCHAR(512) NOT NULL,
    file_size       BIGINT NOT NULL,
    checksum_sha256 VARCHAR(64) NOT NULL,
    release_notes   TEXT,
    min_fw_version  VARCHAR(32),
    created_by      VARCHAR(100),
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_ota_pkg_type ON ota_packages(type, target_model);
```

#### ota_tasks — OTA 任务

```sql
CREATE TABLE ota_tasks (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    package_id      UUID NOT NULL REFERENCES ota_packages(id),
    name            VARCHAR(200) NOT NULL,
    status          VARCHAR(16) NOT NULL DEFAULT 'draft',
    strategy_json   JSONB NOT NULL,             -- 灰度策略
    total_devices   INTEGER NOT NULL DEFAULT 0,
    success_count   INTEGER NOT NULL DEFAULT 0,
    failed_count    INTEGER NOT NULL DEFAULT 0,
    current_stage   INTEGER NOT NULL DEFAULT 0,
    scheduled_at    TIMESTAMPTZ,
    started_at      TIMESTAMPTZ,
    completed_at    TIMESTAMPTZ,
    created_by      VARCHAR(100),
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

#### ota_task_devices — OTA 任务设备明细

```sql
CREATE TABLE ota_task_devices (
    task_id         UUID NOT NULL REFERENCES ota_tasks(id),
    device_id       UUID NOT NULL REFERENCES devices(id),
    status          VARCHAR(16) NOT NULL DEFAULT 'pending',
    progress_pct    REAL NOT NULL DEFAULT 0,
    error_message   TEXT,
    started_at      TIMESTAMPTZ,
    completed_at    TIMESTAMPTZ,
    PRIMARY KEY (task_id, device_id)
);
```

#### models — 模型管理

```sql
CREATE TABLE models (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name            VARCHAR(200) NOT NULL,
    version         VARCHAR(32) NOT NULL,
    framework       VARCHAR(32) NOT NULL,       -- onnx/pytorch/tensorrt
    model_type      VARCHAR(32) NOT NULL,       -- llm/vision/speech/navigation
    storage_url     VARCHAR(512) NOT NULL,
    storage_size    BIGINT,
    config_json     JSONB,
    is_active       BOOLEAN NOT NULL DEFAULT true,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    UNIQUE (name, version)
);
```

#### inference_logs — 推理日志 (ClickHouse)

```sql
CREATE TABLE inference_logs (
    id              UUID,
    device_id       UUID,
    user_id         UUID,
    model_id        UUID,
    model_name      String,
    model_version   String,
    prompt_tokens   UInt32,
    completion_tokens UInt32,
    latency_ms      UInt32,
    cost            Float64,
    success         UInt8,
    error_code      String,
    timestamp       DateTime64(3)
) ENGINE = MergeTree()
PARTITION BY toYYYYMM(timestamp)
ORDER BY (device_id, timestamp)
TTL timestamp + INTERVAL 1 YEAR;
```

#### robot_clusters — 机器人集群

```sql
CREATE TABLE robot_clusters (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name            VARCHAR(200) NOT NULL,
    description     TEXT,
    owner_id        UUID NOT NULL,
    status          VARCHAR(16) NOT NULL DEFAULT 'active',
    config_json     JSONB DEFAULT '{}',
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE cluster_robots (
    cluster_id      UUID NOT NULL REFERENCES robot_clusters(id) ON DELETE CASCADE,
    device_id       UUID NOT NULL REFERENCES devices(id),
    role            VARCHAR(32) NOT NULL DEFAULT 'worker',
    joined_at       TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    PRIMARY KEY (cluster_id, device_id)
);
```

## 5. 数据安全

### 5.1 加密方案

| 数据类别 | 存储加密 | 传输加密 | 密钥管理 |
|---------|---------|---------|---------|
| 设备配置 | AES-256-GCM | mTLS 1.3 | KMS |
| 用户 PII | AES-256-GCM | TLS 1.3 | 用户加密密钥 |
| 模型文件 | AES-256-GCM (at rest) | TLS 1.3 | KMS |
| OTA 包 | 数字签名 (Ed25519) | TLS 1.3 | qooauth 证书体系 |
| 推理日志 | AES-256-GCM | TLS 1.3 | KMS |
| 经验数据 | 可选加密 | TLS 1.3 | 用户加密密钥 |

### 5.2 多租户隔离

- **数据库级别**：共享数据库，通过 `user_id` / `org_id` 行级隔离
- **对象存储**：按 `user_id` 前缀分目录，IAM 策略限制访问
- **缓存**：Key 前缀包含 `tenant:{org_id}`，Redis ACL 限制
- **消息队列**：Topic 按租户 Tag 过滤

### 5.3 数据保留策略

| 数据类型 | 保留期限 | 删除策略 |
|---------|---------|---------|
| 设备数据 | 注销后 30 天 | 硬删除 |
| 设备状态 (时序) | 90 天 | ClickHouse TTL |
| 推理日志 | 1 年 | ClickHouse TTL |
| 经验数据 | 180 天 | MinIO 生命周期 |
| OTA 包 | 永久 | 手动管理 |
| 审计日志 | 3 年 | 归档至冷存储 |
| 联邦学习梯度 | 30 天 | 自动清理 |
