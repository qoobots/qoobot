# qoocloud — 云端服务 功能清单与开发进度

> 最后更新：2026-06-27

---

## 定位

qoocloud 是 QooBot 的**云端大脑**，提供远程大模型推理、设备管理、OTA 升级、多机器人编排等云端核心能力。它连接每一台 QooBot 机器人与云端智能，是端云协同架构的关键枢纽。

| 维度 | 说明 |
|------|------|
| **定位** | 云端大脑：远程大模型推理、设备管理、OTA 升级、多机器人编排 |
| **对标** | AWS IoT Core + OpenAI API + GitHub Actions OTA 模式 |
| **依赖** | qoobrain（通信协议）、qooauth（身份认证） |
| **被依赖** | qoostore（技能分发）、qoocommunity |
| **设计文档** | ✅ 02架构设计 | ✅ 03交互设计 | ✅ 04数据设计 | ✅ 05项目目录结构 |

---

## 设计理念

1. **端云协同**：重型推理上云、轻量推理在端，无缝切换
2. **设备即代码**：所有机器人配置、策略、技能版本化，可追溯、可回滚
3. **弹性伸缩**：按设备数量、推理负载自动扩缩容
4. **安全优先**：所有通信链路加密、设备身份强验证、数据隔离
