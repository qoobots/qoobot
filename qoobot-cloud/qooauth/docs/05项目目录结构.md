## 2. Package 职责一览

| Package | 职责 | 关键框架 |
|---------|------|---------|
| `controller/` | REST API 控制器，请求参数校验，响应封装 | Spring MVC, Jakarta Validation |
| `grpc/` | gRPC 服务实现，供 qoocloud/qooeco/qoosvc 内部调用 | grpc-java |
| `service/` | 业务逻辑层，协调 repository/security/外部服务 | Spring, Bouncy Castle |
| `repository/` | Spring Data JPA Repository + MyBatis-Plus Mapper | JPA, MyBatis-Plus |
| `entity/` | JPA 实体定义，映射数据库表 | Jakarta Persistence |
| `dto/` | 请求/响应 DTO，API 契约 | Lombok, Jackson |
| `security/` | Spring Security 配置、JWT 过滤器、UserDetails | Spring Security, jjwt, Nimbus JOSE |
| `config/` | Bean 配置（PasswordEncoder, Redis, OAuth2 AS, gRPC） | Spring Boot |
| `exception/` | 全局异常定义、错误码、@ControllerAdvice | Spring MVC |

## 3. 与旧设计的对比

| 项目 | 旧设计（v0.2） | 新设计（v0.3） |
|------|-------------|-------------|
| 微服务数量 | 9 个（auth/user/device/oauth/session/key/audit/threat/gateway） | **1 个** |
| Maven 模块数 | 11 个（9 服务 + common + 父 POM） | **1 个** |
| 网关 | qooauth 自带 gateway 子模块 | **由 qoobot-cloud-gateway 统一承担** |
| 服务间通信 | OpenFeign + RocketMQ 跨进程调用 | **进程内方法调用** |
| 部署单元 | 9 个 JAR + 独立 K8s Deployment | **1 个 JAR，多实例水平扩展** |
| 数据一致性 | 分布式事务（Seata） | **本地事务** |

## 4. Java 代码规范

### 4.1 编码规范

- 遵循 [Google Java Style Guide](https://google.github.io/styleguide/javaguide.html)
- 包名：`com.qoobot.qooauth.{package}`
- 类名：`UpperCamelCase` — `AuthController`, `JwtTokenProvider`
- 方法名：`lowerCamelCase` — `validateToken()`, `createSession()`
- 常量：`UPPER_SNAKE_CASE` — `MAX_LOGIN_ATTEMPTS`
- 所有公共 API 必须有 Javadoc
- 使用 Lombok 减少样板代码
- 单元测试覆盖率 > 80%

### 4.2 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 包名 | 全小写 | `com.qoobot.qooauth.service` |
| 类名 | UpperCamelCase | `AuthController`, `JwtTokenProvider` |
| 接口名 | UpperCamelCase（不加 I 前缀） | `AuthService` |
| 实现类 | 接口名 + Impl | `AuthServiceImpl` |
| DTO 类 | 名词 + Request/Response | `LoginRequest`, `LoginResponse` |
| 数据库表 | snake_case 复数 | `api_keys`, `audit_logs` |
| REST 端点 | kebab-case 复数 | `/api/v1/auth/refresh-token` |
| SQL 迁移 | V{序号}__{描述}.sql | `V1__create_users.sql` |

### 4.3 日志规范

```java
@Slf4j
public class AuthService {

    public LoginResponse login(LoginRequest request) {
        log.info("User login attempt: email={}, ip={}",
            maskEmail(request.getEmail()), request.getIp());
        // 禁止记录密码明文
    }

    private String maskEmail(String email) {
        int at = email.indexOf('@');
        if (at <= 1) return email;
        return email.charAt(0) + "***" + email.substring(at);
    }
}
```

### 5.2 生产环境关键配置

```yaml
# application-prod.yml
spring:
  datasource:
    url: ${DB_URL}
    username: ${DB_USER}
    password: ${DB_PASSWORD}
  data:
    redis:
      host: ${REDIS_HOST}
      password: ${REDIS_PASSWORD}
      ssl: true

qooauth:
  jwt:
    access-token-expire: 3600
    refresh-token-expire: 2592000
    issuer: https://auth.qoobot.com
  security:
    password:
      min-length: 12
      require-uppercase: true
      require-lowercase: true
      require-digit: true
      require-special: true
      max-failed-attempts: 5
      lockout-duration: 1800
```

## 6. 构建与运行

### 6.1 本地启动

```bash
# 1. 启动基础设施
docker compose up -d postgres redis nacos

# 2. 编译
mvn clean compile

# 3. 启动
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

### 6.2 构建命令

```bash
mvn clean compile          # 编译
mvn test                   # 运行测试
mvn package -DskipTests    # 打包
docker build -t qooauth:latest .
```
