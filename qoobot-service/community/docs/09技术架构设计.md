## 1. 技术栈

| 领域 | 技术 | 用途 |
|:-----|------|------|
| 文档站点 | MkDocs Material / Docusaurus | 静态文档生成 |
| API 文档 | Sphinx + Breathe + autodoc | 自动 API 文档 |
| 全文搜索 | Algolia / Meilisearch | 文档搜索 |
| 多语言 | Crowdin / GitLocalize | 翻译管理 |
| Playground | Pyodide (CPython → WASM) + Monaco Editor | 浏览器运行 Python |
| 社区后端 | Spring Boot 3 + PostgreSQL | 贡献者/活动管理 |
| 社区前端 | Vue 3 + Nuxt 3 | 社区门户 SSR |
| 论坛 | Discourse (自托管) | 社区论坛 |
| 即时通讯 | Discord API (Bot 集成) | 社区聊天 |
| 视频 | YouTube + B站 + PeerTube | 视频托管 |
| 直播 | OBS + RTMP → 多平台 | 活动直播 |
| CI/CD | GitHub Actions | 自动化流水线 |

## 2. 文档站点技术选型

### 2.1 MkDocs Material vs Docusaurus

| 对比 | MkDocs Material | Docusaurus |
|:-----|:-------------:|:--------:|
| 语言 | Python | JavaScript |
| 构建速度 | 极快 | 快 |
| 主题 | Material 设计 | Infima |
| API 文档集成 | mkdocstrings | — |
| 版本化 | mike 插件 | 内置 |
| 多语言 | 内置 (i18n) | 内置 (i18n) |
| 社区 | Python 生态 | React 生态 |

**决策**：MkDocs Material——与现有 Python 技术栈一致，极快的构建速度，mkdocstrings 自动 API 文档。

### 2.2 Playground 架构

```
浏览器
  │
  ├── Monaco Editor (代码编辑)
  │
  ├── Pyodide (CPython 3.11 → WebAssembly)
  │     │
  │     ├── qoobot-sdk-wasm (预编译 SDK)
  │     └── 用户代码执行
  │
  └── 输出渲染 (Console / Matplotlib / 3D可视化)
```

## 3. 自动化工具链

### 3.1 GitHub Bot

```
社区机器人 (qoo-bot):
  · 新 Issue 自动标签 + 欢迎回复
  · PR 自动 CLA 检查
  · 代码规范自动检查
  · 需要 Review 的 PR 自动分配
  · 过期 Issue/PR 自动提醒
```

### 3.2 社区健康监控

```
指标面板:
  · 新 Issue 响应时间 (目标 <24h)
  · PR Review 时间 (目标 <48h)
  · 社区活跃度趋势
  · 贡献者留存率
  · 文档访问量
```

## 4. 多语言文档流水线

```
源文档 (中文)
    │
    ▼
Crowdin 翻译平台
    │
    ├── 英语翻译 → 英语文档
    ├── 日语翻译 → 日语文档
    └── 其他语言 (社区贡献)
    │
    ▼
GitHub PR (翻译更新)
    │
    ▼
自动构建 + 部署
```

### 4.1 翻译覆盖策略

| 文档 | 中文 | 英语 | 日语 |
|:-----|:---:|:---:|:---:|
| API 参考 | ✅ | ✅ | ✅ |
| 入门教程 | ✅ | ✅ | ✅ |
| 用户指南 | ✅ | ✅ | 🚧 |
| 示例库 | ✅ | ✅ | 🚧 |
| 博客 | ✅ | 精选 | ❌ |

## 6. 数据模型

### 6.1 贡献者数据

| 表 | 字段 | 说明 |
|:---|------|------|
| contributors | github_id, cla_signed, level, joined_at | 贡献者主表 |
| contributions | contributor_id, type, repo, count, date | 贡献记录 |
| cla_records | contributor_id, version, signed_at, ip | CLA 签署记录 |

### 6.2 活动数据

| 表 | 字段 | 说明 |
|:---|------|------|
| events | name, type, start_date, end_date, status | 活动主表 |
| event_registrations | event_id, user_id, ticket_type | 活动报名 |
| cfp_submissions | event_id, speaker, title, status | 演讲投稿 |
