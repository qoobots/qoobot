# qoochain — 仿生人供应链与制造

> 机器人的"富士康 + 全球供应链"：
> 生产制造标准、出厂标定规范、质量检测、BOM 参考设计。

## 定位

qoochain 是 QooBot 生态的供应链与制造基础设施，为机器人硬件厂商提供
从设计到量产的标准化流程，确保每一台 QooBot 兼容机器人的品质一致性。

## 模块

| 模块 | 说明 | 状态 |
|------|------|------|
| `manufacturing/` | 生产制造标准与流程 | ✅ 已完成 |
| `calibration/` | 出厂标定规范（传感器、执行器、运动学） | ✅ 已完成 |

| `traceability/` | 全生命周期追溯（从芯片到整机） | ✅ 已完成 |

## 制造流程

```
设计验证 ──► 工程样机 ──► 小批量试产 ──► 量产爬坡 ──► 稳定量产
   │            │            │              │            │
   │            │            │              │            │
   ├─ BOM 冻结  ├─ DFM 评审  ├─ 良率优化     ├─ 供应链扩产 ├─ 持续改进
   ├─ 仿真验证  ├─ 标定方案  ├─ 工艺定型     ├─ 质检标准化 ├─ 降本增效
   └─ 安规预审  └─ 测试大纲  └─ 认证送检     └─ 物流部署  └─ 售后建立
```

## 标定规范

| 标定项 | 方法 | 精度要求 |
|--------|------|----------|
| 运动学校准 | DH 参数辨识 | ±0.1mm 重复定位 |
| 相机标定 | 张正友标定法 | 重投影误差 <0.3px |
| 手眼标定 | AX=XB 求解 | ±0.5mm |
| 力传感器 | 六维标定台 | ±0.1N / ±0.01Nm |
| IMU | 六面校准 | 偏置 <0.1°/s |
| 激光雷达-相机 | 联合标定 | ±1cm @10m |
