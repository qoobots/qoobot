## 4. 标定流程

### 4.1 出厂标定流水线

```
机器人上料 → 固定工装 → 启动标定程序
    │
    ├── 1. IMU 标定 (2min)
    │       · 静止采集 → 零偏估计
    │       · 多姿态旋转 → 尺度/正交性
    │
    ├── 2. 相机标定 (3min)
    │       · 棋盘格多视角 → 内参/畸变
    │       · 立体标定 → 外参 (双目/深度)
    │
    ├── 3. LiDAR-相机外参 (2min)
    │       · 特征板 → 点云-图像匹配
    │
    ├── 4. 运动学标定 (2min)
    │       · 关节全范围运动 → 参数辨识
    │       · 零位标定 → 编码器零偏
    │
    ├── 5. 力觉标定 (1min)
    │       · 无负载 → 零漂补偿
    │       · 标准负载 → 灵敏度校准
    │
    ├── 6. 标定验证 (1min)
    │       · 各传感器精度复测
    │       · 全部通过 → 存档
    │
    └── 标定完成 (总计 <12min)
```

## 5. 质量门禁

```
来料 (IQC)
    │
    ▼
  合格? ──否──→ 退货/特采
    │ 是
    ▼
装配工序 1
    │
    ▼
  IPQC 检查 → 合格?
    │ 是        │ 否
    ▼           ▼
装配工序 2   返修/报废
    │
    ...
    │
    ▼
 OQC 终检
    ├── 全功能测试
    ├── 老化测试 (48h)
    ├── 外观检查
    └── 安全测试
    │
    ▼
  合格? ──否──→ 返工/降级
    │ 是
    ▼
出厂 ✓
```

| 阶段 | 指标 | 目标 |
|:-----|------|:----:|

| 终检 | 出厂合格率 | >99% |
