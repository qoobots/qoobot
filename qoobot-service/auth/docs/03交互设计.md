# 03 — 交互设计

> 版本：v0.1 | 最后更新：2026-04-22 | 状态：Draft
> 子项目：qooauth（账号与安全）| 架构：Java + Spring Cloud 微服务
> 对标：Apple ID / Google Account 统一身份基础设施

---

## 1. 多入口交互矩阵

qooauth 面向 6 类交互对象提供不同协议入口：

| 入口 | 目标用户 | 交互形式 | 技术栈 |
|------|---------|---------|--------|
| REST API (HTTPS) | Web Portal / Mobile App | JSON over HTTPS | Spring MVC |
| OIDC/OAuth2 端点 | 第三方应用 / 开发者 | OAuth2.0 / OIDC 标准协议 | Spring Authorization Server |
| gRPC API | 内部微服务 / qoocloud | Protocol Buffers | gRPC Java |
| C++ SDK | 机器人端 (onboard_core) | 动态库 + mTLS | C++17 + mbedTLS |
| Web SDK | Web 前端 | TypeScript OIDC Client | TS + OIDC-Client |
| Admin Console | 管理员 | Web 管理后台 | Vue3 + Element Plus |

## 2. REST API

### 2.1 用户认证 API

#### POST /api/v1/auth/register

注册新 QooBot ID 账户。

```
Request:
  POST /api/v1/auth/register
  Content-Type: application/json
  
  {
    "email": "user@example.com",
    "password": "SecureP@ss123",
    "nickname": "RobotMaster",
    "language": "zh-CN",
    "accept_tos": true,
    "accept_privacy": true
  }

Response 201:
  {
    "user_id": "uid_a1b2c3d4",
    "email": "user@example.com",
    "state": "EMAIL_VERIFICATION_PENDING",
    "verification_deadline": "2026-05-27T10:30:00Z"
  }

Response 409 (邮箱已注册):
  {
    "error": "EMAIL_ALREADY_EXISTS",
    "message": "该邮箱已注册 QooBot ID，请直接登录或使用账户恢复"
  }

Errors:
  - 400 INVALID_EMAIL_FORMAT
  - 400 WEAK_PASSWORD (长度<8 / 无大小写/数字/特殊字符)
  - 400 NICKNAME_TOO_LONG (>32字符)
  - 400 TOS_NOT_ACCEPTED
```

#### POST /api/v1/auth/login

用户登录并获取 Token。

```
Request:
  POST /api/v1/auth/login
  Content-Type: application/json
  User-Agent: QooBot-Web/1.0
  X-Device-Fingerprint: fp_xyz789...
  
  {
    "email": "user@example.com",
    "password": "SecureP@ss123"
  }

Response 200:
  {
    "access_token": "eyJhbGciOiJFZERTQSJ9...",
    "refresh_token": "rt_a1b2c3d4...",
    "token_type": "Bearer",
    "expires_in": 3600,
    "id_token": "eyJhbGciOiJFZERTQSJ9...",
    "user": {
      "sub": "uid_a1b2c3d4",
      "email": "user@example.com",
      "nickname": "RobotMaster",
      "avatar_url": "https://cdn.qoobot.com/avatars/uid_a1b2c3d4.jpg",
      "email_verified": true
    },
    "requires_mfa": false
  }

Response 401 (需要 MFA):
  {
    "error": "MFA_REQUIRED",
    "mfa_token": "mfa_session_xyz...",
    "available_methods": ["totp", "fido2"],
    "message": "请输入二次验证码"
  }

Response 403 (账户被锁定):
  {
    "error": "ACCOUNT_LOCKED",
    "locked_until": "2026-04-22T11:00:00Z",
    "message": "账户因多次失败尝试已被临时锁定 15 分钟"
  }

Response 429 (速率限制):
  {
    "error": "RATE_LIMITED",
    "retry_after": 60,
    "message": "请求过于频繁，请 60 秒后重试"
  }
```

#### POST /api/v1/auth/logout

```

### 2.2 OAuth 2.0 / OIDC 端点

#### GET /oauth2/authorize

OAuth 2.0 授权端点 (Authorization Code + PKCE)。

```
Request:
  GET /oauth2/authorize?
    response_type=code
    &client_id=qoostore_web
    &redirect_uri=https://store.qoobot.com/callback
    &scope=openid+profile+email
    &state=random_state_abc123
    &code_challenge=SHA256_BASE64URL(code_verifier)
    &code_challenge_method=S256

Response:
  HTTP 302 Found
  Location: https://store.qoobot.com/callback?
    code=auth_code_xyz...
    &state=random_state_abc123
```

#### POST /oauth2/token

Token 端点。

```
Request:
  POST /oauth2/token
  Content-Type: application/x-www-form-urlencoded
  Authorization: Basic <base64(client_id:client_secret)>
  
  grant_type=authorization_code
  &code=auth_code_xyz...
  &redirect_uri=https://store.qoobot.com/callback
  &code_verifier=plain_code_verifier_xyz

Response 200:
  {
    "access_token": "eyJhbGciOiJFZERTQSJ9...",
    "token_type": "Bearer",
    "expires_in": 3600,
    "refresh_token": "rt_xyz...",
    "id_token": "eyJhbGciOiJFZERTQSJ9...",
    "scope": "openid profile email"
  }
```

#### GET /oauth2/userinfo

UserInfo 端点 (OIDC)。

```
Request:
  GET /oauth2/userinfo
  Authorization: Bearer eyJhbGciOiJFZERTQSJ9...

Response 200:

## 3. gRPC API (服务间通信)

### 3.1 TokenService

```protobuf
syntax = "proto3";

package qooauth.token.v1;

service TokenService {
  // 验证 Access Token
  rpc VerifyToken(VerifyTokenRequest) returns (VerifyTokenResponse);
  
  // 签发新 Token
  rpc IssueToken(IssueTokenRequest) returns (IssueTokenResponse);
  
  // 吊销 Token
  rpc RevokeToken(RevokeTokenRequest) returns (RevokeTokenResponse);
}

message VerifyTokenRequest {
  string access_token = 1;
  repeated string required_scopes = 2;
}

message VerifyTokenResponse {
  bool valid = 1;
  string sub = 2;           // 用户/设备 ID
  string token_type = 3;    // ACCESS / REFRESH / ID
  repeated string scopes = 4;
  int64 expires_at = 5;     // Unix timestamp
  map<string, string> claims = 6;
}
```

### 3.2 UserService

```protobuf
syntax = "proto3";

package qooauth.user.v1;

service UserService {
  rpc GetUser(GetUserRequest) returns (GetUserResponse);
  rpc GetUserByEmail(GetUserByEmailRequest) returns (GetUserResponse);
  rpc ListUsers(ListUsersRequest) returns (ListUsersResponse);
  rpc UpdateUser(UpdateUserRequest) returns (UpdateUserResponse);
  rpc DeleteUser(DeleteUserRequest) returns (DeleteUserResponse);
}

message GetUserRequest {
  string user_id = 1;
}

message GetUserResponse {
  string user_id = 1;
  string email = 2;
  string nickname = 3;
  string avatar_url = 4;
  string language = 5;
  string timezone = 6;
  bool email_verified = 7;
  bool mfa_enabled = 8;
  int64 created_at = 9;
}
```

## 4. C++ SDK API (机器人端)

```cpp
// qooauth/device_sdk.h
namespace qooauth::device {

// 设备认证客户端
class DeviceAuthClient {
public:
    struct Config {
        std::string auth_server_url = "https://id.qoobot.com";
        std::string tls_ca_bundle;      // CA 证书包路径
        std::string device_cert_path;   // 设备证书路径
        std::string device_key_path;    // 设备私钥路径
    };
    
    explicit DeviceAuthClient(const Config& config);
    
    // 设备激活
    struct ActivateResult {
        bool success;
        std::string device_id;
        std::string certificate_pem;
        std::string binding_token;
        std::string error_message;
    };
    ActivateResult activate(const std::string& device_serial,
                            const std::string& hardware_model);
    
    // 获取设备 Token
    struct TokenResult {
        bool success;
        std::string access_token;
        int64_t expires_in;
        std::string error_message;
    };
    TokenResult get_device_token();
    
    // 检查 Token 是否有效
    bool is_token_valid();
    
    // 刷新 Token
    TokenResult refresh_token();
    
    // 设备解绑
    bool unbind();
};

} // namespace qooauth::device
```

## 5. Web SDK (前端)

```typescript
// @qoobot/auth-sdk
import { QooBotAuth } from '@qoobot/auth-sdk';

const auth = new QooBotAuth({
  clientId: 'qooweb_portal',
  issuer: 'https://id.qoobot.com',
  redirectUri: 'https://qoobot.com/callback',
  scopes: ['openid', 'profile', 'email']
});

// 登录
await auth.signIn();

// 获取当前用户
const user = await auth.getUser();
console.log(user.nickname);

// 获取 Access Token
const token = await auth.getAccessToken();

// 登出
await auth.signOut();
```

## 6. Admin Console (管理后台)

管理后台提供 Web 界面用于：

- **用户管理**：查看/搜索/冻结/删除用户账户
- **设备管理**：查看设备列表、远程锁定/擦除
- **API Key 管理**：审核/吊销第三方 API Key
- **OAuth 客户端管理**：注册/审核第三方 OAuth 应用
- **审计日志**：搜索/导出操作审计记录
- **安全告警**：查看异常登录/批量注册等安全事件
- **系统配置**：限流策略、MFA 策略、密码策略配置

技术栈：TypeScript + Vue3 + Element Plus，通过 REST API 调用后端服务。
