## 2. 模块职责详解

### 2.1 qooauth-common（公共模块）

| 职责 | 内容 |
|:-----|:-----|
| **错误码体系** | 统一 ErrorCode 枚举（AUTH_xxx, USER_xxx, DEVICE_xxx, CRYPTO_xxx） |
| **DTO/VO 定义** | 跨模块共享的数据传输对象 |
| **工具类** | JWT 工具、加密工具、校验工具 |
| **常量定义** | 业务常量、配置键、Redis Key 前缀 |

### 2.2 qooauth-auth（认证服务）

| 职责 | 内容 |
|:-----|:-----|
| **登录认证** | 用户名密码认证、argon2id 密码验证、会话创建 |
| **JWT 管理** | Ed25519 签名 JWT 签发/刷新/吊销 |
| **OAuth2/OIDC** | Authorization Server, Token Endpoint, UserInfo Endpoint |
| **SSO** | 单点登录 Cookie 管理、跨服务 Token 传递 |
| **2FA** | TOTP 验证、FIDO2/WebAuthn 注册与认证 |
| **API Key** | 生成/轮换/吊销，权限绑定 |

### 2.3 qooauth-user（用户服务）

| 职责 | 内容 |
|:-----|:-----|
| **用户注册** | 邮箱/手机验证、密码哈希、用户创建 |
| **个人资料** | 昵称/头像/语言/时区 CRUD |
| **账户管理** | 密码修改、账户恢复、账户注销 |
| **同意管理** | 隐私同意记录、撤回管理、合规版本 |

### 2.4 qooauth-device（设备服务）

| 职责 | 内容 |
|:-----|:-----|
| **设备注册** | 设备首次激活、证书签发 |
| **设备绑定** | 设备-用户关联、多设备管理 |
| **X.509 证书** | ECDSA P-256 证书生命周期管理 |
| **激活锁** | 防盗窃锁定、关联验证 |
| **访客模式** | 受限功能集、数据隔离 |

### 2.5 qooauth-crypto（加密服务）

| 职责 | 内容 |
|:-----|:-----|
| **E2EE** | X3DH 密钥协商、Double Ratchet 消息加密 |
| **Keychain** | 主密钥包裹、HSM 集成、密钥轮换 |
| **签名服务** | 技能签名、固件签名验证 |
| **完整性校验** | 二进制完整性、防篡改检测 |

### 2.6 qooauth-admin（管理服务）

| 职责 | 内容 |
|:-----|:-----|
| **RBAC 管理** | 角色/权限 CRUD、权限树维护 |
| **用户管理** | 后台用户查询、状态管理 |
| **审计日志** | 操作审计、合规审查 |
| **企业 MDM** | 设备策略、配置下发、合规报告 |

## 6. 安全边界

| 边界 | 措施 | 说明 |
|:-----|:-----|:-----|
| **网络边界** | Cloudflare WAF + K8s NetworkPolicy | 只允许 Gateway 入站 |
| **服务间** | mTLS (gRPC) | 所有内部通信加密 |
| **数据存储** | 静态加密 (AES-256) | PostgreSQL TDE + Redis encryption |
| **密钥** | HSM (Hardware Security Module) | 根密钥硬件保护 |
| **应用** | Spring Security + OAuth2 | 细粒度 RBAC |
| **用户数据** | E2EE + 字段级加密 | 敏感数据不出信任域 |
