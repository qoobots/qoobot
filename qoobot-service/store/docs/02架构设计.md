## 一、总体架构

qoostore 是 QooBot 的**机器人 App Store**，采用**端云协同架构**，对标 Apple App Store / Google Play Store。

### 2.2 商业化引擎 (Commerce Engine)

```
定价模型:
  ├── 免费 (Free)
  ├── 一次性购买 (One-time Purchase)  → License 设备绑定
  ├── 订阅制 (Subscription)           → 按月/年续费
  ├── 按使用付费 (Pay-per-Use)         → API 调用计次
  └── 免费试用 (Free Trial)            → 限时全功能试用

收益分成:
  ┌──────────────────────────────────────┐
  │  用户支付 $10                        │
  │    ├── 平台分成 30% → $3.00          │
  │    ├── 开发者分成 70% → $7.00        │
  │    └── 税务代扣 (按地区)              │
  │                                      │
  │  阶梯分成（累计收入 > $1M）:          │
  │    ├── 平台分成 15%                   │
  │    └── 开发者分成 85%                 │
  └──────────────────────────────────────┘

支付集成:
  ├── Stripe (国际信用卡)
  ├── 支付宝 (中国)
  ├── 微信支付 (中国)
  ├── Google Pay (Android)
  └── Apple Pay (iOS)
```

## 四、技术选型

| 组件 | 技术 | 说明 |
|------|------|------|
| **云端框架** | Java 21 + Spring Boot 3.x + Spring Cloud | 微服务架构 |
| **服务注册** | Nacos | 服务注册/发现/配置 |
| **API 网关** | Spring Cloud Gateway | 统一入口/限流/鉴权 |
| **数据库** | PostgreSQL 16 | 技能元数据/用户数据/订单 |
| **缓存** | Redis 7 | 热门技能/排行榜/会话 |
| **对象存储** | MinIO (兼容 S3) | .qooskills 包/截图/视频 |
| **消息队列** | RocketMQ | 异步审核/分析事件 |
| **搜索** | Elasticsearch 8 | 技能全文搜索 |
| **CDN** | Nginx + 边缘节点 | 技能包分发加速 |
| **端侧语言** | C++17 + Python 3.11 | 机器人端运行环境 |
| **容器化** | Docker + Containerd | 技能沙箱隔离 |
| **监控** | Prometheus + Grafana | 平台可观测性 |
| **熔断限流** | Sentinel | 微服务稳定性 |

## 五、安全架构

```
安全层级:
┌──────────────────────────────────────────┐
│  L1: 传输安全 (TLS 1.3 + mTLS)           │
├──────────────────────────────────────────┤
│  L2: 认证授权 (qooauth JWT + OAuth2)      │
├──────────────────────────────────────────┤
│  L3: 代码安全 (静态分析 + 签名验证)        │
├──────────────────────────────────────────┤
│  L4: 运行时安全 (沙箱隔离 + SELinux)       │
├──────────────────────────────────────────┤
│  L5: 数据安全 (加密存储 + 隐私标签)        │
└──────────────────────────────────────────┘

技能签名链:
  开发者私钥 → 签名 .qooskills → qoostore 验证 → qoostore 共签
  → 机器人端验证(开发者证书链 + 商店证书链)
```
