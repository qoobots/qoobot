# 03 — 交互设计

> 版本：v0.1 | 最后更新：2026-03-16 | 状态：Draft
> 子项目：qoocloud（云端服务）| 架构：Spring Cloud 微服务集群

---

## 1. 多入口交互矩阵

qoocloud 作为云端服务平台，面向 4 类用户提供交互入口：

| 入口 | 目标用户 | 交互形式 | 技术栈 |
|------|---------|---------|--------|
| REST API | 机器人端/Web 控制台/第三方 | HTTP/JSON | SpringDoc OpenAPI 3.0 |
| gRPC API | 内部微服务 / 高性能场景 | Protobuf | grpc-java |
| WebSocket | 机器人端实时通信 | 双向消息 | Spring WebSocket |
| Web 控制台 | 运维/管理员/开发者 | Web UI | Vue 3 + Element Plus |

### 2.3 错误码体系

| HTTP 状态码 | 错误码 | 含义 |
|------------|--------|------|
| 400 | INVALID_REQUEST | 请求参数错误 |
| 401 | UNAUTHORIZED | 未认证 |
| 403 | FORBIDDEN | 权限不足 |
| 403 | DEVICE_NOT_ACTIVATED | 设备未激活 |
| 404 | DEVICE_NOT_FOUND | 设备不存在 |
| 404 | MODEL_NOT_FOUND | 模型不存在 |
| 409 | DEVICE_ALREADY_REGISTERED | 设备已注册 |
| 409 | OTA_IN_PROGRESS | 设备有进行中的升级 |
| 429 | RATE_LIMITED | 请求频率过高 |
| 429 | GPU_QUEUE_FULL | GPU 推理队列已满 |
| 500 | INFERENCE_FAILED | 推理失败 |
| 500 | OTA_ROLLBACK_FAILED | 回滚失败 |
| 503 | GPU_UNAVAILABLE | GPU 资源不可用 |
| 503 | SERVICE_DEGRADED | 服务降级中 |

## 4. gRPC 内部 API

### 4.1 服务定义

```protobuf
syntax = "proto3";
package qoocloud.v1;

option java_multiple_files = true;
option java_package = "com.qoobot.qoocloud.grpc";

// 推理服务
service InferenceService {
  rpc Chat(ChatRequest) returns (stream ChatResponse);
  rpc VisionInference(VisionRequest) returns (VisionResponse);
  rpc BatchInference(BatchRequest) returns (BatchResponse);
  rpc ListModels(ListModelsRequest) returns (ListModelsResponse);
}

// 设备管理服务
service DeviceService {
  rpc RegisterDevice(RegisterDeviceRequest) returns (Device);
  rpc GetDeviceStatus(GetDeviceStatusRequest) returns (DeviceStatus);
  rpc UpdateDeviceConfig(UpdateConfigRequest) returns (google.protobuf.Empty);
  rpc SendCommand(SendCommandRequest) returns (CommandResponse);
}

// OTA 服务
service OTAService {
  rpc CheckUpdate(CheckUpdateRequest) returns (UpdateInfo);
  rpc ReportProgress(ReportProgressRequest) returns (google.protobuf.Empty);
  rpc RequestRollback(RequestRollbackRequest) returns (RollbackResponse);
}

// 数据同步服务
service DataSyncService {
  rpc UploadExperience(stream ExperienceData) returns (UploadResponse);
  rpc SyncKnowledge(SyncKnowledgeRequest) returns (SyncKnowledgeResponse);
  rpc SubmitGradient(SubmitGradientRequest) returns (google.protobuf.Empty);
  rpc GetGlobalModel(GetGlobalModelRequest) returns (ModelData);
}

// 多机器人编排服务
service OrchestraService {
  rpc RegisterRobot(RegisterRobotRequest) returns (ClusterInfo);
  rpc AssignTask(AssignTaskRequest) returns (Task);
  rpc ReportTaskStatus(ReportTaskStatusRequest) returns (google.protobuf.Empty);
}

// 可观测性服务
service ObservabilityService {
  rpc ReportMetrics(ReportMetricsRequest) returns (google.protobuf.Empty);
  rpc ReportTrace(stream TraceSpan) returns (google.protobuf.Empty);
}
```

## 6. CLI 工具

### 6.1 命令体系

```bash
# 设备管理
qoo cloud device list                      # 设备列表
qoo cloud device status <device_id>        # 设备状态
qoo cloud device diagnose <device_id>      # 远程诊断
qoo cloud device config <device_id>        # 查看/修改配置

# 推理服务
qoo cloud inference chat "你好"            # 对话推理
qoo cloud inference models                 # 可用模型列表
qoo cloud inference upload <model_path>    # 上传模型

# OTA 管理
qoo cloud ota create --package <pkg> --target <model>  # 创建升级任务
qoo cloud ota status <task_id>             # 任务状态
qoo cloud ota rollback <task_id>           # 回滚任务

# 可观测性
qoo cloud metrics device <device_id>       # 设备指标
qoo cloud logs device <device_id>          # 设备日志
qoo cloud trace <trace_id>                 # 链路追踪
```
