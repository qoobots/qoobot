## OpenLoong 硬件图纸 PDF 矢量检测

- 对 `mujoco/hardware/OpenLoongHardware` 目录下 329 个 PDF 进行了矢量/光栅检测
- 使用 pymupdf (fitz) 分析每个 PDF 的矢量路径数、位图数、文字块数
- 结果: 208 纯矢量(VECTOR) + 64 轻量矢量(VECTOR_LITE) + 57 混合(MIXED) + 0 扫描件(RASTER)
- 结论: 全部 PDF 均含矢量数据，PDF→DXF 转换路径完全可行，无需 OCR
- MIXED 文件中的位图是 3D 渲染预览或 Logo，不影响矢量图纸提取
- 同时确认目录中有 226 个 STEP 文件可用于 STEP→STL 3D 打印格式转换
- 检测脚本: `detect_pdf_vector.py`，结果数据: `pdf_vector_detection.json`
- 生成两张样本预览图: `preview_vector_sample.png`, `preview_mixed_sample.png`

## 3D打印与零件加工服务商调研

- 调研了4家国内主流制造服务商的工艺和文件格式要求:

  - 嘉立创CNC (jlc-cnc.com): CNC铣削/车削/钣金, 必须STEP/STP + 可选DXF/DWG/PDF
  - 魔猴网 (mohou.com): SLA/FDM/SLS/MJF/SLM/PolyJet, 主要STL, 50+材料, 按克计费
  - 未来工厂 (weilaigongchang.com): 3D打印+CNC+钣金+注塑, STEP/STL+DXF/DWG+PDF, 支持NDA
- 关键结论: 用户已有的226个STEP文件可直接上传给CNC服务商加工; 3D打印需先转STL
