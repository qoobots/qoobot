## 3. 材料选择指南

### 3.1 结构件材料

| 部位 | 材料 | 工艺 | 原因 |
|------|------|------|------|
| 主体骨架 | 7075-T6 铝合金 | CNC 加工 | 高强度/轻量 |
| 外壳覆盖件 | ABS/PC 合金 | 注塑 | 低成本/美观 |
| 关节壳体 | 钛合金 TC4 | CNC + 表面处理 | 高比强度/耐腐蚀 |
| 腿部结构 | Q235 钢 + 粉末涂层 | 焊接 + 喷涂 | 低成本/高刚性 |
| 末端法兰 | 不锈钢 316L | CNC | 耐腐蚀/高精度 |

### 3.2 防护等级分级

| 部位 | IP 等级 | 说明 |
|------|---------|------|
| 躯干主体 | IP54 | 防溅水 + 防尘 |
| 关节转动面 | IP67 | 可短时浸水 (线缆接口密封) |
| 传感器窗口 | IP65 | 防喷水 (光学窗口清洁维护) |
| 底盘底部 | IP54 | 防溅水 |
| 充电触点 | IP67 (对接时) | 充电底座需密封 |

## 4. 热管理方案

### 4.1 散热策略

```
热源分布:
┌────────────────────────────────┐
│  计算平台 (SoC): 15~40W        │
│  ├── 主动散热 (风扇 + 热管)     │
│  └── 导热路径: SoC→散热片→风道 │
│                                │
│  电机驱动板: 2~5W/个           │
│  ├── 被动散热 (铝合金壳体导热)  │
│  └── 导热路径: MOSFET→PCB→壳体 │
│                                │
│  电机绕组: 视工况 5~50W         │
│  └── 被动散热 (壳体传导 + 辐射) │
└────────────────────────────────┘

整体热设计:
- 躯干顶部: 进风口 + 防尘网
- 躯干底部: 出风口 + 风扇
- 关节: 散热鳍片 (壳体一体化)
- 热仿真: 环境 40°C 满载运行，SoC < 85°C
```
