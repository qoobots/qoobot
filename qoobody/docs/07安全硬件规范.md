# 07 — 安全硬件规范

> 版本：v0.1 | 最后更新：2026-06-26 | 状态：Draft

---

## 1. 概述

安全硬件是 QooBot 的最后一道防线，独立于主控计算平台运行。它负责碰撞检测、急停响应、安全状态管理，确保在任何主控故障情况下机器人仍能进入安全状态。

---

## 3. 安全控制器

### 3.1 安全 MCU 要求

| 参数 | 要求 |
|------|------|
| 架构 | 双核锁步 Cortex-R52/R82 或 TI Hercules |
| 安全认证 | SIL 2 / ASIL B 认证 |
| 内存 ECC | 片上 SRAM + Flash 均带 ECC |
| 独立时钟 | 独立于主控的时钟源 |
| 独立电源 | 独立 3.3V LDO 供电轨 |
| 看门狗 | 独立窗口看门狗 |
| 故障响应 | < 10ms (检测到执行) |

### 3.2 安全状态机

## 4. 碰撞检测硬件

### 4.1 检测方案

| 方案 | 传感器 | 检测范围 | 响应时间 | 可靠性 |
|------|--------|---------|---------|--------|
| 关节力矩异常 | 关节扭矩传感器 | 各关节 | < 1ms | 高 |
| 电机电流异常 | 驱动器电流采样 | 各关节 | < 1ms | 高 |
| 电容式接近 | 电容接近传感器 | 0~20cm | < 10ms | 中 |
| 电子皮肤 | 触觉阵列 | 接触面 | < 5ms | 中 |
| 视觉碰撞预测 | 深度相机 + NPU | 0~3m | < 50ms | 低 (预测) |

### 4.2 碰撞检测阈值

| 部位 | 力阈值 | 压力阈值 | 说明 |
|------|--------|---------|------|
| 手臂 (自由空间) | 50N | — | 轻微接触立即停止 |
| 手臂 (约束空间) | 150N | — | 准静态接触限值 |
| 手掌/手指 | 10N | 50 kPa | 精细操作 |
| 底盘 | 100N | — | 移动碰撞 |
| 头部 | 30N | — | 脆弱部位 |

### 4.3 碰撞响应策略

| 策略 | 响应 | 适用场景 |
|------|------|---------|
| STOP | 立即停止所有运动 | 碰撞力 > 阈值 |
| RETRACT | 反向运动 50mm 后停止 | 手臂碰撞 |
| COMPLIANT | 切换到零力模式 | 人机交互 |
| CONTINUE | 记录日志, 继续运动 | 轻微接触 |

## 5. 紧急制动系统

### 5.1 急停按钮

| 要求 | 规范 |
|------|------|
| 颜色 | 红色蘑菇头 (RAL 3000) |
| 背景 | 黄色 (RAL 1003) |
| 触点 | NC (常闭), 双通道 |
| 防护 | IP65 |
| 位置 | 头部 ×1, 背部 ×1, 底盘 ×1 |
| 标准 | ISO 13850 |

### 5.2 安全继电器回路

```
  24V 安全电源
      │
      ├── 急停按钮 1 (NC) ──┐
      │                     ├── 安全继电器 K1 线圈 ── GND
      ├── 急停按钮 2 (NC) ──┘
      │
      │         ┌── K1 触点 1 ── 动力电源接触器线圈
      │         │
      24V ──────┤
                │
                └── K1 触点 2 ── 安全 MCU 反馈
                
  强制导向继电器：K1 NO 与 NC 触点机械联动
  当线圈断电时, NO 必须断开, NC 必须闭合
```

### 5.3 制动器

| 类型 | 制动力 | 响应时间 | 适用 |
|------|--------|---------|------|
| 电磁抱闸 | 2× 额定扭矩 | < 30ms | 垂直轴 |
| 动态制动 | 电机绕组短路 | < 10ms | 水平轴 |
| 再生制动 | 能量回馈 | < 50ms | 减速 |
| 机械刹车 | 摩擦制动 | < 100ms | 底盘 |

## 6. 状态指示

### 6.1 LED 灯环

```
  头部 LED 灯环 (12×RGB LED)：
  
  状态颜色定义：
  ● 绿色 常亮    ── 正常待机
  ● 绿色 呼吸    ── 运行中
  ● 蓝色 旋转    ── 倾听/思考
  ● 黄色 闪烁    ── 警告 (低电量/故障预警)
  ● 红色 闪烁    ── 紧急停止/故障
  ● 白色 常亮    ── 充电中
  ● 彩虹 旋转    ── 启动/自检
```

### 6.2 安全模式指示

| 安全模式 | LED 颜色 | 蜂鸣器 | 显示屏 |
|---------|---------|--------|--------|
| NORMAL | 绿色呼吸 | 无 | 正常界面 |
| WARNING | 黄色闪烁 | 间歇 | 警告信息 |
| PROTECTIVE STOP | 红色慢闪 | 连续短鸣 | 停止原因 |
| EMERGENCY STOP | 红色快闪 | 连续长鸣 | 紧急停止 |
| FAULT | 红色常亮 | 连续 | 故障代码 |

## 7. 物理防篡改

### 7.1 防篡改措施

| 措施 | 说明 |
|------|------|
| 外壳开启检测 | 微动开关检测外壳是否被打开 |
| 安全螺丝 | Torx Security (带中心柱) 螺丝 |
| 防拆标签 | 关键接缝处防拆 VOID 标签 |
| 密封检测 | 气压传感器检测密封破坏 |
| 固件防篡改 | 安全启动 + 固件签名验证 |

### 7.2 篡改响应

```
  外壳开启检测触发：
  1. 安全 MCU 记录事件 (时间戳 + 状态)
  2. 擦除 RAM 中的密钥材料
  3. 锁定安全存储区
  4. 通知主控 / 云端
  5. 需要重新认证才能恢复
```

## 8. 安全验证与测试

### 8.1 出厂安全测试

| 测试项 | 方法 | 标准 |
|--------|------|------|
| 急停功能 | 按下急停, 测量制动时间 | < 100ms |
| 碰撞检测 | 施加标准力, 验证停止 | 力阈值 ±10% |
| 安全继电器 | 强制断开, 验证触点动作 | 必须断开 |
| 制动器 | 满载制动, 测量滑动距离 | < 5° (旋转) / < 10mm (直线) |
| 绝缘电阻 | 500V 兆欧表 | > 10MΩ |
| 接地连续性 | 接地阻抗测试 | < 0.1Ω |

### 8.2 定期自检

| 自检项 | 频率 | 方法 |
|--------|------|------|
| 安全 MCU 自检 | 每次上电 | CPU/内存/外设 BIST |
| 急停回路 | 每日 | 回路阻抗监测 |
| 制动器 | 每周 | 制动测试 (短时) |
| 传感器校准 | 每月 | 自动标定验证 |
