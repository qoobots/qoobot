# 02 — 感知系统接口规范

> 版本：v0.1 | 最后更新：2026-06-26 | 状态：Draft

---

## 1. 概述

感知系统是 QooBot 获取环境信息的硬件入口。本规范定义各类传感器的电气接口、通信协议、数据格式和时间同步机制，确保传感器数据可靠、低延迟地传输到计算平台。

---

## 2. 视觉传感器接口

### 2.1 相机类型与接口

| 相机类型 | 分辨率 | 帧率 | 接口 | 用途 |
|---------|--------|------|------|------|
| RGB 全局快门 | 1920×1200 | 60fps | MIPI CSI-2 4-lane | SLAM 前端、视觉里程计 |
| RGB-D (ToF) | 640×480 深度 | 30fps | MIPI CSI-2 / USB 3.0 | 近距离深度感知 |
| RGB-D (结构光) | 1280×720 深度 | 30fps | MIPI CSI-2 | 高精度物体重建 |
| 双目立体 | 1280×720×2 | 30fps | GMSL2 / MIPI 虚拟通道 | 中远距离深度估计 |
| 全景相机 | 1920×1920×2 (前后) | 30fps | MIPI CSI-2 ×2 | 环境感知、导航 |

### 2.2 MIPI CSI-2 通道分配

```
  主控 SoC MIPI CSI 接口
  ┌──────────────────────────────────────┐
  │ CSI-0 (4-lane) ── 前视 RGB (主 SLAM)  │
  │ CSI-1 (4-lane) ── 前视 RGB-D (ToF)   │
  │ CSI-2 (4-lane) ── 双目左 / 全景前     │
  │ CSI-3 (4-lane) ── 双目右 / 全景后     │
  └──────────────────────────────────────┘
```

### 2.3 相机触发同步

```
  安全 MCU                    相机组
  ┌──────┐                ┌──────────┐
  │ PWM  │ ──── FSIN ───► │ 相机 0   │
  │ 输出 │ ──── FSIN ───► │ 相机 1   │
  │      │ ──── FSIN ───► │ 相机 2   │
  │      │ ──── FSIN ───► │ 相机 3   │
  └──────┘                └──────────┘
  
  触发频率：30Hz (默认) / 60Hz (高速模式)
  同步精度：< 100μs (硬件触发)
```

## 3. 激光雷达接口

### 3.1 LiDAR 选型

| 类型 | 线数/点数 | 测距 | FOV (H×V) | 接口 | 用途 |
|------|---------|------|-----------|------|------|
| 机械旋转 | 16/32/64/128 线 | 150m | 360°×30° | Ethernet UDP | 室外导航 |
| 半固态 (MEMS) | 等效 128 线 | 200m | 120°×25° | Ethernet UDP | 前向感知 |
| 固态 Flash | 640×480 | 30m | 60°×45° | Ethernet / MIPI | 近场避障 |
| 2D 单线 | 1 线 | 25m | 360° | UART / USB | 低成本避障 |

### 3.2 LiDAR 数据协议

| 协议层 | 规范 |
|--------|------|
| 传输层 | UDP over Ethernet (1000BASE-T) / USB 3.0 |
| 数据包 | 自定义 UDP 包，包含时间戳、点云数据、反射率 |
| 时间同步 | PTP (IEEE 1588) 或 gPTP (IEEE 802.1AS) |
| 带宽 | 典型 20~100 Mbps（取决于线数和帧率） |

### 3.3 LiDAR 安装位置建议

```
  机器人顶部视图
  ┌──────────────────────────────┐
  │         360° 主 LiDAR        │ ← 头部/最高点
  │         (机械旋转式)          │
  │                              │
  │    ┌──────────────────┐     │
  │    │  前向固态 LiDAR   │     │ ← 胸部/前向
  │    │  (MEMS/Flash)     │     │
  │    └──────────────────┘     │
  │                              │
  │  ┌──────────┐  ┌──────────┐ │
  │  │ 后 2D    │  │ 后 2D    │ │ ← 底部/后向避障
  │  └──────────┘  └──────────┘ │
  └──────────────────────────────┘
```

## 4. IMU 惯性导航

### 4.1 IMU 性能要求

| 参数 | 导航级 | 消费级 |
|------|--------|--------|
| 陀螺仪零偏稳定性 | < 1°/h | < 10°/h |
| 加速度计零偏 | < 100μg | < 1mg |
| 输出频率 | ≥ 200Hz | ≥ 100Hz |
| 接口 | SPI | SPI / I²C |
| 温度补偿 | 内置 | 可选 |

### 4.2 IMU 安装位置

```
  推荐安装位置：躯干重心附近
  原因：
  - 接近旋转中心，减少杠杆臂效应
  - 振动隔离安装（橡胶减震垫）
  - 远离电机/风扇等振动源
```

## 5. 麦克风阵列

### 5.1 阵列规格

| 参数 | 规格 |
|------|------|
| 麦克风数量 | 4 / 6 / 8 颗 MEMS 数字麦克风 |
| 阵列布局 | 环形 / 线形 / L 形 |
| 间距 | 30~60mm（取决于目标频率） |
| 采样率 | 16kHz (ASR) / 48kHz (全频) |
| 位深 | 24-bit |
| 接口 | I²S / TDM 多通道 |
| AEC 参考信号 | 扬声器回路 I²S 输入 |

### 5.2 8 麦环形阵列布局

```
          MIC-0 (0°)
             │
     MIC-7 ──┼── MIC-1
      (315°) │   (45°)
             │
  MIC-6 ─────●───── MIC-2
   (270°)    │      (90°)
             │
     MIC-5 ──┼── MIC-3
      (225°) │   (135°)
             │
          MIC-4 (180°)
          
  圆直径：60mm
```

## 6. 传感器时间同步

### 6.1 同步架构

```
                    ┌────────────────┐
                    │  gPTP Grandmaster│
                    │  (交换机/主控)   │
                    └───────┬────────┘
                            │
          ┌─────────────────┼─────────────────┐
          │                 │                 │
  ┌───────▼──────┐ ┌───────▼──────┐ ┌───────▼──────┐
  │ LiDAR        │ │ 相机组       │ │ IMU          │
  │ PTP Slave    │ │ 硬件触发同步 │ │ 外部时钟输入  │
  │ UDP 时间戳   │ │ FSIN 信号   │ │ 1PPS + GPRMC │
  └──────────────┘ └──────────────┘ └──────────────┘
```

### 6.2 同步精度要求

| 传感器对 | 精度要求 | 同步方式 |
|---------|---------|---------|
| 相机 ↔ LiDAR | < 1ms | gPTP + 硬件触发 |
| 相机 ↔ IMU | < 100μs | 硬件触发 + 时间戳对齐 |
| 相机 ↔ 相机 (多目) | < 100μs | 同源 FSIN 信号 |
| 麦克风阵列 ↔ 相机 | < 1ms | gPTP |

### 6.3 时间戳格式

```
  统一时间戳格式：Unix 纳秒 (int64)
  
  数据包时间戳字段：
  ┌──────────────┬──────────────┬──────────────┐
  │ 采集时刻 (ns) │ 到达时刻 (ns) │ 序列号       │
  │ 64-bit       │ 64-bit       │ 32-bit       │
  └──────────────┴──────────────┴──────────────┘
```
