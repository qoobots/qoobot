# 06 — 结构与散热设计

> 版本：v0.1 | 最后更新：2026-02-08 | 状态：Draft

---

## 1. 概述

结构设计定义 QooBot 的物理形态、关节模组、防护等级、电磁兼容和可制造性要求。散热设计确保计算平台和执行器在热包络内稳定运行。

## 4. 线束管理

### 4.1 线束规范

| 线束类型 | 线规 | 屏蔽 | 弯曲寿命 | 用途 |
|---------|------|------|---------|------|
| 动力线 | 16~20 AWG | 无 | > 100万次 | 电机供电 |
| 信号线 | 26~30 AWG | 编织屏蔽 | > 500万次 | CAN/SPI/I²C |
| 高速信号 | 同轴/差分 | 铝箔+编织 | > 200万次 | MIPI/USB/Ethernet |
| 柔性扁平 | FFC 0.5mm pitch | 可选 | > 1000万次 | 关节内部走线 |

### 4.2 走线原则

```
  1. 动力线与信号线分离，间距 > 20mm
  2. 高速差分对保持等长、等间距
  3. 关节处预留足够的弯曲半径 (≥ 线径 ×10)
  4. 过孔/滑环处使用应力释放
  5. 全部连接器带锁扣
```

## 6. 电磁兼容 EMC

### 6.1 设计要点

| 项目 | 措施 |
|------|------|
| PCB 层叠 | ≥ 4 层, 完整地平面 |
| 屏蔽 | 关键模组金属屏蔽罩 |
| 滤波 | 电源入口 π 型滤波、共模扼流圈 |
| 接地 | 单点接地、数字/模拟地分割 |
| 电机 EMI | 电机相线磁环、驱动器 PWM 扩频 |

## 7. 可制造性设计 (DFM/DFA)

### 7.1 DFM 检查清单

| 检查项 | 要求 |
|--------|------|
| 最小壁厚 | ≥ 1.5mm (塑胶) / ≥ 1.0mm (金属) |
| 拔模角度 | ≥ 1.5° (外观面 ≥ 2°) |
| 圆角 | 内角 R ≥ 0.5× 壁厚 |
| 螺纹 | 避免自攻螺丝, 优先嵌件螺母 |
| 分型线 | 避开外观面 |
| 公差 | 一般 ±0.2mm, 配合面 ±0.05mm |

### 7.2 DFA 检查清单

| 检查项 | 要求 |
|--------|------|
| 零件数 | 尽量减少, 模块化设计 |
| 装配方向 | 优先 Z 轴装配 (从上往下) |
| 防错 | 不对称定位销/槽 |
| 紧固件 | 统一螺丝规格 (优先 M3/M4) |
| 连接器 | 防反插设计, 颜色编码 |
| 维修便利性 | 关键模块 (电池/计算) 可快速拆装 |

### 7.3 模块化装配层级

```
  整机
  ├── 头部模组 (可独立测试)
  │   ├── 相机模组
  │   ├── LiDAR 模组
  │   └── 麦克风阵列
  ├── 躯干模组 (可独立测试)
  │   ├── 计算平台模组
  │   ├── 电池模组
  │   └── 散热模组
  ├── 臂模组 ×2 (可独立测试)
  │   └── 关节模组 ×N
  └── 底盘模组 (可独立测试)
      ├── 驱动轮模组
      └── 悬挂模组
```

## 8. 散热设计

### 8.1 热源分布

| 热源 | 典型功耗 | 允许温升 | 散热方式 |
|------|---------|---------|---------|
| SoC | 15W | ΔT < 40°C | 主动 (热管+风扇) |
| 电机驱动器 | 5W×N | ΔT < 50°C | 传导至外壳 |
| DC/DC 电源 | 3W | ΔT < 40°C | 传导+自然对流 |
| 电池 | 2W (内阻) | ΔT < 15°C | 自然对流 |
| LiDAR | 10W | ΔT < 30°C | 内置风扇 |

### 8.2 散热路径

```
  SoC → 导热硅脂 → 铜均温板 → 热管 → 铝散热鳍片 → 风扇强制对流 → 外部
  
  电机驱动器 → 导热垫 → 外壳 (作为散热器) → 自然对流
  
  电池 → 自然对流 (底部通风孔)
```

### 8.3 风扇控制策略

| 温度区间 | 风扇占空比 | 噪音目标 |
|---------|-----------|---------|
| < 40°C | 0% (停转) | 0 dBA |
| 40~55°C | 20~50% | < 25 dBA |
| 55~70°C | 50~80% | < 35 dBA |
| > 70°C | 100% + 降频 | < 45 dBA |
| > 85°C | 紧急关机 | — |
