# 04 — 能源系统设计

> 版本：v0.1 | 最后更新：2026-02-08 | 状态：Draft

---

## 1. 概述

能源系统为 QooBot 所有子系统提供稳定、可靠的电力供应。包括电池管理 (BMS)、多级电源树、充电系统和低功耗管理。

---

## 2. 电池管理 BMS

### 2.1 电池组规格

| 参数 | 移动平台 | 固定基座 |
|------|---------|---------|
| 标称电压 | 48V (13S Li-ion) | 24V (7S Li-ion) |
| 容量 | 1~2 kWh | 可选 |
| 电芯类型 | 21700 / 18650 锂离子 | 同左 |
| 持续放电 | 20A (1kW) | 10A (240W) |
| 峰值放电 | 40A (2kW, 10s) | 20A (480W, 10s) |
| 充电功率 | 500W (快充) | 240W |
| 循环寿命 | ≥ 500 次 (80% SOH) | ≥ 800 次 |

### 2.2 BMS 功能

| 功能 | 说明 | 实现 |
|------|------|------|
| 电压监测 | 每节电芯电压 ±5mV | 专用 BMS AFE 芯片 |
| 温度监测 | 每 2~4 节一个 NTC 热敏电阻 | NTC + ADC |
| SOC 估算 | 库仑计数 + 开路电压校正 | 扩展卡尔曼滤波 |
| SOH 估算 | 内阻变化 + 容量衰减 | 在线参数辨识 |
| 均衡管理 | 被动均衡 (100mA) / 主动均衡 | BMS IC 内置 |
| 保护功能 | 过充/过放/过流/短路/过温 | 硬件 + 固件双保护 |

### 2.3 BMS 通信

```
  BMS ─── CAN FD ─── 安全 MCU
   │
   ├── SMBus / I²C ─── 主控 SoC (SOC/SOH/状态查询)
   │
   └── 独立硬线 ─── 安全继电器 (故障时直接断开)
```

## 3. 电源树设计

### 3.1 电源域划分

```
  电池 48V (或 24V)
      │
      ├── 48V BUS ──────────── 大型伺服电机
      │
      ├── 48V→24V DC/DC ───── 中小型伺服电机
      │                       └── 末端执行器
      │
      ├── 48V→12V DC/DC ───── 传感器供电 (LiDAR/深度相机)
      │                       └── 风扇/散热
      │
      ├── 48V→5V DC/DC ────── USB 接口
      │                       └── 逻辑电路
      │
      ├── 48V→3.3V LDO ────── 安全 MCU 独立供电
      │                       └── 安全回路
      │
      └── 48V→Vcore PMIC ──── SoC 核心供电
                              ├── VDD_CPU
                              ├── VDD_GPU
                              ├── VDD_NPU
                              └── VDD_DDR
```

### 3.2 上电时序

```
  时序要求：
  T0: 3.3V 安全 MCU (最先上电，最先稳定)
  T0 + 10ms: 3.3V 外设
  T0 + 20ms: 1.8V IO
  T0 + 30ms: VDD_DDR
  T0 + 40ms: VDD_CPU
  T0 + 50ms: VDD_GPU / VDD_NPU
  T0 + 100ms: 5V / 12V / 24V
  T0 + 200ms: 48V 动力电 (最后上电，防止浪涌)
  
  下电时序：反向顺序
```

### 3.3 电源隔离

| 隔离域 | 方式 | 原因 |
|--------|------|------|
| 动力电 ↔ 信号电 | DC/DC 隔离模块 | 防止电机噪声耦合到信号链 |
| 传感器电 ↔ 计算电 | LDO + π 型滤波 | 模拟/数字隔离 |
| 安全电 ↔ 主控电 | 独立 LDO 轨 | 安全 MCU 不受主控掉电影响 |

## 4. 充电系统

### 4.1 充电方案

| 方案 | 功率 | 接口 | 效率 | 适用 |
|------|------|------|------|------|
| 有线直流快充 | 500W | 专用充电口 (XT60) | > 95% | 主充电方式 |
| 无线充电 | 100W | Qi 2.0 / 定制线圈 | > 85% | 自主回充 |
| 底座触点充电 | 500W | 弹性触点 + 磁吸对位 | > 98% | 底座充电 |

### 4.2 自主回充流程

```
  状态机：
  IDLE ──► 低电量检测 (< 20% SOC)
          │
          ▼
  NAV_TO_DOCK ──► 导航至充电底座区域
          │
          ▼
  DOCK_ALIGN ──► 红外/视觉精确定位
          │        ±5mm / ±3° 精度
          │
          ▼
  DOCK_ENGAGE ──► 触点接触 / 线圈对位
          │
          ▼
  CHARGING ──► 充电握手协议 → 电流爬坡 → 恒流 → 恒压
          │
          ▼
  CHARGE_DONE ──► SOC > 90% 或用户唤醒
          │
          ▼
  UNDOCK ──► 脱离充电底座 → IDLE
```

### 4.3 充电安全

| 保护 | 方式 |
|------|------|
| 反接保护 | MOSFET 理想二极管 |
| 过流保护 | 可复位保险丝 + 电流监测 |
| 过温保护 | NTC + 降功率/停止充电 |
| 绝缘监测 | 电池组对地绝缘电阻 > 1MΩ |
| 通信加密 | 充电器 ↔ BMS CAN 通信 CRC + 认证 |

## 6. 低功耗模式

### 6.1 模式定义

| 模式 | 功耗 | 唤醒方式 | 恢复时间 | 状态保持 |
|------|------|---------|---------|---------|
| 运行 (Active) | 15~200W | — | — | 全功能 |
| 待机 (Standby) | < 10W | 语音/触摸/定时 | < 1s | 内存保持 |
| 休眠 (Suspend) | < 2W | 电源键/定时 | < 10s | 内存→存储 |
| 深度休眠 (Off) | < 0.5W | 电源键 | < 30s | 完全断电 |
| 运输模式 | < 0.1W | 充电器接入 | 首次启动 | 电池断开 |

### 6.2 待机功耗分解

```
  待机模式 (10W 预算)：
  计算平台 (LP mode)  .......... 2W
  安全 MCU ................... 0.5W
  麦克风阵列 (唤醒词) ......... 1W
  相机 (低帧率 5fps) .......... 2W
  Wi-Fi/BLE 待机 ............. 0.5W
  BMS + 电源树损耗 ............ 2W
  其他 ......................... 2W
  ─────────────────────────────
  合计 ........................ 10W
```
