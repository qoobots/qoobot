# 03 — 执行器接口规范

> 版本：v0.1 | 最后更新：2026-06-26 | 状态：Draft

---

## 1. 概述

执行系统是 QooBot 的"肌肉"和"手"，将控制指令转化为物理运动。本规范定义关节电机、末端执行器、紧急制动硬件的电气接口、通信协议和机械标准。

---

## 2. 关节电机规范

### 2.1 电机分类

| 类型 | 扭矩范围 | 转速 | 适用关节 | 驱动方式 |
|------|---------|------|---------|---------|
| 微型伺服 | 1~5 Nm | 60~120 rpm | 手指、腕部 | CANopen |
| 中小型伺服 | 10~50 Nm | 30~60 rpm | 肘部、头部 | CANopen / EtherCAT |
| 大型伺服 | 50~200 Nm | 15~30 rpm | 肩部、腰部 | EtherCAT CoE |
| 直线执行器 | 500~5000N | 50~200 mm/s | 升降机构 | CANopen / RS-485 |

### 2.2 通信协议栈

```
  ┌─────────────────────────────────────────┐
  │            应用层                        │
  │  CiA 402 (驱动与运动控制设备行规)         │
  │  ┌───────┬───────┬───────┬───────┐      │
  │  │  PP   │  PV   │  PT   │  CSP  │      │
  │  │ 位置  │ 速度  │ 转矩  │ 周期  │      │
  │  │ 模式  │ 模式  │ 模式  │ 同步  │      │
  │  └───────┴───────┴───────┴───────┘      │
  ├─────────────────────────────────────────┤
  │            协议层                        │
  │  ┌──────────────┐  ┌──────────────┐    │
  │  │  CANopen     │  │ EtherCAT CoE │    │
  │  │  (CiA 301)   │  │ (CANopen over│    │
  │  │              │  │  EtherCAT)   │    │
  │  └──────┬───────┘  └──────┬───────┘    │
  ├─────────┼─────────────────┼────────────┤
  │ 物理层  │ CAN FD          │ 100BASE-TX │
  │         │ (5Mbps)         │ (100Mbps)  │
  └─────────┴─────────────────┴────────────┘
```

### 2.3 控制模式

| 模式 | 缩写 | 说明 | 典型应用 |
|------|------|------|---------|
| 轮廓位置模式 | PP | 目标位置 + 速度/加速度限制 | 点到点运动 |
| 轮廓速度模式 | PV | 目标速度 | 传送带跟随 |
| 轮廓转矩模式 | PT | 目标转矩 | 力控制 |
| 周期同步位置 | CSP | 1kHz 周期同步位置指令 | 全身协调控制 |
| 周期同步速度 | CSV | 1kHz 周期同步速度指令 | 柔顺控制 |
| 周期同步转矩 | CST | 1kHz 周期同步转矩指令 | 阻抗/导纳控制 |

### 2.4 电机供电规范

| 电压等级 | 范围 | 峰值电流 | 适用电机 |
|---------|------|---------|---------|
| 24V DC | 18~30V | 10A | 微型/小型伺服 |
| 48V DC | 36~58V | 20A | 中小型伺服 |
| 320V DC | 280~380V | 30A | 大型伺服（工业） |

## 3. 末端执行器接口

### 3.1 机械法兰标准

```
  ┌──────────────────────────────────────┐
  │          QooBot 工具法兰标准          │
  │                                      │
  │  安装面外径：Φ60mm / Φ80mm / Φ100mm  │
  │  定位销：2×Φ4mm 对角                │
  │  螺栓：4×M4 / 6×M5 均布             │
  │  中心孔：Φ12mm (走线/气路)           │
  │                                      │
  │  ┌────────────────────────────┐     │
  │  │    ○  ○ (定位销)          │     │
  │  │                            │     │
  │  │ ●    中心孔    ●           │     │
  │  │                            │     │
  │  │    ●           ●           │     │
  │  │    (M4/M5 安装孔 ×4/6)     │     │
  │  └────────────────────────────┘     │
  └──────────────────────────────────────┘
```

### 3.2 电气接口

| 信号 | 引脚数 | 电压/电流 | 说明 |
|------|--------|---------|------|
| 供电 | 2 | 24V/5A | 工具电机/电磁阀供电 |
| 逻辑供电 | 1 | 5V/500mA | 工具 MCU 供电 |
| CAN 总线 | 2 | CAN FD | 工具控制指令 |
| GPIO | 4 | 3.3V | 工具状态/传感器 |
| 气路控制 | 2 | 24V PWM | 电磁阀/真空发生器 |

### 3.3 工具识别

```
  每次换刀流程：
  1. 法兰接触 → 工具识别芯片上电
  2. 主控读取 1-Wire EEPROM 工具 ID
  3. 加载对应工具参数（质量、惯量、控制参数）
  4. 工具就绪信号 → 开始操作
```

## 4. 全身运动学参考

### 4.1 自由度分配建议

| 关节链 | 自由度数 | 关节类型 | 说明 |
|--------|---------|---------|------|
| 头部 | 2~3 DOF | 旋转 | Pan/Tilt + Roll (可选) |
| 躯干 | 1~3 DOF | 旋转 | 腰部旋转/俯仰 |
| 单臂 | 6~7 DOF | 旋转 | 肩3 + 肘1 + 腕3 |
| 单手掌 | 1~5 DOF | 旋转/直线 | 简易夹爪 ~ 灵巧手 |
| 移动底盘 | 2~3 DOF | 差速/全向 | 平移 + 旋转 |

**典型配置**：双臂 7+7 + 头部 2 + 躯干 1 + 底盘 3 = **20 DOF**

### 4.2 关节限位

```
  推荐限位方式：
  1. 软件限位（主控 qoobrain 运动规划）
  2. 固件限位（电机驱动器内部）
  3. 电气限位（限位开关，接安全 MCU）
  4. 机械限位（物理挡块，最后一道防线）
```

## 5. 力控与柔顺

### 5.1 力感知方案

| 方案 | 传感器 | 精度 | 延迟 | 适用场景 |
|------|--------|------|------|---------|
| 关节力矩 | 关节扭矩传感器 | ±0.1Nm | < 1ms | 全身力控 |
| 末端力 | 六维力/力矩传感器 | ±0.5N | < 1ms | 精密装配 |
| 电机电流 | 驱动器电流环 | ±5% 额定 | < 1ms | 碰撞检测 |
| 电子皮肤 | 触觉阵列 | ±0.1N | < 5ms | 人机交互 |

### 5.2 阻抗/导纳控制

```
  控制架构：
  参考轨迹 → ┌──────────┐     ┌──────────┐     ┌──────────┐
            │ 阻抗控制  │ ──► │ 位置控制  │ ──► │ 电机驱动  │
  力反馈  → │ M·ẍ+D·ẋ+K·x │     │ PID 内环  │     │ 电流环    │
            │ = F_ext    │     └──────────┘     └──────────┘
            └──────────┘
  
  参数：
  M (惯性): 可调 0.1~10× 实际惯量
  D (阻尼): 临界阻尼的 0.5~2×
  K (刚度): 0 (零力) ~ 5000 N/m (高刚度)
```

## 6. 紧急制动硬件

### 6.1 急停回路

```
  ┌──────────────────────────────────────────────────────┐
  │                    急停安全回路                        │
  │                                                      │
  │  ┌──────────┐     ┌──────────┐     ┌──────────┐     │
  │  │ 急停按钮  │ ──► │ 安全继电器│ ──► │ 动力电源  │     │
  │  │ (NC 触点) │     │ (强制导向)│     │ 接触器    │     │
  │  └──────────┘     └──────────┘     └──────────┘     │
  │        │                │                            │
  │        │         ┌──────▼──────┐                     │
  │        │         │  安全 MCU   │                     │
  │        │         │  独立监控   │                     │
  │        │         └─────────────┘                     │
  │        │                                            │
  │        └── 冗余回路 ──────────────────────────►     │
  │                   动力电源                            │
  └──────────────────────────────────────────────────────┘
  
  要求：
  - 双通道冗余 (Cat. 3 / PL d 以上)
  - 急停响应时间 < 100ms
  - 制动距离 < 200mm (全速→停止)
```

### 6.2 制动方式

| 方式 | 响应时间 | 制动力 | 适用 |
|------|---------|--------|------|
| 电机电磁制动 | < 50ms | 额定扭矩 ×2 | 关节保持 |
| 动力断电 | < 100ms | 全关节失能 | 紧急停止 |
| 机械抱闸 | < 30ms | 额定扭矩 ×3 | 垂直轴防坠落 |
| 底盘急停 | < 200ms | 最大减速度 | 移动平台 |

## 7. 线性执行器接口 (工业场景)

### 7.1 接口规范

| 参数 | 规格 |
|------|------|
| 行程 | 100~1000mm |
| 速度 | 50~500 mm/s |
| 推力 | 500~5000N |
| 通信 | CANopen (CiA 402) / RS-485 Modbus RTU |
| 供电 | 48V DC |
| 反馈 | 编码器 (增量/绝对) + 限位开关 |

## 8. 执行器供电拓扑

```
  电池/电源
      │
      ├── 48V BUS ─── 大型伺服电机 (关节 1~4)
      │
      ├── 24V BUS ─┬─ 中小型伺服电机 (关节 5~N)
      │            ├─ 末端执行器
      │            └─ 线性执行器
      │
      └── 紧急制动 ─── 安全继电器 ─── 断开所有动力电源
```
