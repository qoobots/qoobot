## 3. 模块职责一览

| 模块 | 职责 | 关键依赖 |
|------|------|---------|
| `engine/checklist_gen.py` | 按产品+市场生成检查清单 | knowledge/, products/ |
| `engine/gap_analyzer.py` | 对比现状与法规要求，生成差距报告 | checklists/, knowledge/ |
| `engine/doc_generator.py` | 使用 Jinja2 模板生成合规文档 | templates/, products/ |
| `engine/monitor.py` | 爬取法规变化，分析影响 | httpx, bs4 |
| `engine/risk_assessor.py` | HAZOP/FMEA 辅助分析 | templates/risk_*.md.j2 |
| `knowledge/loader.py` | 加载和验证知识库 YAML 文件 | PyYAML |
| `knowledge/parser.py` | 解析法规原文文本 | re, text processing |
| `knowledge/indexer.py` | 知识库全文搜索索引 | SQLite FTS5 |
| `tracker/certification.py` | 认证生命周期管理 | db/repository.py |
| `tracker/progress.py` | 合规完成度追踪 | db/repository.py |
| `tracker/audit.py` | 操作审计记录 | db/repository.py |
| `cli/` | 命令行界面 | Typer, Rich |
| `web/` | REST API 服务 | FastAPI |
| `web-ui/` | Web 仪表盘前端 | Vue 3 + Element Plus |

## 4. 构建与运行

### 4.1 本地安装

```bash
# 安装
pip install -e .

# CLI 使用
qoo-regs --help
qoo-regs check QooBot-Home --market eu
qoo-regs cert dashboard

# Web 仪表盘
pip install -e ".[web]"
uvicorn qoocompliance.web.app:app --reload
```

### 4.2 知识库维护

```bash
# 添加新法规
qoo-regs kb add safety/iso_3691_4.yaml

# 搜索法规
qoo-regs kb search "功能安全"

# 列出某类别所有法规
qoo-regs kb list --category wireless
```

### 4.3 定期任务

```bash
# 每日法规变更检查
qoo-regs monitor check

# 认证到期检查
qoo-regs cert check-expiry

# 生成月度合规报告
qoo-regs report monthly --product QooBot-Home
```
