# 03 — 交互设计

> 版本：v0.1 | 最后更新：2026-06-27 | 状态：Draft
> 子项目：qoocompliance（法规合规）| 架构：文档驱动的合规知识库 + 自动化检查工具

---

## 1. 多入口交互矩阵

qoocompliance 作为合规知识库与工具平台，面向 3 类用户提供交互入口：

| 入口 | 目标用户 | 交互形式 | 技术栈 |
|------|---------|---------|--------|
| CLI 工具 | 工程师/合规专员 | 命令行 | Typer + Rich (Python) |
| 知识库文件 | 所有团队 | Markdown + YAML | Git 版本控制 |
| 合规仪表盘 | 管理层/项目经理 | Web UI | Vue 3 + Element Plus |

---

## 2. CLI 工具

### 2.1 命令体系

```bash
# 合规检查
qoo regs check <product> --market <market>     # 按产品+市场生成检查清单
qoo regs check <product> --all-markets         # 所有目标市场检查
qoo regs gap <product> --market <market>       # 差距分析
qoo regs risk <product> --method fmea          # 风险评估 (HAZOP/FMEA)

# 文档生成
qoo regs doc ce-tech-file <product>            # 生成 CE 技术文件
qoo regs doc doc <product> --market eu         # 生成合规声明
qoo regs doc dpia <product>                    # 生成 DPIA 报告
qoo regs doc fcc-report <product>              # 生成 FCC 测试报告模板

# 认证追踪
qoo regs cert list                             # 列出所有认证
qoo regs cert status <cert_id>                 # 认证状态
qoo regs cert renew <cert_id>                  # 标记续期
qoo regs cert dashboard                        # 认证仪表盘

# 法规监控
qoo regs monitor status                        # 监控状态
qoo regs monitor check                         # 手动检查法规更新
qoo regs monitor alerts                        # 查看法规变更告警

# 知识库管理
qoo regs kb list                               # 列出知识库条目
qoo regs kb search <keyword>                   # 搜索法规
qoo regs kb update <regulation>                # 更新法规条目
```

## 3. 知识库文件格式

### 3.1 法规条目 (YAML)

```yaml
# knowledge/safety/iso_13482.yaml
regulation:
  id: "ISO_13482_2014"
  title: "Robots and robotic devices — Safety requirements for personal care robots"
  version: "2014"
  category: "safety"
  markets: ["global", "eu", "cn"]
  
  summary: >
    规定了个人护理机器人的安全要求，包括风险降低、
    安全防护等级、使用信息安全等。

  check_items:
    - id: "13482_RISK_ASSESSMENT"
      title: "风险识别与评估"
      description: "完成机器人全生命周期的风险识别与评估"
      priority: P0
      verification_method: "document_review"
      applicable_products: ["home_robot", "medical_robot"]
      
    - id: "13482_SAFEGUARDING"
      title: "安全防护措施"
      description: "实施与风险等级匹配的安全防护措施"
      priority: P0
      sub_items:
        - "紧急停止装置"
        - "安全限速"
        - "碰撞检测与响应"
        - "安全距离保持"
        
    - id: "13482_INSTRUCTION"
      title: "使用信息安全"
      description: "提供完整的安全使用说明和警告标识"
      priority: P1
      verification_method: "document_review"
```
