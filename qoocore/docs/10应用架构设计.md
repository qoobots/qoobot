## 2. 模块职责

| 模块 | 路径 | 职责 |
|:-----|:-----|:-----|
| **core/engine** | 推理引擎入口 | load_model/infer/infer_async/batch/multi_model |
| **core/scheduler** | 实时调度器 | 固定优先级抢占、时间片轮转、多级队列 |
| **core/pipeline** | 流水线执行器 | 感知→认知→规划阶段并行 |
| **core/dispatcher** | 后端分派器 | 模型→后端匹配、异构分派 |
| **compiler/onnx** | ONNX 解析器 | ONNX Runtime C++ API 图解析 |
| **compiler/ir** | IR 构建器 | IrGraph/IrBuilder/IrValidator |
| **compiler/optimizer** | 图优化器 | 3级Pass(常量折叠/算子融合/内存复用) |
| **compiler/quantizer** | 量化器 | INT8/FP16 PTQ、校准统计 |
| **compiler/pruner** | 剪枝器 | L1/L2通道剪枝 + 幅值剪枝 |
| **compiler/codegen** | 代码生成器 | NPU/GPU/CPU后端代码生成 |
| **hal/npu** | NPU 抽象层 | QNN/BPU/RKNN HAL + dlopen |
| **hal/gpu** | GPU 后端 | CUDA/OpenCL/Vulkan统一抽象 |
| **hal/dsp** | DSP 后端 | Hexagon/CEVA推理卸载 |
| **hal/cpu** | CPU 后端 | Neon/SSE/AVX SIMD内核 |
| **hal/ion** | ION 分配器 | DMA-BUF零拷贝内存 |
| **memory** | 内存管理 | Arena/权重池/激活池/LRU |
| **op/** | 专用算子 | vision/lidar/bev/trajectory/signal/multimodal |
| **observability** | 观测模块 | Profiler/Monitor/ChromeTrace |

## 3. 核心数据流

### 3.1 编译数据流

```
model.onnx
    │
    ▼
┌──────────────┐
│ ONNX Parser  │  1. onnxruntime::Graph → IrNode 列表
└──────┬───────┘
       ▼
┌──────────────┐
│ IR Builder   │  2. IrNode → IrGraph (DAG + 拓扑排序)
└──────┬───────┘
       ▼
┌──────────────┐
│ Optimizer    │  3. O1常量折叠 → O2算子融合 → O3内存复用
└──────┬───────┘
       ▼
┌──────────────┐
│ Quantizer    │  4. 校准数据→scale/zero_point→权重转换
└──────┬───────┘
       ▼
┌──────────────┐
│ Pruner       │  5. 通道L1/L2评分→阈值裁剪→精度验证
└──────┬───────┘
       ▼
┌──────────────┐
│ CodeGen      │  6. IrGraph→NPU指令/GPU kernel/CPU向量化
└──────┬───────┘
       ▼
┌──────────────┐
│ Packager     │  7. 128字节Header + 序列化图 + 权重 → .qoomodel
└──────────────┘
```

### 3.2 推理数据流

```
.qoomodel ──→ load_model ──→ 解析Header → 还原IrGraph → 创建Runtime
                                        │
                          ┌─────────────┼─────────────┐
                          ▼             ▼             ▼
                      NPU后端       GPU后端        CPU后端
                      (首选)       (备选)         (兜底)
                          │             │             │
                          └─────────────┼─────────────┘
                                        │
                                   ┌────┴────┐
                                   │ Scheduler│
                                   │ 实时调度 │
                                   └────┬────┘
                                        │
                              ┌─────────┴─────────┐
                              │                   │
                         同步 infer          异步 infer_async
                              │                   │
                         直接返回             callback/promise
```

## 5. 安全边界

| 边界 | 措施 |
|:-----|:-----|
| **模型完整性** | .qoomodel SHA256 + Ed25519 签名 |
| **进程隔离** | qoocore 独立进程或动态库隔离 |
| **内存保护** | Arena 内存边界检查、W^X 内存策略 |
| **侧面信道** | 恒定时间加密操作、随机化执行顺序 |
| **降级策略** | NPU故障→GPU→CPU自动回退 |
