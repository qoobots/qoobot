## 2. 测试分层

### 2.1 L1 — 单元测试

| 维度 | 要求 |
|:-----|:-----|
| **框架** | Google Test + Google Mock |
| **覆盖率目标** | ≥ 85% 行覆盖率, ≥ 75% 分支覆盖率 |
| **Mock 策略** | MockNpuHal 模拟 NPU 行为 |
| **重点覆盖** | IR 构建/验证、图优化 Pass、量化计算、算子正确性 |

```cpp
// 示例：IrGraph 构建测试
TEST(IrGraphTest, BuildSimpleConvNet) {
    IrBuilder builder;
    auto input = builder.AddInput("input", {1, 3, 224, 224});
    auto conv = builder.AddConv(input, "conv1", 64, 3, 1);
    auto bn = builder.AddBatchNorm(conv);
    auto relu = builder.AddRelu(bn);
    auto output = builder.SetOutput(relu);
    
    IrGraph graph = builder.Build();
    EXPECT_EQ(graph.Nodes().size(), 5);
    EXPECT_TRUE(IrValidator::Validate(graph));
}

// 示例：量化精度测试
TEST(QuantizerTest, Int8PrecisionLoss) {
    auto graph = BuildTestGraph();
    Quantizer quantizer(graph, calibration_data);
    auto result = quantizer.Quantize(QuantizationMode::INT8);
    
    // INT8 量化精度损失 < 1%
    EXPECT_LT(result.accuracy_drop, 0.01f);
}
```

## 3. 质量门禁

### 3.1 PR 检查清单

```
□ clang-format 代码格式化通过
□ clang-tidy 静态分析零错误
□ cppcheck 零高危告警
□ 编译零警告 (-Wall -Wextra -Werror)
□ 单元测试全通过 (Debug + Release)
□ Address Sanitizer 通过
□ Undefined Behavior Sanitizer 通过
□ 代码覆盖率 ≥ 85%
□ 性能基准回归 ≤ 5%
□ Doxygen 文档无破损引用
```

### 3.2 Sanitizer 测试矩阵

| Sanitizer | 编译标志 | 目的 |
|:----------|:---------|:-----|
| **AddressSanitizer** | `-fsanitize=address` | 内存错误 (OOB/use-after-free/double-free) |
| **UndefinedBehaviorSanitizer** | `-fsanitize=undefined` | 未定义行为 (整数溢出/空指针/移位) |
| **ThreadSanitizer** | `-fsanitize=thread` | 数据竞争 |
| **LeakSanitizer** | `-fsanitize=leak` | 内存泄漏 |
| **MemorySanitizer** | `-fsanitize=memory` | 未初始化内存读取 |

## 6. 安全测试

| 测试类型 | 工具 | 频率 |
|:---------|:-----|:-----|
| **模糊测试** | libFuzzer + custom mutators | 每版本 |
| **内存安全** | ASAN + UBSAN + MSAN | 每次 PR |
| **模型签名** | 篡改 .qoomodel → 验证拒绝加载 | 每次发布 |
| **对抗样本** | 已知对抗攻击样本库 | 每版本 |
| **资源耗尽** | 超大模型/超多模型/超大批量 | 每版本 |

```cpp
// Fuzz 测试示例
extern "C" int LLVMFuzzerTestOneInput(const uint8_t* data, size_t size) {
    // 尝试解析随机数据为 .qoomodel
    auto result = Engine::Create(EngineConfig{});
    if (result.ok()) {
        auto& engine = result.value();
        engine->LoadModel(data, size); // 不应崩溃
    }
    return 0;
}
```

## 7. 代码规范
