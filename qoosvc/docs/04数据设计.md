### 2.2 表结构定义

#### map_meta — 地图元数据

```sql
CREATE TABLE map_meta (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    name            TEXT NOT NULL,              -- 地图名称
    version         TEXT NOT NULL DEFAULT '1.0',
    floor_count     INTEGER NOT NULL DEFAULT 1,
    resolution      REAL NOT NULL DEFAULT 0.05, -- 米/像素
    origin_x        REAL NOT NULL DEFAULT 0.0,
    origin_y        REAL NOT NULL DEFAULT 0.0,
    width_m         REAL NOT NULL,
    height_m        REAL NOT NULL,
    file_path       TEXT NOT NULL,              -- .qoomap 文件路径
    semantic_path   TEXT,                       -- 语义标注文件路径
    is_active       INTEGER NOT NULL DEFAULT 1,
    created_at      TEXT NOT NULL DEFAULT (datetime('now')),
    updated_at      TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_map_active ON map_meta(is_active);
```

#### face_library — 人脸库

```sql
CREATE TABLE face_library (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id         TEXT NOT NULL UNIQUE,       -- 关联 qooauth 用户 ID
    display_name    TEXT NOT NULL,
    face_embedding  BLOB NOT NULL,              -- 512 维浮点特征向量
    embedding_model TEXT NOT NULL DEFAULT 'insightface_v4',
    image_path      TEXT,                       -- 注册图像路径
    registered_at   TEXT NOT NULL DEFAULT (datetime('now')),
    last_seen_at    TEXT,
    last_location   TEXT,                       -- 最后出现位置 JSON
    is_active       INTEGER NOT NULL DEFAULT 1
);

CREATE INDEX idx_face_active ON face_library(is_active);
```

#### zone_config — 区域配置

```sql
CREATE TABLE zone_config (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    map_id          INTEGER NOT NULL REFERENCES map_meta(id),
    zone_type       TEXT NOT NULL,              -- restricted/speed/preferred
    name            TEXT NOT NULL,
    polygon_json    TEXT NOT NULL,              -- GeoJSON Polygon
    properties_json TEXT,                       -- {max_speed: 0.5, ...}
    is_active       INTEGER NOT NULL DEFAULT 1,
    created_at      TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_zone_map ON zone_config(map_id);
CREATE INDEX idx_zone_type ON zone_config(zone_type);
```

#### calibration_log — 校准记录

```sql
CREATE TABLE calibration_log (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    sensor_name     TEXT NOT NULL,              -- imu/camera_left/lidar/...
    calib_type      TEXT NOT NULL,              -- intrinsic/extrinsic/bias
    params_json     TEXT NOT NULL,              -- 校准参数
    quality_score   REAL,                       -- 校准质量评分 0-100
    method          TEXT,                       -- 标定方法
    operator        TEXT,                       -- 操作者（auto/manual）
    created_at      TEXT NOT NULL DEFAULT (datetime('now')),
    expires_at      TEXT                        -- 校准有效期
);

CREATE INDEX idx_calib_sensor ON calibration_log(sensor_name, created_at DESC);
```

#### diag_history — 诊断历史

```sql
CREATE TABLE diag_history (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    session_id      TEXT NOT NULL,              -- POST 会话 ID
    component       TEXT NOT NULL,              -- sensor/motor/battery/comm
    item_name       TEXT NOT NULL,              -- 诊断项名称
    status          TEXT NOT NULL,              -- OK/WARNING/ERROR/NOT_TESTED
    message         TEXT,
    value           REAL,                       -- 测量值
    unit            TEXT,                       -- 单位
    min_normal      REAL,                       -- 正常范围下限
    max_normal      REAL,                       -- 正常范围上限
    created_at      TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_diag_session ON diag_history(session_id);
CREATE INDEX idx_diag_component ON diag_history(component, created_at DESC);
```

#### speech_log — 语音交互日志

```sql
CREATE TABLE speech_log (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    query_text      TEXT NOT NULL,              -- 用户语音文本
    intent          TEXT,                       -- 识别的意图
    slots_json      TEXT,                       -- 槽位填充结果
    confidence      REAL,                       -- NLU 置信度
    speaker_id      TEXT,                       -- 说话人 ID
    language        TEXT DEFAULT 'zh-CN',
    response_text   TEXT,                       -- 机器人回复文本
    latency_ms      INTEGER,                    -- 端到端延迟
    created_at      TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_speech_created ON speech_log(created_at DESC);
```

#### nav_history — 导航历史

```sql
CREATE TABLE nav_history (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    goal_json       TEXT NOT NULL,              -- 导航目标
    path_json       TEXT,                       -- 规划路径
    result          TEXT NOT NULL,              -- SUCCESS/FAILED/CANCELED
    distance_m      REAL,                       -- 实际行驶距离
    duration_sec    REAL,                       -- 导航耗时
    error_message   TEXT,
    created_at      TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX idx_nav_created ON nav_history(created_at DESC);
```

## 3. 配置文件格式

### 3.1 语音服务配置 (voice_config.yaml)

```yaml
voice:
  wake_word:
    enabled: true
    words:
      - text: "Hey QooBot"
        language: "zh"
        threshold: 0.7
        sensitivity: medium
      - text: "你好机器人"
        language: "zh"
        threshold: 0.65
        sensitivity: medium
    false_trigger_suppression: true
    energy_threshold: 300  # 环境能量阈值

  asr:
    engine: whisper_cpp
    model: models/asr_zh_v3.onnx
    languages: ["zh", "en", "ja"]
    sample_rate: 16000
    streaming: true
    vad_sensitivity: 2  # 1-3, 越高越敏感

  nlu:
    engine: onnx_runtime
    model: models/nlu_v1.onnx
    intent_threshold: 0.6
    slot_threshold: 0.5
    context_window: 5  # 多轮对话上下文轮数

  tts:
    engine: piper
    default_voice: "warm_female"
    voices:
      warm_female:
        model: models/tts_female_v2.onnx
        sample_rate: 22050
      calm_male:
        model: models/tts_male_v1.onnx
        sample_rate: 22050
    speed: 1.0
    pitch: 1.0
```

## 5. 数据生命周期

### 5.1 数据保留策略

| 数据类型 | 存储位置 | 保留期限 | 清理策略 |
|---------|---------|---------|---------|
| 地图数据 | 本地文件 | 永久（用户删除前） | 手动管理 |
| 人脸特征 | SQLite | 永久（用户删除前） | 用户可删除 |
| 校准数据 | SQLite | 最新 10 条记录 | 滚动清理 |
| 诊断历史 | SQLite | 90 天 | 自动过期清理 |
| 语音日志 | SQLite | 30 天（本地） | 自动过期清理 |
| 导航历史 | SQLite | 30 天 | 自动过期清理 |
| 音频缓存 | 文件系统 | 10 分钟 | 环形缓冲覆盖 |
| 传感器缓存 | 共享内存 | 实时 | 新数据覆盖旧数据 |
| 配置文件 | 文件系统 | 永久 | 版本控制 |

### 5.2 隐私设计

- **本地优先**：语音数据、人脸特征、导航轨迹默认存储在本地
- **云端同步需授权**：上传诊断报告、地图备份需用户显式授权
- **数据脱敏**：云端上传前自动脱敏人脸图像、语音内容
- **可删除**：用户可随时删除本地所有个人数据
- **差分隐私**：匿名化的使用统计数据添加 Laplace 噪声 (ε=1.0)
