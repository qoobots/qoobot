## 定位

qoogear 是 **Made for QooBot (MFQ)** 认证体系，为第三方外设、末端执行器、传感器模组、可穿戴设备定义兼容性标准。它确保任何通过认证的配件都能即插即用、安全可靠地与 QooBot 机器人协同工作——正如 MFi 让第三方配件无缝融入 Apple 生态。

| 维度 | 说明 |
|------|------|
| **定位** | Made for QooBot 认证体系：第三方外设、末端执行器、可穿戴设备的兼容性标准 |
| **对标** | Apple MFi (Made for iPhone/iPad) 认证计划 |
| **依赖** | qoobody（接口标准）、qoochain（认证流程） |
| **被依赖** | 无 |

---

## 设计理念

1. **即插即用**：认证配件自动识别、自动配置，无需用户手动设置
2. **安全隔离**：第三方外设故障不影响机器人核心系统
3. **开放但可控**：接口标准开放，但认证确保质量和安全
4. **分级认证**：从基础兼容到深度集成，多级认证满足不同需求

## v1.0.0 初始化记录 (2026-06-29)

| 改进项 | 说明 |
|--------|------|
| qoogear-web 前端门户 | Vue3 + Element Plus + TypeScript，25个Vue页面(含路由/状态管理/API层)，4大入口(公开门户/开发者中心/管理后台/实验室门户) |
