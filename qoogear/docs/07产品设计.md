## 1. 产品定位

qoogear 是 QooBot 的 **Made for QooBot (MFQ)** 生态体系。如同 MFi 让第三方配件无缝融入 Apple 生态，MFQ 让任何通过认证的外设、传感器、可穿戴设备都能即插即用，安全可靠地与 QooBot 协同。

## 3. 即插即用体验

```
用户插入配件
    │
    ▼
机器人自动检测 (电气连接检测)
    │
    ▼
读取配件 EEPROM/固件 (VID/PID/Serial)
    │
    ▼
查询配件能力描述 (Capability Manifest)
    │
    ▼
自动加载驱动 (ROS2 Node / gRPC Service)
    │
    ▼
显示通知: "MFC认证配件: 某某夹爪 已连接"
    │
    ▼
配件立即可用 ✓
```

## 5. 认证流程

```
配件开发者
    │
    ▼
注册 MFQ 开发者账号
    │
    ▼
获取技术规范 + SDK + 参考设计
    │
    ▼
开发配件 + 自测
    │
    ▼
提交认证申请 + 自测报告
    │
    ▼
认证实验室测试 (或授权自测)
    │
    ▼
审核      ──不通过──→ 反馈修改
    │ 通过
    ▼
颁发 MFQ 证书 + 标志授权
    │
    ▼
配件上架 qoostore 配件专区 ✓
```
