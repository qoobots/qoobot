## 2. SDK 分发

### 2.1 Python SDK

```bash
pip install qoogear-sdk
```

### 2.2 C++ SDK

```cmake
# CMakeLists.txt
find_package(qoogear REQUIRED)
target_link_libraries(my_accessory qoogear::sdk)
```

### 2.3 版本管理

```
qoogear-sdk 版本: MAJOR.MINOR.PATCH
  · MAJOR: 接口不兼容变更
  · MINOR: 新增功能（向后兼容）
  · PATCH: 问题修复

MFQ 规范版本: 独立于 SDK，按年份+修订号
  · MFQ-Spec-2026-01: 2026 年第 1 版
```

## 4. 运维手册

### 4.1 证书管理

- MFQ 根证书: 10 年有效期，离线 HSM 存储
- MFQ Intermediate CA: 5 年有效期
- 配件证书: 3 年有效期，支持在线续签
- 证书吊销: CRL 每日更新 + OCSP 在线查询

### 4.2 规范更新

| 步骤 | 描述 |
|:-----|------|
| 1. 提案 | 技术委员会发起规范变更提案 |
| 2. 评审 | 社区 + 配件厂商评审期 (30天) |
| 3. 发布 | 新版本规范 + 过渡期 (90天) |
| 4. 废弃 | 旧版本规范 180 天后废弃 |

### 4.3 故障处理

| 故障 | 处理 |
|:-----|------|
| 配件无法认证 | 检查证书有效期、签名验证日志 |
| 配件通信故障 | 检查总线连接、终端电阻、波特率 |
| 驱动加载失败 | 检查驱动兼容性、依赖库、ROS2 环境 |
