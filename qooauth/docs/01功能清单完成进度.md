# qooauth — 账号与安全 功能清单与开发进度

> 最后更新：2026-06-26
>
> **定位**：统一身份基础设施。对标 Apple ID，为所有云端/设备端/用户端服务提供认证、授权、加密能力。
> **依赖**：无（独立基础服务）
> **被依赖**：qoocloud、qooeco
>
> **设计理念**：对标 Apple ID 账户体系，为每一台 QooBot 机器人及其用户提供统一、安全、无缝的身份体验。机器人不仅是设备，更是家庭成员或工作伙伴——账户体系需同时兼顾个人隐私、家庭共享与企业管控。作为独立基础服务，qooauth 是所有上层能力的"信任锚点"。
