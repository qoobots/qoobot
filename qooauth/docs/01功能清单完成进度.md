# qooauth — 账号与安全 功能清单与开发进度

> 最后更新：2026-06-28
>
> **定位**：统一身份基础设施。对标 Apple ID，为所有云端/设备端/用户端服务提供认证、授权、加密能力。
> **依赖**：无（独立基础服务）
> **被依赖**：qoocloud、qoostore
>
> **平台覆盖**：🖥️ onboard_core（C++ SDK）+ ☁️ cloud_service（OIDC/OAuth2 服务端）+ 🌐 web_portal（管理后台）+ 📱 mobile_app（生物特征认证）+ 📲 mini_program（扫码登录）+ 📱 mobile_h5（登录/授权页）
>
> **技术栈**：Java 17 + Spring Cloud + Spring Security + PostgreSQL + Redis + gRPC
>
> **设计文档**：✅ 02架构设计 | ✅ 03交互设计 | ✅ 04数据设计 | ✅ 05项目目录结构
>
> **基础设施**：✅ Maven 多模块构建系统 | ✅ qooauth-common 公共模块 | ✅ qooauth-auth 认证服务骨架 | ✅ qooauth-user 用户服务骨架 | ✅ Flyway 数据库迁移 | ✅ Docker Compose 部署
