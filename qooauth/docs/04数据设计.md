### 1.2 表结构定义

#### users — 用户主表

```sql
CREATE TABLE users (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email           VARCHAR(255) NOT NULL UNIQUE,
    email_verified  BOOLEAN NOT NULL DEFAULT FALSE,
    phone           VARCHAR(32),
    phone_verified  BOOLEAN NOT NULL DEFAULT FALSE,
    password_hash   VARCHAR(128),           -- argon2id hash, 可为 NULL (Passkey 用户)
    display_name    VARCHAR(100) NOT NULL,
    avatar_url      VARCHAR(512),
    locale          VARCHAR(10) DEFAULT 'en-US',
    timezone        VARCHAR(64) DEFAULT 'UTC',
    totp_enabled    BOOLEAN NOT NULL DEFAULT FALSE,
    totp_secret     VARCHAR(64),            -- AES-256-GCM 加密存储
    passkey_enabled BOOLEAN NOT NULL DEFAULT FALSE,
    account_status  VARCHAR(16) NOT NULL DEFAULT 'active',  -- active/locked/suspended/deleted
    locked_until    TIMESTAMPTZ,
    failed_attempts INTEGER NOT NULL DEFAULT 0,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    deleted_at      TIMESTAMPTZ              -- 软删除
);

CREATE INDEX idx_users_email ON users(email) WHERE deleted_at IS NULL;
CREATE INDEX idx_users_status ON users(account_status);
```

#### devices — 设备表

```sql
CREATE TABLE devices (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id         UUID NOT NULL REFERENCES users(id),
    device_name     VARCHAR(100) NOT NULL,
    device_type     VARCHAR(32) NOT NULL,     -- home_robot/factory_robot/accessory
    serial_number   VARCHAR(64) NOT NULL UNIQUE,
    certificate_fp  VARCHAR(128) NOT NULL,    -- X.509 SHA-256 指纹
    public_key_pem  TEXT NOT NULL,            -- 设备公钥 PEM
    firmware_version VARCHAR(32),
    hardware_model  VARCHAR(64),
    online_status   VARCHAR(16) NOT NULL DEFAULT 'offline',
    last_seen_at    TIMESTAMPTZ,
    last_location   JSONB,                    -- {lat, lng, address}
    activation_locked BOOLEAN NOT NULL DEFAULT FALSE,
    lock_reason     VARCHAR(64),              -- lost/stolen/maintenance
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_devices_user ON devices(user_id);
CREATE INDEX idx_devices_cert ON devices(certificate_fp);
CREATE UNIQUE INDEX idx_devices_serial ON devices(serial_number);
```

#### sessions — 会话表

```sql
CREATE TABLE sessions (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id         UUID NOT NULL REFERENCES users(id),
    token_hash      VARCHAR(128) NOT NULL,    -- SHA-512(JWT refresh_token)
    device_info     JSONB,                    -- {user_agent, ip, os, device_fingerprint}
    ip_address      INET NOT NULL,
    location        JSONB,                    -- {country, city, lat, lng}
    is_trusted      BOOLEAN NOT NULL DEFAULT FALSE,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    expires_at      TIMESTAMPTZ NOT NULL,
    revoked_at      TIMESTAMPTZ
);

CREATE INDEX idx_sessions_user ON sessions(user_id) WHERE revoked_at IS NULL;
CREATE INDEX idx_sessions_expires ON sessions(expires_at) WHERE revoked_at IS NULL;
```

#### api_keys — API 密钥表

```sql
CREATE TABLE api_keys (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id         UUID NOT NULL REFERENCES users(id),
    key_prefix      VARCHAR(16) NOT NULL,     -- "qoo_sk_" + 前 8 字符（用于 UI 显示）
    key_hash        VARCHAR(128) NOT NULL,    -- SHA-512(API Key)
    key_name        VARCHAR(100) NOT NULL,
    scopes          TEXT[] NOT NULL,          -- {device.read, skill.install, ...}
    last_used_at    TIMESTAMPTZ,
    expires_at      TIMESTAMPTZ,
    revoked_at      TIMESTAMPTZ,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_apikeys_user ON api_keys(user_id) WHERE revoked_at IS NULL;
```

#### audit_logs — 审计日志表

```sql
CREATE TABLE audit_logs (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id         UUID,                     -- 可为 NULL（系统操作）
    action          VARCHAR(64) NOT NULL,     -- user.login/user.logout/device.activate...
    resource_type   VARCHAR(32) NOT NULL,     -- user/device/session/api_key
    resource_id     VARCHAR(128),
    details         JSONB,                    -- 操作详情
    ip_address      INET NOT NULL,
    user_agent      VARCHAR(512),
    success         BOOLEAN NOT NULL,
    error_code      VARCHAR(64),
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

-- 分区表（按月分区）
CREATE INDEX idx_audit_user ON audit_logs(user_id, created_at DESC);
CREATE INDEX idx_audit_action ON audit_logs(action, created_at DESC);
CREATE INDEX idx_audit_created ON audit_logs(created_at DESC);
```

#### families — 家庭表

```sql
CREATE TABLE families (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name            VARCHAR(100) NOT NULL,
    owner_id        UUID NOT NULL REFERENCES users(id),
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE family_members (
    family_id       UUID NOT NULL REFERENCES families(id) ON DELETE CASCADE,
    user_id         UUID NOT NULL REFERENCES users(id),
    role            VARCHAR(16) NOT NULL DEFAULT 'member',  -- admin/member/child
    joined_at       TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    PRIMARY KEY (family_id, user_id)
);
```

## 2. Token 与密钥体系

### 2.1 JWT 结构

```json
// Header
{
  "alg": "EdDSA",
  "typ": "JWT",
  "kid": "key_2026_q1"
}

// Payload
{
  "iss": "https://auth.qoobot.com",
  "sub": "usr_abc123",
  "aud": ["qoocloud", "qooeco"],
  "exp": 1751108400,
  "iat": 1751104800,
  "jti": "jti_xyz789",
  "scopes": ["profile", "device.read"],
  "email": "user@example.com",
  "email_verified": true
}

// Signature: Ed25519(header.payload, private_key)
```

### 2.2 Token 生命周期

| Token 类型 | 有效期 | 刷新策略 | 存储位置 |
|-----------|--------|---------|---------|
| Access Token (JWT) | 1 小时 | 用 Refresh Token 刷新 | 客户端内存 |
| Refresh Token | 30 天 | 滚动刷新（每次使用重新签发） | HttpOnly Cookie / 安全存储 |
| ID Token (OIDC) | 10 分钟 | 不刷新（一次性使用） | 客户端 |
| API Key | 无限制（手动吊销） | 不适用 | 服务端加密存储 |
| 设备证书 | 2 年 | 到期前自动续签 | 设备安全存储 |

### 2.3 密钥派生层次

```
                    ┌──────────────────┐
                    │  HSM 根密钥 (Root)│
                    │  PKCS#11 / FIPS  │
                    └────────┬─────────┘
                             │ HKDF-Extract
                    ┌────────▼─────────┐
                    │  服务主密钥 (SMK) │
                    │  AES-256 (HSM)   │
                    └────────┬─────────┘
                             │ HKDF-Expand
            ┌────────────────┼────────────────┐
            │                │                │
   ┌────────▼──────┐ ┌──────▼──────┐ ┌───────▼──────┐
   │ JWT 签名密钥   │ │ 用户加密密钥 │ │ 设备认证密钥  │
   │ Ed25519 密钥对│ │ AES-256-GCM │ │ ECDSA 密钥对  │
   └───────────────┘ └─────────────┘ └──────────────┘
                             │
                             │ HKDF-Expand(user_salt)
                    ┌────────▼─────────┐
                    │  用户数据加密密钥  │
                    │  per-user unique  │
                    └──────────────────┘
```

### 2.4 密码存储

```python
# 密码哈希（argon2id）
password_hash = argon2id.hash(
    password,
    salt=os.urandom(32),
    time_cost=3,           # 迭代次数
    memory_cost=65536,     # 64 MB
    parallelism=4,         # 4 线程
    hash_len=32            # 256-bit 输出
)

# 存储格式
# $argon2id$v=19$m=65536,t=3,p=4$<salt_base64>$<hash_base64>
```

## 3. 数据安全

### 3.1 加密方案

| 数据类别 | 存储加密 | 传输加密 | 密钥管理 |
|---------|---------|---------|---------|
| 密码哈希 | argon2id | N/A（不可逆） | per-user salt |
| JWT 签名密钥 | HSM 内部 | N/A | HSM 管理 |
| 用户 PII（邮箱/手机） | AES-256-GCM | TLS 1.3 | 用户加密密钥 |
| 会话 Token | SHA-512 哈希 | TLS 1.3 | N/A（不可逆） |
| API Key | SHA-512 哈希 | TLS 1.3 | N/A（不可逆） |
| TOTP Secret | AES-256-GCM | TLS 1.3 | 服务主密钥 |
| 设备私钥 | 设备端安全存储 | mTLS 1.3 | 设备 HSM/TPM |
| 审计日志 | AES-256-GCM（可选） | TLS 1.3 | 服务主密钥 |

### 3.2 数据保留策略

| 数据类型 | 保留期限 | 删除策略 |
|---------|---------|---------|
| 用户账户数据 | 账户注销后 30 天 | 硬删除（GDPR 要求） |
| 会话记录 | 90 天 | 自动过期清理 |
| 审计日志 | 3 年 | 归档后删除 |
| API Key | 吊销后立即失效 | 30 天后清理 |
| 设备证书 | 吊销后保留 1 年 | 归档至冷存储 |
| 登录失败记录 | 24 小时 | 自动清理 |

### 3.3 隐私设计

- **数据最小化**：仅采集提供服务所必需的数据
- **目的限制**：数据仅用于声明的目的，不用于其他用途
- **本地优先**：敏感数据（如摄像头/麦克风数据）默认本地处理
- **用户可控**：隐私仪表盘可查看/导出/删除所有个人数据
- **差分隐私**：分析统计数据添加 Laplace 噪声（ε=1.0）
