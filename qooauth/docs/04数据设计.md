### 1.2 数据库分库策略

| 数据库 | 用途 | 服务 | 分片策略 |
|--------|------|------|----------|
| qooauth_users | 用户账户、资料、偏好 | User Service | 按 user_id 哈希 |
| qooauth_auth | 认证会话、MFA 配置 | Auth Service | 按 user_id 哈希 |
| qooauth_devices | 设备注册、证书、绑定 | Device Service | 按 device_id 哈希 |
| qooauth_oauth | OAuth 客户端、授权码 | Auth Service | 按 client_id 哈希 |
| qooauth_audit | 审计日志 (分区表) | Audit Service | 按日期分区 |
| qooauth_config | 系统配置、策略规则 | 共享 | 不分片 |

## 2. 核心数据模型

### 2.1 用户账户 (users)

```sql
CREATE TABLE users (
    -- 主键
    user_id         VARCHAR(32) PRIMARY KEY,       -- uid_ + 28位随机字符
    
    -- 认证凭据
    email           VARCHAR(255) NOT NULL UNIQUE,   -- 邮箱 (小写)
    phone           VARCHAR(20) UNIQUE,             -- 手机号 (E.164)
    password_hash   VARCHAR(255) NOT NULL,          -- argon2id 哈希
    password_salt   VARCHAR(64) NOT NULL,           -- 独立 salt
    
    -- 个人资料
    nickname        VARCHAR(64) NOT NULL,
    avatar_hash     VARCHAR(64),                    -- 头像文件 SHA256
    language        VARCHAR(10) DEFAULT 'zh-CN',    -- BCP 47
    timezone        VARCHAR(50) DEFAULT 'Asia/Shanghai',
    
    -- 状态
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVE',
    -- ACTIVE / EMAIL_VERIFICATION_PENDING / LOCKED / SUSPENDED / DELETED
    
    -- 安全
    email_verified  BOOLEAN DEFAULT FALSE,
    phone_verified  BOOLEAN DEFAULT FALSE,
    mfa_enabled     BOOLEAN DEFAULT FALSE,
    mfa_methods     JSONB,                          -- ["totp", "fido2"]
    
    -- 时间戳
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    last_login_at   TIMESTAMPTZ,
    deleted_at      TIMESTAMPTZ                     -- 软删除
);

CREATE INDEX idx_users_email ON users(email) WHERE state != 'DELETED';
CREATE INDEX idx_users_phone ON users(phone) WHERE state != 'DELETED';
CREATE INDEX idx_users_state ON users(state);
CREATE INDEX idx_users_created ON users(created_at);
```

### 2.2 MFA 配置 (mfa_configs)

```sql
CREATE TABLE mfa_configs (
    id              BIGSERIAL PRIMARY KEY,
    user_id         VARCHAR(32) NOT NULL REFERENCES users(user_id),
    
    -- TOTP
    totp_secret     VARCHAR(64),                    -- Base32 编码
    totp_enabled    BOOLEAN DEFAULT FALSE,
    
    -- FIDO2/WebAuthn
    webauthn_credentials JSONB,                     -- [{credential_id, public_key, ...}]
    webauthn_enabled BOOLEAN DEFAULT FALSE,
    
    -- 备用恢复码
    recovery_codes  JSONB,                          -- [bcrypt_hash1, bcrypt_hash2, ...]
    recovery_codes_used INTEGER DEFAULT 0,
    
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    
    UNIQUE(user_id)
);
```

### 2.3 设备注册 (devices)

```sql
CREATE TABLE devices (
    device_id       VARCHAR(32) PRIMARY KEY,        -- dev_ + 28位随机字符
    device_serial   VARCHAR(64) NOT NULL UNIQUE,    -- 出厂序列号
    hardware_model  VARCHAR(64) NOT NULL,           -- QB-Home-V1
    hardware_version VARCHAR(32),
    firmware_version VARCHAR(32),
    
    -- X.509 证书
    certificate_sn  VARCHAR(64) NOT NULL UNIQUE,    -- 证书序列号
    certificate_pem TEXT,                            -- 设备证书 PEM
    certificate_expires_at TIMESTAMPTZ NOT NULL,
    
    -- 绑定关系
    bound_user_id   VARCHAR(32) REFERENCES users(user_id),
    device_name     VARCHAR(64),                    -- 用户命名
    bound_at        TIMESTAMPTZ,
    
    -- 状态
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVATED',
    -- ACTIVATED / BOUND / LOCKED / LOST / WIPED
    
    -- 设备指纹
    cpu_id          VARCHAR(128),
    tpm_ek_hash     VARCHAR(64),
    mac_address     VARCHAR(17),
    
    -- 位置
    last_ip         INET,
    last_location   JSONB,                          -- {lat, lng, accuracy}
    last_seen_at    TIMESTAMPTZ,
    
    -- 时间戳
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_devices_bound_user ON devices(bound_user_id);
CREATE INDEX idx_devices_state ON devices(state);
CREATE INDEX idx_devices_serial ON devices(device_serial);
```

### 2.4 OAuth 客户端 (oauth_clients)

```sql
CREATE TABLE oauth_clients (
    client_id       VARCHAR(64) PRIMARY KEY,
    client_secret_hash VARCHAR(255) NOT NULL,       -- bcrypt
    
    client_name     VARCHAR(128) NOT NULL,
    client_type     VARCHAR(32) NOT NULL,           -- CONFIDENTIAL / PUBLIC
    owner_user_id   VARCHAR(32) REFERENCES users(user_id),
    
    -- 重定向 URI
    redirect_uris   JSONB NOT NULL,                 -- ["https://app.example.com/callback"]
    
    -- 授权配置
    grant_types     JSONB NOT NULL,                 -- ["authorization_code", "refresh_token"]
    scopes          JSONB NOT NULL,                 -- ["openid", "profile", "email"]
    
    -- Token 配置
    access_token_ttl_seconds INTEGER DEFAULT 3600,
    refresh_token_ttl_seconds INTEGER DEFAULT 2592000,
    
    -- 状态
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVE',
    -- ACTIVE / SUSPENDED / REVOKED
    
    -- 审核
    verified        BOOLEAN DEFAULT FALSE,
    verified_at     TIMESTAMPTZ,
    
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

### 2.5 OAuth 授权码 (oauth_authorization_codes)

```sql
CREATE TABLE oauth_authorization_codes (
    code            VARCHAR(128) PRIMARY KEY,
    client_id       VARCHAR(64) NOT NULL,
    user_id         VARCHAR(32) NOT NULL,
    
    scopes          JSONB NOT NULL,
    redirect_uri    VARCHAR(512) NOT NULL,
    code_challenge  VARCHAR(128),                   -- PKCE (SHA256)
    code_challenge_method VARCHAR(16),              -- S256 / plain
    
    state           VARCHAR(128),
    nonce           VARCHAR(128),                   -- OIDC nonce
    
    expires_at      TIMESTAMPTZ NOT NULL,
    used            BOOLEAN DEFAULT FALSE,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_oauth_codes_expires ON oauth_authorization_codes(expires_at)
    WHERE used = FALSE;
```

### 2.6 API Key (api_keys)

```sql
CREATE TABLE api_keys (
    key_id          VARCHAR(32) PRIMARY KEY,        -- ak_ + 28位随机字符
    user_id         VARCHAR(32) NOT NULL REFERENCES users(user_id),
    
    key_prefix      VARCHAR(12) NOT NULL,           -- 前 12 位用于展示
    key_hash        VARCHAR(255) NOT NULL,          -- SHA256 哈希
    
    name            VARCHAR(128) NOT NULL,
    permissions     JSONB NOT NULL,                 -- ["device:read", "device:control"]
    
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVE',
    -- ACTIVE / REVOKED
    
    expires_at      TIMESTAMPTZ,
    last_used_at    TIMESTAMPTZ,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    revoked_at      TIMESTAMPTZ
);

CREATE INDEX idx_api_keys_user ON api_keys(user_id);
CREATE INDEX idx_api_keys_prefix ON api_keys(key_prefix);
```

### 2.7 审计日志 (audit_logs)

```sql
CREATE TABLE audit_logs (
    id              BIGSERIAL,
    event_id        UUID NOT NULL,
    event_time      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    
    -- 操作主体
    actor_type      VARCHAR(32) NOT NULL,           -- USER / DEVICE / SERVICE / ADMIN
    actor_id        VARCHAR(32) NOT NULL,
    
    -- 操作
    action          VARCHAR(64) NOT NULL,           -- LOGIN / REGISTER / TOKEN_ISSUE / ...
    resource_type   VARCHAR(32),                    -- USER / DEVICE / OAUTH_CLIENT / ...
    resource_id     VARCHAR(32),
    
    -- 结果
    result          VARCHAR(16) NOT NULL,           -- SUCCESS / FAILURE / DENIED
    error_code      VARCHAR(32),
    
    -- 上下文
    client_ip       INET,
    user_agent      VARCHAR(512),
    geo_country     VARCHAR(2),                     -- ISO 3166-1
    geo_city        VARCHAR(128),
    
    -- 详情
    details         JSONB,                          -- 操作相关附加信息
    
    -- 追踪
    trace_id        VARCHAR(64),
    span_id         VARCHAR(32),
    
    PRIMARY KEY (event_time, id)
) PARTITION BY RANGE (event_time);

-- 按月分区
CREATE TABLE audit_logs_2026_06 PARTITION OF audit_logs
    FOR VALUES FROM ('2026-06-01') TO ('2026-07-01');
CREATE TABLE audit_logs_2026_07 PARTITION OF audit_logs
    FOR VALUES FROM ('2026-07-01') TO ('2026-08-01');

CREATE INDEX idx_audit_actor ON audit_logs(actor_type, actor_id, event_time DESC);
CREATE INDEX idx_audit_action ON audit_logs(action, event_time DESC);
CREATE INDEX idx_audit_trace ON audit_logs(trace_id);
```

### 2.8 会话管理 (sessions) - Redis

```
Redis Key 设计:

# 用户会话 (Hash)
qooauth:session:{session_id}
  - user_id: "uid_xxx"
  - device_id: "dev_xxx"
  - client_id: "qooweb_portal"
  - created_at: "1719563400"
  - expires_at: "1719567000"
  - last_activity: "1719564000"
  - ip: "192.168.1.1"
  - user_agent: "QooBot-Web/1.0"
  TTL: 3600s

# 用户活跃会话索引 (Set)
qooauth:user_sessions:{user_id}
  - members: [session_id_1, session_id_2, ...]

# Token 黑名单 (String)
qooauth:token_blacklist:{jti}
  - value: "revoked"
  - TTL: token剩余有效期

# 登录失败计数 (String)
qooauth:login_fail:{email}
  - value: "5"
  - TTL: 900s (15分钟窗口)

# 速率限制 (String)
qooauth:rate_limit:{api}:{user_id_or_ip}:{window}
  - value: "45"
  - TTL: 窗口时长
```

### 2.9 设备证书 (certificates)

```sql
CREATE TABLE certificates (
    id              BIGSERIAL PRIMARY KEY,
    serial_number   VARCHAR(64) NOT NULL UNIQUE,
    
    -- 证书类型
    cert_type       VARCHAR(32) NOT NULL,           -- DEVICE / CA / SERVICE
    
    -- 主体
    subject_cn      VARCHAR(256) NOT NULL,
    subject_org     VARCHAR(128),
    
    -- 关联
    device_id       VARCHAR(32) REFERENCES devices(device_id),
    
    -- 有效期
    not_before      TIMESTAMPTZ NOT NULL,
    not_after       TIMESTAMPTZ NOT NULL,
    
    -- 状态
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVE',
    -- ACTIVE / REVOKED / EXPIRED
    revoked_at      TIMESTAMPTZ,
    revocation_reason VARCHAR(256),
    
    -- 指纹
    sha256_fingerprint VARCHAR(64) NOT NULL UNIQUE,
    
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_certs_device ON certificates(device_id);
CREATE INDEX idx_certs_state ON certificates(state);
```

### 2.10 家庭/团队 (family_groups)

```sql
CREATE TABLE family_groups (
    group_id        VARCHAR(32) PRIMARY KEY,
    name            VARCHAR(128) NOT NULL,
    owner_user_id   VARCHAR(32) NOT NULL REFERENCES users(user_id),
    
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVE',
    max_members     INTEGER DEFAULT 6,
    member_count    INTEGER DEFAULT 0,
    
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE family_members (
    id              BIGSERIAL PRIMARY KEY,
    group_id        VARCHAR(32) NOT NULL REFERENCES family_groups(group_id),
    user_id         VARCHAR(32) NOT NULL REFERENCES users(user_id),
    
    role            VARCHAR(32) NOT NULL DEFAULT 'MEMBER',
    -- OWNER / ADMIN / MEMBER / CHILD
    
    joined_at       TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    
    UNIQUE(group_id, user_id)
);
```

### 2.11 开发者证书 (developer_certs)

```sql
CREATE TABLE developer_certificates (
    cert_id         VARCHAR(32) PRIMARY KEY,
    user_id         VARCHAR(32) NOT NULL REFERENCES users(user_id),
    
    cert_type       VARCHAR(32) NOT NULL,           -- DEVELOPMENT / DISTRIBUTION
    team_id         VARCHAR(32),                    -- 关联开发团队
    
    serial_number   VARCHAR(64) NOT NULL UNIQUE,
    sha256_fingerprint VARCHAR(64) NOT NULL UNIQUE,
    
    expires_at      TIMESTAMPTZ NOT NULL,
    state           VARCHAR(32) NOT NULL DEFAULT 'ACTIVE',
    
    -- 关联的能力/权限
    capabilities    JSONB,                          -- ["skill_sign", "cloud_build"]
    
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    revoked_at      TIMESTAMPTZ
);

CREATE INDEX idx_dev_certs_user ON developer_certificates(user_id);
```

## 3. Redis 缓存策略

| 缓存项 | 数据结构 | TTL | 失效策略 |
|--------|----------|-----|----------|
| 用户资料 | Hash | 300s | 写时失效 |
| 用户权限 | Set | 600s | 权限变更时失效 |
| Session | Hash | 3600s | 按需刷新 TTL |
| Token 黑名单 | String | Token 剩余有效期 | 到期自动删除 |
| 速率限制计数 | String | 窗口时长 | 到期自动删除 |
| 临时验证码 | String | 300s | 使用后删除 |
| 设备在线状态 | String | 60s | 心跳刷新 |
| OAuth 授权码 | Hash | 600s | 使用后删除 |

## 4. 数据安全

### 4.1 敏感字段加密

| 字段 | 加密方式 | 密钥管理 |
|------|----------|----------|
| password_hash | argon2id (不可逆) | N/A |
| totp_secret | AES-256-GCM (可逆) | HSM 派生密钥 |
| api_key_hash | SHA256 (不可逆) | N/A |
| recovery_codes | bcrypt (不可逆) | N/A |
| 用户 PII (phone) | AES-256-GCM | HSM 派生密钥 |

### 4.2 数据保留策略

| 数据类型 | 保留期 | 过期处理 |
|----------|--------|----------|
| 活跃用户数据 | 永久 | N/A |
| 已删除用户数据 | 30天 | 永久擦除 |
| 审计日志 | 3年 | 归档→删除 |
| 认证会话 | 1小时 | 自动过期 |
| Token 黑名单 | Token 有效期 | 自动过期 |
| 登录失败计数 | 15分钟 | 自动过期 |
| 临时验证码 | 5分钟 | 自动过期 |
