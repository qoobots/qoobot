## 1. 安全控制器硬件方案

### 1.1 安全 MCU 选型

| MCU | 架构 | 安全等级 | 特点 |
|-----|------|---------|------|
| TI TMS570LC4357 | ARM Cortex-R5F 双核锁步 | SIL 3 | ECC 内存, 硬件 BIST |
| NXP S32K344 | ARM Cortex-M7 锁步 | ASIL D | 硬件安全引擎, Cortex-M4 协处理 |
| ST SPC58NN84 | PowerPC e200z4 双核 | ASIL D | 广泛用于汽车安全 |

### 1.3 安全功能列表

| 安全功能 | SIL 等级 | 响应时间 | 说明 |
|---------|---------|---------|------|
| 急停 | SIL 3 | < 20ms | 双通道冗余, Cat. 3 |
| 力矩限制 | SIL 2 | < 10ms | 每关节独立监控 |
| 速度限制 | SIL 2 | < 20ms | 编码器速度监控 |
| 碰撞检测 | SIL 2 | < 10ms | 力矩/电流/接近多传感器融合 |
| 看门狗 | SIL 2 | < 50ms | 窗口看门狗, 独立时钟 |
| 安全停止 (STO) | SIL 3 | < 5ms | 硬件切断电机驱动使能 |
| 制动控制 | SIL 2 | < 50ms | 电磁抱闸, 失电制动 |

## 2. 急停回路设计

### 2.1 双通道冗余架构

```
急停回路 (Cat. 3, PL d):

    24V ──┬── 急停按钮 CH1 (NC) ──┬── 安全继电器 K1 线圈 ── GND
          │                       │
          ├── 急停按钮 CH2 (NC) ──┼── 安全继电器 K2 线圈 ── GND
          │                       │
          └── 安全 MCU 监控 ──────┤ (读取触点状态)
                                  │
    安全继电器输出:
    K1 (常开) ──┬── 48V 电机驱动使能
    K2 (常开) ──┘  (双继电器串联, 任一断开即切断)

    安全 MCU 读取:
    CH1 状态 ── GPIO ── 安全 MCU
    CH2 状态 ── GPIO ── 安全 MCU
    → 双通道不一致 → 故障 → 进入安全状态
```

### 2.2 制动器接口

```
制动器控制 (每个关节):

    24V ── 安全继电器输出 ──┬── 制动器 1 线圈 ── MOSFET (安全 MCU) ── GND
                            ├── 制动器 2 线圈 ── MOSFET ── GND
                            └── ...

    工作原理:
    - 通电释放 (fail-safe): 断电自动抱闸
    - 抱闸力矩: ≥ 1.5× 关节最大持续力矩
    - 响应时间: < 50ms (断电到抱闸)
    - 磨损监控: 制动器动作次数计数 + 磨损指示
```

## 3. 状态指示硬件

### 3.1 LED 灯环

```
LED 灯环 (360° 可见):
┌──────────────────────────────────┐
│   24× RGB LED (WS2812B 或类似)   │
│   环形排列, 间距 15°             │
│                                  │
│   模式:                           │
│   ├── 启动中: 蓝色旋转            │
│   ├── 就绪: 绿色常亮              │
│   ├── 运行中: 青色呼吸            │
│   ├── 思考中: 紫色旋转            │
│   ├── 警告: 黄色闪烁 (1Hz)        │
│   ├── 错误: 红色闪烁 (4Hz)        │
│   ├── 急停: 红色常亮 + 蜂鸣       │
│   ├── 充电中: 绿色流水            │
│   └── 低电量: 红色慢闪 (0.5Hz)    │
└──────────────────────────────────┘
```

### 3.2 显示屏 (可选)

| 参数 | 规格 |
|------|------|
| 尺寸 | 2.4" ~ 4.0" |
| 分辨率 | 240×320 ~ 480×800 |
| 类型 | TFT-LCD / OLED |
| 接口 | SPI / MIPI DSI |
| 触摸 | 电容式 (可选) |

显示内容: 机器人状态、电池电量、Wi-Fi 信号、当前技能、表情动画

### 3.3 蜂鸣器

| 参数 | 规格 |
|------|------|
| 类型 | 电磁式 |
| 频率范围 | 2kHz ~ 4kHz |
| 声压级 | ≥ 85dB @ 10cm |
| 驱动 | PWM (安全 MCU) |

告警模式:
- 启动完成: 短促 "嘀" (200ms)
- 碰撞/故障: 连续 "嘀-嘀-嘀" (500ms 间隔)
- 急停: 持续蜂鸣
- 低电量: 间隔长 "嘀---嘀---" (2s 间隔)
