## 1. 电池组规格

### 1.1 电池选型

| 参数 | 规格 | 说明 |
|------|------|------|
| 电芯类型 | 21700 锂离子 (NCA/NCM) | 高能量密度 |
| 串并联 | 13S4P | 48V 标称, 52 颗电芯 |
| 标称电压 | 48.1V (3.7V×13) | |
| 电压范围 | 36.4V ~ 54.6V | 2.8V ~ 4.2V/节 |
| 额定容量 | 20Ah (960Wh) | |
| 持续放电 | 40A (约 1900W) | 2C 放电 |
| 峰值放电 | 80A (约 3800W, 10s) | 4C 脉冲 |
| 充电电流 | 10A (0.5C) 标准 / 20A (1C) 快充 | |
| 重量 | 约 5.5 kg | |
| 循环寿命 | ≥ 500 次 (80% 容量) | |

## 2. 电源树设计

### 2.1 电压域分配

```
电池组 48V (36.4~54.6V)
│
├── 电机驱动母线 ───────────────── 48V (直通)
│   └── 6× 关节电机 (CAN FD 控制)
│
├── DC/DC 降压 ──→ 12V @ 10A (传感器/通信)
│   ├── RGB-D 相机 (5V via LDO)
│   ├── LiDAR (12V)
│   ├── Wi-Fi/BLE 模组 (3.3V via LDO)
│   └── 风扇 (12V PWM)
│
├── DC/DC 降压 ──→ 5V @ 5A (逻辑电路)
│   ├── USB-C 调试口
│   ├── LED 灯环
│   └── 蜂鸣器
│
├── DC/DC 降压 ──→ 3.3V @ 3A (MCU/编码器)
│   ├── 安全 MCU (常供电)
│   ├── EtherCAT 从站控制器
│   └── 编码器供电
│
└── PMIC ──→ SoC 核心供电
    ├── VDD_CPU (0.8V, 动态调压)
    ├── VDD_GPU (0.9V)
    ├── VDD_NPU (0.85V)
    ├── DDR (1.1V)
    └── IO (1.8V/3.3V)
```

### 2.2 上电时序

```
上电时序:
t=0ms    安全 MCU 域 (3.3V) 先上电
t=10ms   BMS 自检通过, 接通主回路
t=20ms   SoC PMIC 上电序列开始
t=100ms  12V 传感器域上电
t=200ms  5V 外设域上电
t=500ms  SoC 启动完成
t=1s     电机驱动使能 (安全 MCU 确认)

下电时序 (反向):
急停/故障 → 电机驱动立即断开 (t<1ms)
         → SoC 安全关机 (t<5s)
         → 安全 MCU 最后断电
```

### 2.3 电源保护

| 保护类型 | 实现方式 | 阈值 | 响应时间 |
|---------|---------|------|---------|
| 输入过压 | TVS + 保险丝 | > 60V | < 1μs |
| 输入反接 | 理想二极管控制器 | — | < 100μs |
| 输出过流 | eFuse (电子保险丝) | 按域设定 | < 10μs |
| 输出短路 | eFuse + 电流检测 | > 额定×2 | < 1μs |
| 热保护 | NTC + PMIC 关断 | > 125°C | < 100ms |

## 3. 充电系统

### 3.1 充电底座方案

```
充电底座:
┌─────────────────────────────┐
│   充电触点 (弹簧针 ×4)       │
│   VCC / GND / CAN / 导引    │
├─────────────────────────────┤
│   充电控制板                 │
│   ├── 48V/10A 充电器         │
│   ├── CC/CV 充电曲线         │
│   ├── 红外对接引导           │
│   └── CAN 通信 (与机器人)    │
└─────────────────────────────┘

对接流程:
1. 机器人导航到充电区域
2. 红外信标引导粗对准 (< 10cm)
3. 机械导向销引导精对准 (< 1mm)
4. 导引引脚接触 → 确认身份 (CAN)
5. 主电源接通 → 开始充电
6. 充电完成 → 断开主电源 → 机器人驶离
```

### 3.2 充电策略

| 阶段 | 电流 | 截止条件 |
|------|------|---------|
| 预充 (深度放电) | 2A (0.1C) | 电压 > 3.0V/节 |
| 恒流 (CC) | 10A (0.5C) | 电压 > 4.1V/节 |
| 恒压 (CV) | 递减 | 电流 < 1A (0.05C) |
| 涓流 | 1A | SOC 100% |
| 完成 | 0A | 电池已满 |
