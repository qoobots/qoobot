# qoobody 文件结构

> 最后更新：2026-06-29 | 版本：v0.6

```
qoobody/
├── README.md                                   # 项目说明
├── CMakeLists.txt                              # 根构建系统（多平台编译：x86_64/aarch64/armv7-r）
├── Dockerfile                                  # Docker 多阶段构建（dev → build → test → deploy）
├── .gitignore                                  # Git 忽略规则
├── cmake/
│   ├── qoobody_platform.cmake                  # 平台检测与交叉编译支持
│   └── toolchains/
│       ├── aarch64-linux-gnu.cmake             # Jetson Orin / RK3588 交叉编译工具链
│       └── arm-none-eabi.cmake                 # Safety MCU bare-metal 工具链
├── docs/                                       # 设计文档 (12 份)
│   ├── 00系统架构总览.md                        # 硬件架构、模块划分、接口矩阵
│   ├── 00文件结构.md                            # 本文件 — 完整文件结构描述
│   ├── 01功能清单完成进度.md                     # 功能清单与完成进度追踪 (68/68 = 100%)
│   ├── 01计算平台设计.md                        # SoC 选型、异构拓扑、安全模块
│   ├── 02感知系统接口规范.md                    # 视觉/LiDAR/IMU/时间同步
│   ├── 03执行器接口规范.md                      # CANopen/EtherCAT、急停回路
│   ├── 04能源系统设计.md                        # BMS、电源树、充电、低功耗
│   ├── 05通信总线规范.md                        # CAN FD/EtherCAT/gPTP/Wi-Fi
│   ├── 06结构与散热设计.md                      # 形态因子、关节模组、DFM
│   ├── 07安全硬件规范.md                        # 安全 MCU、碰撞检测、制动器
│   ├── 08测试与验证规范.md                      # HIL、可靠性、基准、互操作
│   └── 09操作系统配置.md                        # Linux 内核裁剪、PREEMPT_RT、设备树
├── .github/
│   └── workflows/
│       └── ci.yml                              # GitHub Actions CI/CD 流水线
├── hal/                                        # 硬件抽象层 (10 个 C 头文件)
│   ├── qoo_hal.h                                # 主入口 — 初始化/反初始化/版本
│   ├── qoo_hal_types.h                          # 通用类型、数学库、配置结构体
│   ├── qoo_hal_sensor.h                         # 相机/LiDAR/IMU/麦克风/触觉接口
│   ├── qoo_hal_actuator.h                       # 电机/末端执行器/全身控制接口
│   ├── qoo_hal_power.h                          # BMS/电源树/充电/低功耗接口
│   ├── qoo_hal_safety.h                         # 安全控制器/碰撞检测/急停/指示
│   ├── qoo_hal_comm.h                           # CAN FD/EtherCAT/Wi-Fi/BLE/UWB
│   ├── qoo_hal_time.h                           # gPTP 时间同步/硬件触发
│   ├── qoo_hal_remote.h                         # 远程遥控接口扩展
│   └── qoo_hal_error.h                          # 错误处理与诊断
├── drivers/                                     # 驱动参考实现 (17 个 C/H 文件)
│   ├── qoo_drv_sensor_base.h                    # 传感器驱动基类 (VTable 模式)
│   ├── qoo_drv_imu_bmi270.c                     # Bosch BMI270 IMU 完整驱动
│   ├── qoo_drv_imu.c                            # 通用 IMU 驱动（BMI270/ICM-42688/TDK）
│   ├── qoo_drv_motor_canopen.c                  # CANopen CiA 402 电机完整驱动
│   ├── qoo_drv_motor_ethercat.c                 # EtherCAT CoE 电机驱动
│   ├── qoo_drv_camera.c                         # MIPI CSI-2 相机驱动
│   ├── qoo_drv_lidar.c                          # LiDAR UDP/SPI 驱动
│   ├── qoo_drv_canfd.c                          # CAN FD 控制器驱动
│   ├── qoo_drv_mic_array.c                      # I2S/TDM 麦克风阵列驱动
│   ├── qoo_drv_time_sync.c                      # gPTP 硬件时间同步驱动
│   ├── qoo_drv_tactile.c                        # 触觉传感器 SPI 驱动
│   ├── qoo_drv_wireless.c                       # Wi-Fi/BLE/UWB 无线驱动适配
│   ├── qoo_drv_gptp_stack.c                     # gPTP 协议栈适配
│   ├── qoo_drv_force_ctrl.c                     # 力控伺服驱动
│   ├── qoo_drv_linear_actuator.c                # 线性执行器驱动
│   ├── qoo_drv_end_effector.c                   # 末端执行器驱动
│   └── qoo_drv_env_sensor.c                     # 环境传感器驱动
├── firmware/                                    # 嵌入式固件 (2 个 C 文件)
│   ├── safety_mcu/
│   │   └── qoo_safety_fw.c                      # SIL2 安全 MCU 固件 (双通道急停/碰撞检测)
│   └── motor_ctrl/
│       └── qoo_motor_ctrl.c                     # 电机 FOC 三环 PID 控制固件
├── compute/                                     # 计算平台参考实现 (6 个文件)
│   ├── qoo_compute_guide.md                     # 计算平台方案指南 (SoC 选型/NPU 推理/实时配置)
│   ├── qoo_memory_storage.c                     # LPDDR5/eMMC/NVMe 内存存储管理
│   ├── qoo_realtime_mcu.c                       # Cortex-R52/R82 安全 MCU 实时计算参考
│   ├── qoo_thermal_mgmt.c                       # 散热管理 (被动/主动/液冷三级策略)
│   ├── qoo_ai_accelerator.c                     # NPU 驱动适配、推理运行时集成
│   └── qoo_tpm_security.c                       # TPM/安全芯片 信任根 + 密钥存储
├── power/                                       # 能源系统参考实现 (5 个文件)
│   ├── qoo_power_guide.md                       # 能源系统参考实现指南
│   ├── qoo_battery_validation.c                 # 电池组实物验证 (组装/充放电循环)
│   ├── qoo_low_power.c                          # 低功耗模式管理 (待机/休眠/深度休眠)
│   ├── qoo_power_budget.c                       # 功耗预算模型 (各子系统功率基线)
│   └── qoo_power_charging.c                     # 充电管理 (有线快充/无线/底座触点)
├── mechanical/                                  # 机械结构参考设计
│   ├── qoo_mechanical_guide.md                  # 机械结构参考设计指南 (关节模组/材料/散热)
│   ├── mujoco/                                  # MuJoCo 物理仿真模型 (qoobot_float.xml + 控制源码)
│   │   ├── eigen-3.4.0/                         # Eigen 线性代数库
│   │   ├── glad/                                # GLAD OpenGL 加载器
│   │   ├── glfw-3.3.8/                          # GLFW 窗口库
│   │   ├── mujoco-3.3.0/                        # MuJoCo 物理引擎源码
│   │   └── qpOASES/                             # QP 求解器 (MPC/WBC 依赖)
│   ├── openloong-dyn-control/                   # 运动控制仿真 (开源仿生人控制)
│   └── step/                                    # STEP 格式 CAD 模型
├── safety_hw/                                   # 硬件安全参考实现 (1 个文件)
│   └── qoo_safety_hw_guide.md                   # 硬件安全参考实现指南
├── test/                                        # 测试与验证 (6 个文件)
│   ├── qoo_test_guide.md                        # 测试参考指南
│   ├── qoo_hil_platform.c                       # HIL 仿真平台 (信号注入/故障注入)
│   ├── qoo_hil_test.c                           # HIL 测试用例 (传感器/运动/安全/电源)
│   ├── qoo_perf_benchmark.c                     # 性能基准套件 (延迟/带宽/吞吐量)
│   ├── qoo_reliability_test.c                   # 可靠性测试 (环境/机械/老化/EMC)
│   └── qoo_interop_test.c                       # 互操作性测试 (即插即用/协议一致性)
├── ros2_pkgs/                                   # ROS 2 生态集成 (26+ 文件)
│   ├── README.md                                # ROS 2 包说明
│   ├── qoobot_description/                      # URDF/Xacro 机器人模型描述
│   │   ├── CMakeLists.txt + package.xml
│   │   ├── urdf/qoobot.urdf.xacro
│   │   ├── meshes/ (STL 视觉网格)
│   │   ├── launch/ (display.launch.py, gazebo.launch.py)
│   │   ├── rviz/qoobot.rviz
│   │   └── worlds/ (earth.world, stairs.world)
│   └── qoobody_ros_bridge/                      # ROS 2 桥接节点
│       ├── CMakeLists.txt + package.xml
│       ├── qoobody_ros_bridge/bridge_node.py
│       ├── scripts/ (keyboard_teleop.py, force_injector.py)
│       └── launch/ (bridge.launch.py)
└── os/                                          # 操作系统配置 (9 个文件)
    ├── qoo_kernel_defconfig                     # Linux 内核配置 (PREEMPT_RT + 机器人外设)
    ├── qoo_device_tree.dts                      # 设备树源码 (QooBot v1)
    ├── qoo_patch_rt.md                          # PREEMPT_RT 补丁详细步骤
    ├── sample_rt_thread.c                       # 实时线程示例代码
    ├── qoobody-image.bb                         # Yocto 镜像配方
    ├── qoo-kernel-rt.bbappend                   # Yocto 实时内核配方追加
    ├── qoo-drivers.bbappend                     # Yocto 驱动配方追加
    ├── qoo-systemd-conf.bbappend                # Yocto systemd 配置追加
    └── qoo-security.bbappend                    # Yocto 安全基线配方
```

## 统计 (v0.6)

| 类别 | 文件数 | 说明 |
|------|--------|------|
| 设计文档 (docs/) | 12 | 系统架构 + 9 个子系统规范 + 功能进度 + 文件结构 |
| 硬件抽象层 (hal/) | 10 | 标准 C 头文件，定义 HAL 全部接口 |
| 驱动参考 (drivers/) | 17 | 传感器/电机/CAN/LiDAR/相机等完整驱动参考实现 |
| 嵌入式固件 (firmware/) | 2 | 安全 MCU + 电机控制固件 |
| 计算平台 (compute/) | 6 | 内存/实时/散热/AI加速/TPM 安全 + 方案指南 |
| 能源系统 (power/) | 5 | 电池验证/低功耗/预算/充电 + 系统指南 |
| 硬件安全 (safety_hw/) | 1 | 硬件安全参考实现指南 |
| 测试与验证 (test/) | 6 | HIL/可靠/性能/互操作 + 测试指南 |
| ROS 2 生态 (ros2_pkgs/) | 26 | qoobot_description + qoobody_ros_bridge + 脚本 |
| OS 配置 (os/) | 9 | 内核配置/设备树/RT补丁/示例 + 5×Yocto 配方 |
| 构建系统 (cmake/) | 3 | 根 CMakeLists.txt + 平台检测 + 2×工具链 |
| CI/CD (.github/) | 1 | GitHub Actions 流水线 (5 并行任务) |
| 机械结构 (mechanical/) | 大量 | MuJoCo/Eigen3/qpOASES/GLFW/STEP models |
| 工程配置 | 4 | README.md + .gitignore + Dockerfile + 本文件 |
| **合计** | **101+** | (不含 mechanical/ 第三方库) |
