### 2.2 单元测试示例

```python
# test_asr.py
class TestASREngine:
    def test_recognize_clear_speech(self):
        """清晰语音应准确识别"""
        engine = ASREngine(model="sherpa-zh-small")
        audio = load_test_audio("clear_hello.wav")
        result = engine.recognize(audio)
        assert result.text == "你好"
        assert result.confidence > 0.9

    def test_recognize_noisy_speech(self):
        """噪声环境下降噪后应可识别"""
        engine = ASREngine(model="sherpa-zh-small")
        audio = load_test_audio("noisy_hello.wav", snr=-5)
        result = engine.recognize(audio)
        assert result.confidence > 0.7

    def test_unsupported_language(self):
        """不支持的语言应抛出异常"""
        engine = ASREngine(model="sherpa-zh-small")
        with pytest.raises(UnsupportedLanguageError):
            engine.recognize(audio, language="fr-FR")
```

## 4. 质量门禁

### 4.1 PR 门禁检查

| 检查项 | 工具 | 阻断条件 |
|:-------|------|---------|
| Python 代码风格 | flake8 + black | 任何错误 |
| C++ 代码风格 | clang-format | 格式不匹配 |
| 类型检查 | mypy (Python) | 新引入的类型错误 |
| 单元测试 | pytest / GTest | 失败或覆盖率下降 |
| 集成测试 | ROS2 launch test | 任何测试失败 |
| 依赖审计 | pip-audit / cargo-deny | 已知漏洞 |
| 文档更新 | 人工检查 | API 变更无文档 |

### 4.2 发布门禁

| 检查项 | 标准 |
|:-------|------|
| 单元测试覆盖率 | >80%（总体）/ >90%（关键模块） |
| 集成测试通过率 | 100% |
| E2E 测试通过率 | 100% |
| 性能基准未劣化 | 所有指标在阈值内 |
| 安全扫描 | 无 Critical/High 漏洞 |
| 文档完整 | 所有公开 API 有文档 |

## 7. 代码规范

### 7.1 Python 规范

- PEP 8 + 120 字符行长
- 类型注解 (Type Hints) 强制要求
- docstring 遵循 Google Style
- 所有公共 API 需有使用示例

### 7.2 C++ 规范

- Google C++ Style Guide
- C++17 特性 (no C++20 yet)
- RAII + 智能指针优先
- 裸指针需注释说明所有权
