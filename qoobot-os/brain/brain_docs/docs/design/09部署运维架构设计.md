## 启动流程

```bash
# 1. ROS2 启动
ros2 launch qoobrain brain.launch.py

# 启动流程:
#   → 加载配置文件
#   → 初始化硬件驱动
#   → 启动 Perception Node
#   → 启动 Cognition Node (加载LLM)
#   → 启动 Decision Node
#   → 启动 Control Node
#   → 加载已安装技能
#   → 启动 HITL WebSocket Server
#   → 执行开机自检(POST)
#   → 进入运行循环
```

### launch.py 示例

```python
from launch import LaunchDescription
from launch_ros.actions import Node

def generate_launch_description():
    return LaunchDescription([
        Node(package='qoobrain', executable='perception_node'),
        Node(package='qoobrain', executable='cognition_node'),
        Node(package='qoobrain', executable='decision_node'),
        Node(package='qoobrain', executable='control_node'),
        Node(package='qoobrain', executable='skill_runtime'),
        Node(package='qoobrain', executable='hitl_server'),
        Node(package='qoobrain', executable='dashboard'),
    ])
```

## OTA 升级策略

```
云端 qoocloud                        机器人 qoobrain
     │                                     │
     │  1. 版本检查                         │
     │ <────────────────────────────────────│
     │                                     │
     │  2. 可用更新列表                     │
     │ ─────────────────────────────────────>│
     │                                     │
     │  3. 下载增量包 (A/B分区)            │
     │ ─────────────────────────────────────>│
     │                                     │
     │                              4. 写入B分区
     │                              5. 校验SHA256+Ed25519
     │                              6. 原子切换 A↔B
     │                              7. 重启验证
     │                                     │
     │  8. 更新状态上报                     │
     │ <────────────────────────────────────│
```

### A/B 分区

| 分区 | 当前 | 更新时 |
|:-----|:-----|:-------|
| A | 运行中(只读) | 新版本写入目标 |
| B | 空闲 | 新版本写入目标 |
| 切换 | — | `ln -sf B active` → 重启 |

### 回滚条件

- 启动失败 (3次)
- 健康检查失败
- 关键传感器异常
- 手动触发

## 监控

### ROS2 Diagnostics

```yaml
# 诊断项
analyzers:
  perception:
    type: diagnostic_aggregator/AnalyzerGroup
    path: Perception
    analyzers:
      camera:
        type: diagnostic_aggregator/GenericAnalyzer
        path: Camera
        expected: ['Camera Front: OK', 'Camera Depth: OK']
      detection:
        type: diagnostic_aggregator/GenericAnalyzer
        path: Detection
        timeout: 1.0
        
  control:
    type: diagnostic_aggregator/AnalyzerGroup
    path: Control
    analyzers:
      joints:
        type: diagnostic_aggregator/GenericAnalyzer
        path: Joints
        expected: ['']
      safety:
        type: diagnostic_aggregator/GenericAnalyzer
        path: Safety
        expected: ['Safety: OK']
```

### Prometheus 指标

| 指标 | 类型 | 描述 |
|:-----|:-----|:-----|
| `qoobrain_perception_fps` | Gauge | 感知帧率 |
| `qoobrain_decision_tick_rate` | Gauge | 决策频率 |
| `qoobrain_control_loop_ms` | Histogram | 控制回路延迟 |
| `qoobrain_model_confidence` | Gauge | 模型置信度 |
| `qoobrain_skills_running` | Gauge | 运行中技能数 |
| `qoobrain_memory_used_bytes` | Gauge | 内存使用 |
| `qoobrain_temperature_celsius` | Gauge | 芯片温度 |
| `qoobrain_battery_percent` | Gauge | 电池电量 |

## 故障恢复

| 故障 | 检测方式 | 恢复策略 |
|:-----|:---------|:---------|
| 节点崩溃 | ROS2 lifecycle | 自动重启 |
| 传感器超时 | 心跳超时 | 降级模式+告警 |
| LLM推理超时 | 超时检测 | 云端Fallback |
| OOM | cgroup通知 | 卸载非关键技能 |
| CPU过热 | 温度传感器 | 降频+告警 |
| 通信中断 | 连接超时 | 本地自治+重连 |
| OTA失败 | 启动检查 | 回滚到A分区 |
