## 质量门禁

### PR检查

```
□ pytest 全通过 (≥85% 覆盖率)
□ Google Test 全通过 (≥85% 覆盖率)
□ pylint 评分 ≥ 9/10
□ clang-format + clang-tidy 通过
□ ROS2 colcon build 成功
□ 仿真回归测试通过
□ 无新增 ASAN/UBSAN 告警
```

### 发布门禁

| 门禁 | 标准 |
|:-----|:-----|
| 单元测试 | 100%通过 |
| 集成测试 | 100%通过 |
| 仿真测试 | 关键场景100% |
| 性能基准 | 延迟 < 基线×1.05 |
| 安全扫描 | 零Critical漏洞 |
| HIL测试 | 核心场景通过 |
