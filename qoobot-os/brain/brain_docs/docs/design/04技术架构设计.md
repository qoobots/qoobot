## 关键技术决策

### Python + C++ 混合架构

```
           ┌─────────────────┐
           │  Python 3.11    │  ← 高层逻辑
           │  感知/认知/NLU  │     快速迭代
           │  技能运行时     │     AI/ML生态
           └────────┬────────┘
                    │ pybind11
           ┌────────┴────────┐
           │    C++ 17       │  ← 性能关键路径
           │   决策/控制     │     实时性保证
           │   行为树引擎    │     内存控制
           └─────────────────┘
```

### ROS2 通信框架

选用 ROS2 Humble 而非自研IPC：成熟的发布-订阅机制、丰富的工具生态(tf2/rviz2/rosbag2)、行业标准接口。

### 行为树 + RL 混合决策

| 场景 | 决策方式 | 理由 |
|:-----|:---------|:-----|
| 结构化任务(如家务) | 行为树 | 可解释、可调试 |
| 非结构化(如避障) | 强化学习 | 无需规则编码 |
| 长尾场景 | HITL | 人工兜底 |

### 端云协同推理

```python
class HybridInferencePipeline:
    async def infer(self, task: Task) -> Result:
        # 1. 端侧快速推理
        local_result = await self.onboard_inference(task)
        
        # 2. 置信度评估
        if local_result.confidence < self.threshold:
            # 3. 云端Fallback
            cloud_result = await self.cloud_inference(task)
            return self.fuse(local_result, cloud_result)
        
        return local_result
```

## 性能设计

| 优化 | 策略 | 预期收益 |
|:-----|:-----|:---------|
| 模型量化 | INT8/FP16 via qoocore | -50%推理延迟 |
| 流水线并行 | 感知→认知→规划Overlap | +50%吞吐 |
| 内存池 | Arena分配+权重共享 | -30%内存 |
| SIMD加速 | Neon/AVX via qoocore | 2-4x算子加速 |
| ROS2零拷贝 | loaned messages | 减少大帧拷贝 |
