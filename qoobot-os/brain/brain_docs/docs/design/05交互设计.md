## 语音交互流程

```
用户: "嘿 QooBot，帮我拿一下桌上的水杯"
         │
         ▼
    唤醒词检测 (Porcupine / 自定义)
         │
         ▼
    ASR: Whisper Stream → 文本
         │
         ▼
    NLU: Qwen2.5-7B → 意图+槽位
    {
      "intent": "fetch_object",
      "slots": {
        "object": "water_cup",
        "location": "on_table"
      }
    }
         │
         ▼
    任务分解 → 行为树生成 → 执行
         │
         ▼
    TTS: "好的，我来拿桌上的水杯"
```

## HITL 人工干预

### 触发条件

| 条件 | 阈值 | 响应 |
|:-----|:-----|:-----|
| 决策置信度低 | < 0.7 | 推送轨迹选项 |
| 碰撞风险 | < 0.3m | 自动暂停+通知 |
| 未知物体 | 检测置信度<0.5 | 请求标记 |
| 任务超时 | > 预期×2 | 人工接管选项 |
| 用户主动 | 语音/按钮 | 立即切换 |

### 干预界面

```
┌────────────────────────────────────────────┐
│  ⚠️ 不确定如何处理，请选择：               │
│                                            │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐   │
│  │ 轨迹 A   │ │ 轨迹 B   │ │ 轨迹 C   │   │
│  │ 安全:95% │ │ 安全:88% │ │ 安全:72% │   │
│  │ 时间:12s │ │ 时间:8s  │ │ 时间:6s  │   │
│  └──────────┘ └──────────┘ └──────────┘   │
│                                            │
│  [3秒后自动选择轨迹A]                       │
│  [手动选择] [取消任务] [接管控制]           │

```

## 移动端控制
