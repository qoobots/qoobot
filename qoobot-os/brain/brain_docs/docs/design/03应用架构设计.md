## 模块职责

| 模块 | 技术 | 职责 |
|:-----|:-----|:-----|
| **Perception Node** | Python + ONNX + qoocore | 目标检测/分割/跟踪/SLAM/深度估计 |
| **Cognition Node** | Python + LLM | NLU/任务分解/场景理解/知识推理 |
| **Decision Node** | C++ + BehaviorTree.CPP | 行为树编排/RL策略/轨迹生成/评估 |
| **Control Node** | C++ | 全身运动控制/灵巧操作/力控/避障 |
| **Skill Runtime** | Python | 技能注册/沙箱隔离/生命周期管理 |
| **HITL Manager** | WebSocket + gRPC | 人工干预接口/状态同步 |
| **Scene Graph** | C++ (in-memory) | 场景实体/关系/空间拓扑维护 |

## 云端Fallback机制

```
端侧置信度 < 阈值？
    │
    ├── YES → gRPC调用 qoocloud → 云端大模型推理 → 结果回传
    │                                          │
    │                                    端侧融合+执行
    │
    └── NO  → 端侧直接执行
```
