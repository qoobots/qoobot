## 核心数据流

```
传感器输入 → 感知 → 认知 → 决策 → 控制 → 执行器输出
    │          │       │       │       │
    │     ┌────┴───────┴───────┴───┐   │
    │     │       qoocore          │   │
    │     │     (推理加速)         │   │
    │     └────────────────────────┘   │
    │                                  │
    └──── 端侧主推理 ──────────────────┘
                  │ (置信度<阈值)
                  ▼
          云端大模型Fallback
          (qoocloud DeepSeek-V3)
```

| 阶段 | 时间 | 里程碑 |

| **Alpha** | 2026 Q1-Q2 | 感知+决策+控制基础闭环 |

| **v1.0** | 2027 Q1 | 完整技能市场集成、语音交互、自主导航 |

---

## 设计原则

1. **实时优先**: 控制回路<1ms，感知<30ms，一切为实时性让路
2. **端云协同**: 端侧主推理(低延迟)+云端大模型(高能力)
3. **安全内置**: Safety不是功能层而是基础层
4. **人机协作**: HITL在长尾场景中兜底
5. **技能可扩展**: 第三方开发者可通过Skill SDK扩展能力
