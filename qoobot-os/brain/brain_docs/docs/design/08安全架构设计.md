# 08 安全架构设计

QooBot Brain 安全架构——分层安全策略，确保机器人安全可靠运行。

---

## 安全分层

```
┌─────────────────────────────────────────────────────────────────┐
│                    qoobrain 安全分层                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  L3 — 操作安全 (Operational Safety)                            │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ HITL兜底 │ 地理围栏 │ 速度限制 │ 任务审批               │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                 │
│  L2 — 软件安全 (Software Safety)                                │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 行为树验证 │ 技能沙箱 │ 动作边界检查 │ 冲突检测        │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                 │
│  L1 — 硬件安全 (Hardware Safety)                                │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 急停按钮 │ 力/力矩限制 │ 碰撞检测 │ 安全继电器          │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 硬件安全 (L1)

| 机制 | 实现 | 响应时间 |
|:-----|:-----|:---------|
| **急停按钮** | 物理按钮 + 安全继电器 | < 50ms |
| **力/力矩限制** | 关节力矩传感器 + 软限位 | < 1ms |
| **碰撞检测** | FCL实时碰撞 + 电流异常 | < 5ms |
| **冗余制动** | 双回路刹车系统 | < 100ms |
| **看门狗** | 硬件看门狗定时器 | 1s超时 |
| **电源保护** | 过流/过放/过热保护 | 实时 |

## 软件安全 (L2)

### 行为树安全验证

```
行为树 XML
    │
    ▼

          ▼
┌─────────────────────┐
│ 动态边界检查         │
│ · 关节位置限位      │

│ · 工作空间检查      │
└─────────────────────┘
```

### 技能沙箱

| 限制 | 值 |
|:-----|:-----|
| CPU | max 2 cores |
| 内存 | max 512MB |
| 执行时间 | max 30s/tick |
| 文件系统 | read-only /tmp |
| 网络 | localhost only |
| 可用API | 白名单模式 |

## 操作安全 (L3)

### HITL 干预矩阵

| 场景 | 自动响应 | HITL介入 | 紧急处理 |
|:-----|:---------|:---------|:---------|
| 低置信度决策 | 暂停+提示 | 轨迹选择 | 自动安全轨迹 |
| 碰撞预警 | 减速+避让 | 方向确认 | 急停 |
| 传感器失效 | 降级模式 | 远程接管 | 安全停止 |
| 电池低电 | 返回充电站 | 确认位置 | 原地停靠 |
| 通信中断 | 本地自治 | — | 安全位置待机 |

```
│    │   (全速)         │    │
│    │                  │    │
│    └────────┬─────────┘    │
│             │              │

```

## 威胁模型

| 威胁 | 缓解 | 严重度 |
|:-----|:-----|:------:|
| **传感器欺骗** | 多模态交叉验证 | 高 |
| **指令注入** | 输入校验+白名单 | 高 |
| **权限提升** | 技能沙箱+最小权限 | 高 |
| **模型投毒** | 签名验证+沙箱测试 | 中 |
| **通信拦截** | TLS 1.3 + mTLS | 中 |
