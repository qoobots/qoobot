## 3. Brain SDK (Python API)

### 3.1 认知 API (brain_ai)

```python
from brain_ai import BrainAI

# 初始化
brain = BrainAI(config_path="config/brain.yaml")

# 感知
objects = await brain.perception.detect(image)        # 目标检测
segments = await brain.perception.segment(image)       # 语义分割
pose = await brain.perception.estimate_pose(obs)       # 位姿估计

# 理解
scene = await brain.cognition.understand(context)      # 场景理解
intent = await brain.cognition.recognize_intent(text)   # 意图识别

# 规划
plan = await brain.planning.create_task_plan(goal)     # 任务规划
traj = await brain.planning.plan_motion(start, end)    # 运动规划

# 语音
await brain.speech.speak("你好")                        # TTS
text = await brain.speech.listen(duration=5.0)         # ASR

# LLM
response = await brain.llm.chat("帮我拿一下桌上的水杯")
```

### 3.2 SDK API (brain_sdk)

```python
from qoobot_brain import QooBrainClient

client = QooBrainClient("localhost:50051")

# 场景 API
async with client.scenario("pick_and_place") as scene:
    await scene.set_target("水杯")
    await scene.execute()

# 回调管理
@client.on_event("collision_detected")
def handle_collision(event):
    print(f"碰撞检测: 位置={event.position}, 力度={event.force}N")

# 流式处理
async for chunk in client.stream_inference(camera_stream):
    result = process(chunk)
```

## 4. gRPC 服务交互

### 4.1 服务定义

qoobot-os 内部使用 gRPC 进行进程间服务调用。关键服务：

| 服务 | Proto 定义 | 调用方 | 说明 |
|------|-----------|--------|------|
| `ControlService` | `control.proto` | brain_core | 运动控制指令下发 |
| `PerceptionService` | `perception.proto` | brain_ai | 感知结果查询 |
| `SafetyService` | `safety.proto` | brain_core | 安全状态订阅/紧急制动 |
| `DeviceService` | `device.proto` | QooSvc | 设备注册/发现/状态 |
| `VoiceService` | `voice.proto` | QooSvc | 语音交互 (ASR/TTS/NLU) |
| `NavigationService` | `navigation.proto` | QooSvc | 导航指令/路径查询 |
| `DiagnosticService` | `diagnostic.proto` | QooSvc | 诊断/健康报告 |

### 4.2 通信模式

| 模式 | 使用场景 | 实现 |
|------|----------|------|
| Unary RPC | 配置查询、状态获取 | `request → response` |
| Server Streaming | 传感器数据流、日志推送 | `request → stream<response>` |
| Client Streaming | 批量数据上传、模型同步 | `stream<request> → response` |
| Bidirectional Streaming | 实时控制指令、安全心跳 | `stream<req> ↔ stream<resp>` |

### 5.1 Topic 话题

| Topic | 消息类型 | 频率 | 发布者 | 订阅者 |
|-------|----------|:----:|--------|--------|
| `/joint_states` | `sensor_msgs/JointState` | 1kHz | HAL | brain_core, RViz |
| `/imu/data` | `sensor_msgs/Imu` | 200Hz | HAL (sensor) | brain_ai, QooSvc |
| `/camera/rgb` | `sensor_msgs/Image` | 30Hz | HAL (sensor) | brain_ai |
| `/camera/depth` | `sensor_msgs/Image` | 30Hz | HAL (sensor) | brain_ai |
| `/lidar/points` | `sensor_msgs/PointCloud2` | 10Hz | HAL (sensor) | brain_ai |
| `/odom` | `nav_msgs/Odometry` | 100Hz | brain_core | QooSvc (nav) |
| `/cmd_vel` | `geometry_msgs/Twist` | 100Hz | brain_core | HAL (actuator) |
| `/diagnostics` | `diagnostic_msgs/DiagnosticArray` | 1Hz | QooSvc (diag) | external tools |

### 5.2 Service 服务

| Service | 提供者 | 说明 |
|---------|--------|------|

| `/emergency_stop` | HAL (safety) | 紧急制动 |
| `/get_map` | QooSvc (nav) | 获取当前地图 |
| `/start_exploration` | QooSvc (nav) | 开始自主探索 |

### 5.3 Action 动作

| Action | 提供者 | 说明 |

## 6. 直接 C/C++ API

### 6.1 QooCore C API

```c
// 引擎生命周期
qoocore_engine_t* engine = qoocore_engine_create(config);
qoocore_engine_load_model(engine, "detection.qoomodel");
qoocore_tensor_t* output = qoocore_engine_infer(engine, input_tensor);
qoocore_engine_destroy(engine);

// 编译器
qoocore_compiler_t* compiler = qoocore_compiler_create();
qoocore_compiler_compile(compiler, "model.onnx", "model.qoomodel", options);

// 内存
qoocore_ion_handle_t ion_fd = qoocore_ion_alloc(size, flags);
qoocore_tensor_t* tensor = qoocore_tensor_from_ion_fd(ion_fd, shape, dtype);
```

## 7. 交互协议矩阵

| 交互入口 | 目标用户 | 协议 | 带宽要求 | 延迟要求 |
|----------|----------|------|:--------:|:--------:|
| qoocli CLI | 模型开发者 | CLI stdio | — | — |
| brain CLI | 技能开发者 | CLI stdio | — | — |
| Brain Python SDK | Python 开发者 | gRPC (Python) | > 10 Mbps | < 100ms |
| C/C++ Direct API | C++ 开发者 | 头文件+共享库 | — | < 1ms |
| gRPC Services | 进程间通信 | gRPC (C++) | > 100 Mbps | < 5ms |
| ROS 2 Topics | ROS 生态集成 | DDS | > 1 Gbps | < 1ms |
| ROS 2 Services | ROS 工具链 | DDS (RPC) | > 100 Mbps | < 10ms |
| ROS 2 Actions | 长时间任务 | DDS (Action) | > 100 Mbps | — |
| VS Code Plugin | IDE 开发者 | LSP / WebSocket | > 1 Mbps | < 200ms |
| WebSocket API | Web 前端 | WebSocket (JSON) | > 1 Mbps | < 100ms |
| Safety Loop | 硬件安全 | CAN FD | > 1 Mbps | < 100μs |
