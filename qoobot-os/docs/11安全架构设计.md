# 11 — 安全架构设计

> 版本：v0.1 | 最后更新：2026-06-30 | 状态：Draft
> 子项目：qoobot-os（仿生人操作系统）

---

## 1. 安全设计原则

| 原则 | 说明 | 实现 |
|------|------|------|
| 纵深防御 | 多层安全机制，不依赖单一防护 | 硬件 + 固件 + OS + 应用 |
| 最小权限 | 每个模块只拥有完成功能所需的最小权限 | seccomp + cgroup + capability |
| 安全默认 | 出厂默认启用所有安全功能 | 首次启动引导 + 配置校验 |
| 故障安全 | 任何故障导致安全状态（制动/断电） | 安全 MCU 独立监控 |
| 隐私优先 | 数据本地处理优先，云端传输需授权 | TEE + 加密 + 用户授权 |
| 可审计 | 所有安全事件有完整日志 | 安全日志独立分区 |

## 3. 安全启动链

```
ROM Bootloader (不可变)
    │ 验证签名
    ▼
U-Boot / EDK II (签名)
    │ 验证签名
    ▼
Linux Kernel (签名)
    │ 验证签名
    ▼
qoobot-os 核心模块 (签名)
    │ 验证签名
    ▼
用户技能 (签名)
    │
    └── 任何环节验证失败 → 启动失败 / 降级到安全模式
```

### 3.1 信任链密钥

| 密钥 | 存储位置 | 用途 |
|------|----------|------|
| Root of Trust Key | eFuse / OTP | 验证 Bootloader 签名 |
| OS Signing Key | TPM | 验证 Kernel/OS 签名 |
| App Signing Key | 安全存储 | 验证技能签名 |
| Device Key | eFuse (每设备唯一) | 设备身份认证 |

## 4. 运行时安全

### 4.1 进程隔离

| 隔离机制 | 实现 | 适用范围 |
|----------|------|----------|
| 命名空间 | PID/mount/network namespace | 技能进程 |
| 资源限制 | cgroup v2 (memory/cpu/io) | 所有非核心进程 |
| 系统调用过滤 | seccomp-bpf | 技能进程 |
| capabilities | 移除 CAP_SYS_ADMIN 等 | 所有模块 |
| 文件系统 | read-only rootfs + tmpfs | 系统模块 |
| SELinux/AppArmor | MAC 强制访问控制 | 全局 |

### 4.2 安全状态机

```
                    ┌──────────┐
                    │  NORMAL  │ ← 正常运行
                    └────┬─────┘
                         │
          ┌──────────────┼──────────────┐
          │ 碰撞检测      │ 温度超标      │ 心跳丢失
          ▼              ▼              ▼
   ┌────────────┐ ┌────────────┐ ┌────────────┐
   │ DEGRADED   │ │ THERMAL    │ │ COMM_LOSS  │
   │ 降级运行    │ │ 热保护      │ │ 通信丢失    │
   │ 减速/限矩   │ │ 限速/降频   │ │ 保持最后指令│
   └─────┬──────┘ └─────┬──────┘ └─────┬──────┘
         │              │              │
         └──────────────┼──────────────┘
                        │ 条件恶化/超时
                        ▼
                 ┌────────────┐
                 │ SAFE_STOP  │ ← 安全停止
                 │ 零力矩/锁定 │
                 └─────┬──────┘
                       │ 急停触发
                       ▼
                 ┌────────────┐
                 │EMERGENCY   │ ← 紧急制动
                 │ 动力切断   │
                 └────────────┘
```

### 4.3 安全监控指标

| 监控项 | 阈值 | 动作 | 实现 |
|--------|------|------|------|
| 关节力矩 | > 额定 120% | DEGRADED | 电流传感器 |
| 碰撞力 | > 10N | SAFE_STOP | 关节力矩突变 |
| SoC 温度 | > 85°C | THERMAL | 温度传感器 |
| 电机温度 | > 120°C | DEGRADED + 限流 | 电机热敏电阻 |
| 电池电压 | < 3.2V/cell | SAFE_STOP | BMS |
| 心跳超时 | > 100ms | COMM_LOSS | Watchdog |
| 内存 OOM | > 95% | 终止低优先级技能 | cgroup |

## 5. 数据安全

### 5.1 数据分类

| 分类 | 数据 | 保护级别 |
|------|------|:--------:|
| 公开 | 日志摘要、性能指标 | 无 |
| 内部 | 技能名称、配置参数 | 访问控制 |
| 敏感 | 地图数据、物体位置、行为日志 | 加密存储 |
| 机密 | 人脸特征、声纹、健康数据 | TEE + 加密 |

### 5.2 数据加密

| 场景 | 算法 | 密钥管理 |
|------|------|----------|
| 存储加密 (at rest) | AES-256-GCM | TPM 派生密钥 |
| 传输加密 (in transit) | TLS 1.3 (gRPC) | 设备证书 + PKI |
| 模型加密 | AES-256-CBC + Ed25519 签名 | 端侧密钥 |
| 日志加密 | AES-256-GCM | 独立日志密钥 |
| TEE 计算 | TrustZone / SGX | 硬件隔离 |

### 5.3 隐私保护

| 措施 | 说明 |
|------|------|
| 本地优先 | 语音/视觉数据默认本地处理 |
| 用户授权 | 云端传输需显式弹窗授权 |
| 数据最小化 | 仅上传必要的匿名化数据 |
| 数据保留 | 本地数据默认 30 天自动清理 |
| 隐私模式 | 物理开关禁用所有网络通信 |

## 6. 通信安全

### 6.1 内部通信

| 链路 | 安全措施 |
|------|----------|
| gRPC (进程间) | TLS 1.3 + mTLS (双向证书认证) |
| ROS 2 DDS | DDS-Security (认证/加密/访问控制) |
| CAN FD 总线 | 消息认证码 (MAC) + 序列号防重放 |
| DMA-BUF | 仅限同进程组访问 (Unix domain socket fd 传递) |

### 6.2 外部通信

| 链路 | 安全措施 |
|------|----------|
| 云端 gRPC | TLS 1.3 + 设备证书 + JWT token |
| WebSocket (Web) | WSS + Bearer token + CORS |
| OTA 更新 | Ed25519 签名 + SHA-256 校验 + 原子切换 |
| 5G/Wi-Fi | WPA3 / 5G-AKA + VPN (可选) |

## 7. 安全审计

### 7.1 审计日志

```json
{
  "event": "safety_mode_change",
  "timestamp": "2026-06-30T09:00:00.000Z",
  "severity": "WARNING",
  "details": {
    "from": "NORMAL",
    "to": "DEGRADED",
    "reason": "joint_torque_exceeded",
    "joint": "right_elbow",
    "measured_nm": 45.2,
    "limit_nm": 40.0
  }
}
```

### 7.2 审计事件

| 事件 | 记录内容 |
|------|----------|
| 安全模式切换 | 原因、触发条件、转换时间 |
| 技能操作 | 安装/卸载/启动/停止 + 操作者 |
| 配置变更 | 变更项、旧值、新值、操作者 |
| 登录/认证 | 来源、方式、成功/失败 |
| 数据访问 | 访问什么数据、哪个模块 |
| OTA 更新 | 版本、签名、校验结果 |
