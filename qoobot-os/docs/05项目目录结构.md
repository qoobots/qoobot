## 2. 构建系统设计

### 2.1 顶层 CMake Superbuild

```cmake
# qoobot-os/CMakeLists.txt
cmake_minimum_required(VERSION 3.25)
project(qoobot-os VERSION 0.1.0 LANGUAGES CXX C)

# 子项目开关
option(QOOBOT_BUILD_ALL     "Build all subprojects"           OFF)
option(QOOBOT_BUILD_CORE    "Build ai-engine/"                OFF)
option(QOOBOT_BUILD_HAL     "Build hal/"                      OFF)
option(QOOBOT_BUILD_SERVICES "Build services/"                OFF)
option(QOOBOT_BUILD_STORE   "Build store/"                    OFF)

# 按需 add_subdirectory
if(QOOBOT_BUILD_CORE)
    add_subdirectory(ai-engine)
endif()
# ... 其他子项目
```

### 2.2 各子项目独立构建

```bash
# ai-engine/
cd ai-engine && mkdir build && cd build
cmake .. -DQOOCORE_ENABLE_NPU_QNN=ON && cmake --build .

# brain/ (C++ brain_core) — 使用 colcon
cd brain
colcon build --symlink-install --packages-select brain_core

# brain/ (Python)
cd brain && pip install -e ".[ai,dev]"

# hal/
cd hal/mechanical && mkdir build && cd build
cmake .. && cmake --build .

# services/
cd services && pip install -e .
```

### 2.3 一键构建

```bash
cd qoobot-os && mkdir build && cd build
cmake .. -DQOOBOT_BUILD_ALL=ON
cmake --build . --parallel $(nproc)

# QooBrain 需要单独 colcon build
cd ../brain && colcon build --symlink-install
```

## 3. 代码规范

### 3.1 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| C++ 命名空间 | `snake_case` | `qoocore::memory`, `brain_core::hal_interface` |
| C++ 类/结构体 | `PascalCase` | `InferenceEngine`, `ConfigManager` |
| C++ 函数/方法 | `snake_case` | `load_model()`, `get_joint_state()` |
| C++ 成员变量 | `snake_case_` | `model_path_`, `is_loaded_` |
| C++ 常量 | `UPPER_CASE` | `MAX_BATCH_SIZE`, `DEFAULT_TIMEOUT_MS` |
| C++ 头文件 | `snake_case.h` | `inference_engine.h`, `tensor.h` |
| C++ 源文件 | `snake_case.cpp` | `inference_engine.cpp` |
| Python 模块 | `snake_case` | `brain_ai`, `motion_planner` |
| Python 类 | `PascalCase` | `BrainAI`, `ScenarioClient` |
| Python 函数 | `snake_case` | `estimate_pose()`, `create_task_plan()` |
| ROS 2 package | `snake_case` | `brain_core`, `qoobot_description` |
| ROS 2 topic | `/snake_case` | `/joint_states`, `/cmd_vel` |

### 3.2 代码风格工具

| 语言 | 工具 | 配置文件 |
|------|------|----------|
| C++ | clang-format (Google style, 100 columns) | `.clang-format` |
| C++ | clang-tidy | `.clang-tidy` |
| Python | ruff + mypy | `pyproject.toml [tool.ruff]` |
| TypeScript | ESLint + Prettier | `.eslintrc`, `.prettierrc` |

### 3.3 目录组织原则

1. **头文件统一在 `include/<project>/`** — 公共 API 头文件使用 `include/qoocore/`, `include/brain_core/` 结构
2. **实现与头文件对应** — `src/` 下的目录结构与 `include/` 一一对应
3. **每个子目录独立 CMakeLists.txt** — 不写巨型 CMake，子目录自包含
4. **proto/schema 独立目录** — FlatBuffers/Protobuf 定义不混入源码
5. **config/ 与代码分离** — YAML/JSON/URDF 配置文件不在源码目录内
6. **测试与实现共存** — `tests/` 在模块根目录，单元测试镜像源码结构

## 5. 初始化骨架脚本

```bash
#!/bin/bash
# scripts/init_qoobot_os.sh — 一键初始化 qoobot-os 开发环境

set -e

echo "=== QooBot-OS 环境初始化 ==="

# 1. 安装系统依赖
sudo apt-get update
sudo apt-get install -y \
    cmake build-essential clang-format clang-tidy \
    python3.11 python3.11-dev python3.11-venv \
    libprotobuf-dev protobuf-compiler \
    libgrpc++-dev libgrpc-dev \
    libeigen3-dev libopencv-dev

# 2. Python 虚拟环境
python3.11 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip setuptools wheel

# 3. 安装 Python 依赖
cd brain && pip install -e ".[ai,dev]" && cd ..
cd services && pip install -e . && cd ..
cd hal/mechanical && pip install -e . && cd ../..

# 4. CMake 构建 C++ 模块
mkdir -p build && cd build
cmake .. -DQOOBOT_BUILD_ALL=ON -DCMAKE_BUILD_TYPE=Release
cmake --build . --parallel $(nproc)
cd ..

# 5. ROS 2 (可选)
# source /opt/ros/humble/setup.bash
# cd brain && colcon build --symlink-install && cd ..

echo "=== 初始化完成 ==="
```
