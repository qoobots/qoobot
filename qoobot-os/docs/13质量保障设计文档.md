### 4.2 性能回归检测

```cpp
// benchmarks/inference_bench.cpp
#include <benchmark/benchmark.h>

static void BM_YOLOv8_Inference(benchmark::State& state) {
    auto engine = qoocore_engine_create(config);
    qoocore_engine_load_model(engine, "yolov8n.qoomodel");

    for (auto _ : state) {
        qoocore_engine_infer(engine, input);
    }
}
BENCHMARK(BM_YOLOv8_Inference)->Unit(benchmark::kMillisecond);
```

## 5. 代码审查

### 5.1 审查清单

| 检查项 | 说明 |
|--------|------|
| 设计符合架构 | 不违反分层/模块边界 |
| 接口设计合理 | API 命名、参数、错误码 |
| 异常处理完整 | 错误路径有测试覆盖 |
| 线程安全 | 共享数据有同步保护 |
| 资源管理 | RAII，无内存/文件泄漏 |
| 日志充分 | 关键路径有 INFO/ERROR 日志 |
| 测试覆盖 | 新功能有对应单元测试 |
| 文档更新 | API 变更更新设计文档 |

### 5.2 PR 模板

```markdown
## 变更说明
<!-- 简要描述变更内容 -->

## 测试
- [ ] 单元测试通过
- [ ] 集成测试通过
- [ ] 基准测试无退化 (< 5%)

## 文档
- [ ] 更新了相关设计文档
- [ ] API 变更已更新文档

## 安全检查
- [ ] 无新增安全漏洞
- [ ] 外部输入已验证
- [ ] 日志不含敏感信息
```
