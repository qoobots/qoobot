# qooeco — 具身机器人应用生态

> 机器人的"App Store"：技能市场、开发者工具、第三方算法集成。

## 定位

qooeco 是 QooBot 生态的应用市场与开发者平台，让开发者基于 qoobrain SDK
创建、分发、部署机器人技能，形成"机器人 App Store"。

## 模块

| 模块 | 说明 | 状态 |
|------|------|------|
| `skills/` | 技能库（预置技能 + 社区技能） | 📋 规划中 |
| `marketplace/` | 技能市场（发布、发现、安装） | 📋 规划中 |
| `cli/` | 开发者 CLI 工具（创建/调试/发布技能） | 📋 规划中 |
| `registry/` | 模型/技能注册中心 | 📋 规划中 |
| `playground/` | 在线沙盒（技能预览与测试） | 📋 规划中 |
| `docs/` | 开发者文档与教程 | 📋 规划中 |
| `examples/` | 示例技能代码 | 📋 规划中 |

## 技能分类

| 类别 | 示例 |
|------|------|
| 🏠 家庭 | 清洁、整理、烹饪辅助、老人看护 |
| 🏭 工业 | 装配、质检、搬运、焊接 |
| 🏥 医疗 | 手术辅助、康复训练、药品配送 |
| 🛒 零售 | 导购、理货、盘点 |
| 🌾 农业 | 采摘、巡检、喷洒 |
| 🎮 娱乐 | 下棋、舞蹈、陪伴 |

## 开发者工作流

```
qooeco CLI 创建技能 → 本地开发/调试 → 仿真验证 → 发布到市场 → 用户安装
```

## 与 qoobrain 的关系

```
qooeco (生态) ──使用──→ qoobrain SDK
                  ──发布──→ qooeco marketplace
```
