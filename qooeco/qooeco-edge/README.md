# qooeco-edge — 机器人端技能运行环境

> **运行位置**：机器人计算平台（非云端）
> **技术栈**：C++17 + Python 3.11 + CMake
>
> qooeco 的机器人端组件，负责技能的安装、运行、监控和安全管理。

## 职责

| 模块 | 说明 |
|------|------|
| 技能管理 | 安装/卸载/更新/启用/停用，`.qooskills` 包解析 |
| 沙箱隔离 | 技能运行在受限环境中，限制文件系统/网络/传感器访问 |
| 权限管控 | 声明式权限模型，用户授权管理 |
| 运行时监控 | CPU/内存/网络使用监控，异常行为告警 |
| 资源配额 | 单技能资源上限，防资源耗尽 |
| 技能间 IPC | 技能间安全通信，数据共享授权 |

## 与云端的关系

```
qooeco-cloud (云端)                    qooeco-edge (机器人端)
┌──────────────────┐                  ┌──────────────────┐
│ 技能商店后端      │  ──下载技能包──→  │ 技能安装管理      │
│ 发布/审核系统     │                  │ 沙箱运行环境      │
│ 商业化/支付       │  ←──使用统计──   │ 权限管控          │
│ 分析与运营        │                  │ 运行时监控        │
└──────────────────┘                  └──────────────────┘
```
