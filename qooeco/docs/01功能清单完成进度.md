## 定位

qooeco 是 QooBot 的**机器人 App Store**，采用**端云协同架构**，连接开发者与用户，为机器人技能提供发现、分发、商业化的完整闭环。

| 维度 | 说明 |
|------|------|
| **定位** | 机器人 App Store：连接开发者与用户，技能发现、分发、商业化 |
| **架构** | 端云协同：`qooeco-cloud`（云端商店后端）+ `qooeco-edge`（机器人端运行环境） |
| **对标** | Apple App Store（云端商店 + iOS 端 App Store 客户端） |
| **依赖** | qoodev（SDK）、qoocloud（分发/计费）、qooauth（认证/权限） |
| **被依赖** | qoocommunity |

### 端云分工

| 组件 | 运行位置 | 技术栈 | 职责 |
|------|---------|--------|------|
| qooeco-cloud | 云端 | Java 21 + Spring Cloud | 技能商店后端、发布审核、计费、分发、运营分析 |
| qooeco-edge | 机器人端 | C++17 + Python 3.11 | 技能安装管理、沙箱隔离、权限管控、运行时监控 |

---

## 设计理念

1. **开发者优先**：从 SDK 到审核到变现，全链路降低开发者摩擦
2. **信任与安全**：严格审核 + 沙箱运行，用户下载即用、无需担心安全
3. **智能推荐**：基于机器人型号、使用场景、用户偏好的个性化推荐
4. **公平变现**：透明分成、灵活定价、开发者收益可预期
