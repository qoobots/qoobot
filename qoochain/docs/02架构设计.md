# 02 — 架构设计

> 版本：v0.1 | 最后更新：2026-06-28 | 状态：Draft
>
> 子项目：qoochain — 供应链制造
>
> 架构概述：云原生 BOM/产线/标定/质检/追溯一体化平台，后端 Java 微服务 + 前端 Vue3 Web 控制台
>
> 对标系统：消费电子 NPI 流程 + 汽车行业 PPAP + SAP ME/MII 制造执行

---

## 1. 设计目标与原则

### 1.1 核心设计目标

| 目标 | P0 指标 | P1 目标 | 说明 |
|------|---------|---------|------|
| 物料可追溯 | BOM 层级完整、变更可审计 | 替代料兼容性自动验证 | 从器件批次到整机 SN 全链路 |
| 标定自动化 | 6 类传感器全自动标定 | 标定报告自动生成 | 消除人工误差 |
| 质量门禁 | IQC/IPQC/OQC 三关 | SPC 良率实时监控 | 不合格不流入下一环节 |
| 生产可制造性 | DFM/DFA 审查清单 100% 覆盖 | 防错机制 Poka-Yoke 自动校验 | 确保硬件设计可量产 |
| 数字护照 | 每台机器人出厂数字档案 | 对标 EU DPP 标准 | 合规 + 售后支撑 |

### 1.2 设计原则

1. **数据驱动制造**：所有生产决策基于实时质量数据，非人工经验
2. **标定即代码**：标定流程脚本化、版本化，可复现、可审计
3. **供应链韧性**：关键器件多源替代、长交期预警、库存策略联动
4. **合规内建**：数字护照、RoHS/REACH 物料声明内建于 BOM 体系
5. **API 优先**：所有功能提供 RESTful API，支持 ERP/MES/WMS 系统集成

### 2.2 模块职责

| 模块 | 职责 | 输入 | 输出 |
|------|------|------|------|
| **BOM 服务** | 多层 BOM 管理、物料编码、供应商管理、替代料、成本核算 | 设计 BOM、供应商报价 | EBOM→MBOM、成本分析报告 |
| **产线设计服务** | 装配流程 SOP、工位布局、DFM/DFA 审查、防错机制、产能规划 | 硬件设计文件、工艺要求 | 装配指导书、工位配置、节拍分析 |
| **出厂标定服务** | 6 类传感器/执行器全自动标定流程管理 | 标定数据、设备参数 | 标定参数文件、标定报告 |
| **质量检测服务** | IQC/IPQC/OQC 质检标准、老化测试、SPC 分析 | 检测标准、检测数据 | 质检报告、良率趋势、异常告警 |
| **生产追溯服务** | 器件批次→整机 SN 全链路追溯、版本快照 | 生产数据、批次信息 | 追溯链、数字护照 |
| **包装物流服务** | 包装设计、SN 管理、物流追踪 | 产品规格、物流数据 | 包装方案、SN 标签、物流状态 |
| **售后维修服务** | 维修手册、备件管理、RMA 流程、故障分析 | 维修记录、故障报告 | 维修工单、备件预测、设计改进建议 |
| **API 网关** | 统一认证、限流、路由、外部系统集成 | HTTP 请求 | 路由转发、鉴权结果 |

## 3. 技术选型

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| **后端框架** | Java 17 + Spring Boot 3.2 | 企业级生态、与 qoocloud/qooauth 技术栈统一 |
| **微服务治理** | Spring Cloud Gateway + Nacos | 服务发现、配置中心、动态路由 |
| **数据库** | PostgreSQL 15 + PostGIS | 强事务支持、JSONB 灵活 schema、地理空间（物流） |
| **缓存** | Redis 7 | 标定参数热缓存、会话管理 |
| **文件存储** | MinIO (S3 兼容) | 标定数据文件、检测报告、数字护照 PDF |
| **消息队列** | Apache Kafka | 生产事件流、质检数据管道、SPC 实时计算 |
| **前端框架** | TypeScript + Vue 3 + Element Plus | 与 qooweb/qoocloud 前端统一技术栈 |
| **图表可视化** | ECharts 5 | SPC 控制图、良率趋势、产能甘特图 |
| **标定算法** | Python 3.10+ (NumPy/SciPy/Ceres) | 科学计算生态、对标 qoodev 工具链 |
| **PDF 生成** | iText 7 / Puppeteer | 数字护照、质检报告、标定证书 |
| **构建工具** | Maven (Java) + Vite (Vue) | 多模块管理 + 快速 HMR |
| **容器化** | Docker Compose (dev) / Kubernetes (prod) | 统一部署运维 |

## 5. 安全架构

| 安全层 | 措施 | 说明 |
|--------|------|------|
| **认证** | JWT + qooauth OIDC 集成 | 统一身份认证，与 QooBot 生态打通 |
| **授权** | RBAC (产线操作员/工艺工程师/质量经理/管理员) | 最小权限原则 |
| **数据安全** | TLS 1.3 传输加密、敏感字段 AES-256 存储加密 | BOM 成本、供应商报价加密 |
| **审计** | 所有 BOM 变更、标定参数修改、质检判定记录操作日志 | 不可篡改审计链 |
| **API 安全** | Rate Limiting、请求签名、SQL 注入防护 | 标准 Web 安全 |
