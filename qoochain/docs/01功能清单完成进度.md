# qoochain — 供应链制造 功能清单与开发进度

> 最后更新：2026-06-28

---

## 定位

qoochain 是 QooBot 从**实验室走向产线**的量产保障体系。定义 BOM 规范、出厂标定流程、质量检测标准、生产追溯机制——确保每一台 QooBot 机器人出厂品质一致、可追溯、可维护。

| 维度 | 说明 |
|------|------|
| **定位** | 量产保障：BOM 规范、出厂标定、质量检测——让机器人从实验室走向产线 |
| **对标** | 消费电子 NPI 流程 + 汽车行业 PPAP 生产件批准程序 |
| **依赖** | qoobody（硬件参考设计） |
| **被依赖** | qoogear |

---

## 设计理念

1. **可制造性优先**：硬件设计必须经过 DFM/DFA 审查，确保可量产
2. **自动化标定**：传感器和运动学标定全自动化，消除人工误差
3. **全链路追溯**：从元器件批次到整机出厂，每步可追溯
4. **质量门禁**：关键节点设置质量检测关卡，不通过不流入下一环节
