# 03 — 交互设计

> 版本：v0.1 | 最后更新：2026-06-28 | 状态：Draft
>
> 子项目：qoochain — 供应链制造
>
> 交互概述：Web 制造控制台 + RESTful API + CLI 标定工具，面向产线操作员、工艺工程师、质量经理三类用户

---

## 1. 多入口交互矩阵

| 入口 | 目标用户 | 交互形式 | 技术栈 |
|------|----------|----------|--------|
| **Web 制造控制台** | 工艺工程师、质量经理、供应链管理 | 仪表盘、数据表格、甘特图、SPC 控制图 | Vue3 + Element Plus + ECharts |
| **RESTful API** | ERP/MES/WMS 集成、第三方系统 | JSON over HTTPS | Spring Boot REST |
| **CLI 标定工具** | 产线操作员、标定工程师 | 命令行交互式引导 | Python Click + Rich |
| **Webhook 通知** | 质量异常告警、交期预警 | HTTP POST 回调 | Spring Events + Kafka |

## 3. RESTful API

### 3.1 API 概览

| 模块 | 基础路径 | 说明 |
|------|----------|------|
| BOM 管理 | `/api/v1/bom` | 物料清单 CRUD、版本管理、EBOM→MBOM 转换 |
| 供应商管理 | `/api/v1/suppliers` | 供应商名录、审核、评级 |
| 产线设计 | `/api/v1/production-lines` | 产线定义、工位配置、SOP 管理 |
| 出厂标定 | `/api/v1/calibration` | 标定流程、参数管理、报告生成 |
| 质量检测 | `/api/v1/quality` | IQC/IPQC/OQC、SPC 数据、老化测试 |
| 生产追溯 | `/api/v1/traceability` | SN 查询、批次追溯、数字护照 |
| 包装物流 | `/api/v1/logistics` | SN 管理、包装方案、物流追踪 |
| 售后维修 | `/api/v1/aftermarket` | 维修工单、备件管理、RMA 流程 |

## 4. CLI 标定工具

### 4.1 命令体系

```bash
# 标定主命令
qoochain calibrate <type> [options]

# 标定类型
qoochain calibrate camera    --robot-id QB-001 --output ./calib/
qoochain calibrate imu       --robot-id QB-001 --duration 60s
qoochain calibrate lidar     --robot-id QB-001 --target-pattern apriltag
qoochain calibrate kinematic --robot-id QB-001 --trajectory circle
qoochain calibrate force     --robot-id QB-001 --load-weights 1kg,2kg,5kg
qoochain calibrate all       --robot-id QB-001 --output ./calib/

# 标定验证
qoochain calibrate verify    --robot-id QB-001 --report
```

## 5. 通知与告警

### 5.1 告警类型

| 告警 | 触发条件 | 通知方式 | 接收人 |
|------|----------|----------|--------|
| 来料不良超标 | IQC 批次不良率 > 阈值 | Web 通知 + 邮件 | 质量经理 |
| SPC 异常 | 连续 7 点同侧/超出控制限 | Web 通知 + Webhook | 工艺工程师 |
| 长交期预警 | 关键器件交期 > 安全库存覆盖天数 | 邮件 | 供应链管理 |
| 设备故障 | 标定台/测试台异常 | Web 通知 + 短信 | 产线主管 |
| 老化测试失败 | Burn-in 异常退出 | Web 通知 | 质量工程师 |

### 5.2 Webhook 格式

```json
{
  "event": "quality.spc_alert",
  "severity": "WARNING",
  "timestamp": "2026-06-28T10:35:00Z",
  "data": {
    "serialNumber": "QB-PRO-2026-0628-0001",
    "measurement": "bolt_torque_Nm",
    "value": 10.4,
    "limit": "LCL",
    "limitValue": 10.2,
    "rule": "接近控制下限"
  }
}
```
