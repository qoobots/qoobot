## 4. 模块职责一览

| 模块 | 技术 | 职责 | 端口 |
|------|------|------|:----:|
| `qoochain-common` | Java | 公共实体、DTO、工具类、配置 | - |
| `qoochain-gateway` | Spring Cloud Gateway | 统一入口、认证、限流、路由 | 8080 |
| `qoochain-bom` | Spring Boot + JPA | BOM 管理、物料主数据、供应商 | 8081 |
| `qoochain-line` | Spring Boot + JPA | 产线设计、工位配置、DFM 审查 | 8082 |
| `qoochain-calibration` | Spring Boot + JPA | 标定会话管理、标定报告生成 | 8083 |
| `qoochain-quality` | Spring Boot + JPA | 质检记录、老化测试、SPC 统计 | 8084 |
| `qoochain-trace` | Spring Boot + JPA | 生产追溯、数字护照 | 8085 |
| `qoochain-logistics` | Spring Boot + JPA | SN 管理、物流追踪 | 8086 |
| `qoochain-aftermarket` | Spring Boot + JPA | 维修工单、备件、维修分析 | 8087 |
| `qoochain-web` | Vue 3 + Element Plus | Web 制造控制台 | Nginx |
| `qoochain-cli` | Python + Click + Rich | CLI 标定工具 | - |

## 5. 构建与运行

### 5.1 后端启动

```bash
# 1. 启动基础设施 (PostgreSQL, Redis, MinIO, Kafka)
cd qoochain-cloud
docker-compose up -d

# 2. 编译所有模块
mvn clean package -DskipTests

# 3. 按依赖顺序启动
java -jar qoochain-gateway/target/qoochain-gateway-0.1.0.jar &
java -jar qoochain-bom/target/qoochain-bom-0.1.0.jar &
java -jar qoochain-line/target/qoochain-line-0.1.0.jar &
java -jar qoochain-calibration/target/qoochain-calibration-0.1.0.jar &
java -jar qoochain-quality/target/qoochain-quality-0.1.0.jar &
java -jar qoochain-trace/target/qoochain-trace-0.1.0.jar &
java -jar qoochain-logistics/target/qoochain-logistics-0.1.0.jar &
java -jar qoochain-aftermarket/target/qoochain-aftermarket-0.1.0.jar &
```

### 5.2 前端启动

```bash
cd qoochain-web
npm install
npm run dev          # 开发模式 http://localhost:5173
```

### 5.3 CLI 工具安装

```bash
cd qoochain-cli
pip install -e .
qoochain --help
```
