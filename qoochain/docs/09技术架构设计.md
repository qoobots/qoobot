## 2. 标定算法

### 2.1 传感器标定算法选型

| 标定项 | 算法 | 精度目标 |
|:-------|------|:------:|
| 相机内参 | Zhang's (OpenCV calibrateCamera) | 重投影误差 <0.3px |
| 立体相机外参 | 双目立体标定 (OpenCV stereoCalibrate) | 外参误差 <1mm |
| LiDAR-相机外参 | PnP + ICP 联合优化 | 旋转 <0.5°, 平移 <2cm |
| IMU 标定 | Allan Variance + 多姿态 | 零偏稳定性 <10°/h |
| 运动学标定 | 闭环运动学链 + Ceres 优化 | 末端定位误差 <2mm |
| 力传感器标定 | 最小二乘 + 温度补偿 | 线性度 <0.5% FS |

### 2.2 运动学标定示例

```python
class KinematicCalibrator:
    """基于 Ceres Solver 的运动学参数标定"""
    
    def calibrate(self, measurements: List[JointMeasurement]) -> KinematicParams:
        problem = ceres.Problem()
        
        # 待优化参数: 连杆长度、关节零位、减速比
        params = KinematicParams.initial_guess()
        
        for m in measurements:
            # 残差: 正运动学(fk)预测末端位置 vs 实测末端位置
            cost = KinematicResidual(m.joint_angles, m.measured_pose)
            problem.add_residual_block(cost, None, params.data())
        
        ceres.solve(options, problem, params.data())
        return params
```

## 3. SPC 统计过程控制

### 3.1 控制图算法

```
X-bar 控制图:
  UCL = X̄ + A₂ × R̄
  CL  = X̄
  LCL = X̄ - A₂ × R̄

R 控制图:
  UCL = D₄ × R̄
  CL  = R̄
  LCL = D₃ × R̄

CPK 过程能力指数:
  CPK = min((USL - X̄) / 3σ, (X̄ - LSL) / 3σ)
```

### 3.2 告警规则

| 规则 | 条件 | 说明 |
|:-----|------|------|
| 超出控制限 | 任一点超出 UCL/LCL | 立即停线 |
| 连续偏移 | 连续 7 点在中心线同一侧 | 趋势告警 |
| 趋势 | 连续 7 点单调递增/递减 | 漂移告警 |
| 周期 | 数据呈规律性周期波动 | 系统因素排查 |

```sql
-- 查询某批器件的所有整机 SN
SELECT a.serial_no, c.part_number, c.lot_number
FROM assembly a
JOIN assembly_component ac ON a.id = ac.assembly_id
JOIN component c ON ac.component_id = c.id
WHERE c.lot_number = 'LOT20260315';

-- 查询某台机器人的完整制造档案
SELECT * FROM assembly WHERE serial_no = 'QS-2026-00001';
```
