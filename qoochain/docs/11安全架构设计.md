# qoochain — 安全架构设计

> 版本：v0.1 | 最后更新：2026-06-29 | 状态：Draft | 子项目：qoochain（供应链制造）

---

## 1. 安全设计原则

1. **产线隔离**：产线网络与办公网络/互联网隔离
2. **标定安全**：标定参数完整性校验，防止篡改
3. **追溯防伪**：SN 唯一性、数字护照防伪
4. **数据保护**：供应商信息、成本数据分级管理

---

## 2. 网络安全

```
互联网
  │
  ▼
防火墙
  │
  ├──▶ 办公网络 (管理后台、qoochain API)
  │      │
  │      └──▶ 产线网关 (单向、仅允许特定端口)
  │              │
  │              ▼
  └────────── 产线 VLAN
               ├── 标定工作站
               ├── MES 终端
               └── 检测设备
```

---

## 3. 标定数据安全

### 3.1 完整性保护

```
标定参数生成
    │
    ▼
SHA-256 哈希计算
    │
    ▼
Ed25519 签名 (产线密钥)
    │
    ├── 写入机器人本地 (签名附加)
    └── 上传云端 (签名 + 哈希)
    
机器人启动时:
  加载标定参数 → 验证签名 → 通过 → 使用
                            → 失败 → 告警 + 使用出厂默认
```

### 3.2 防篡改

- 标定参数写入后标记只读
- 云端标定数据不可删除（仅追加新版本）
- 任何标定修改操作记录审计日志

---

## 4. 追溯安全

### 4.1 SN 管理

- SN 生成算法：型号代码 + 出厂日期 + 流水号 + 校验码
- SN 中央分配，防止重复/伪造
- 二维码/Data Matrix 编码（含签名信息）

### 4.2 数字护照防伪

- 数字护照 PDF 包含 QR 码，编码验证链接
- 在线验证：扫描 QR → qoobot.ai/verify/QS-2026-00001
- 验证结果：SN 有效 + 出厂日期 + 保修状态

---

## 5. 访问控制

| 角色 | 权限 |
|:-----|------|
| 产线操作员 | 仅工序扫码过站、标定操作 |
| 质量工程师 | 检验记录、SPC 查看 |
| 采购经理 | 供应商管理、BOM 成本查看 |
| 制造工程师 | 全部制造数据读写 |
| 管理员 | 全部权限 + 审计日志 |

---

## 6. 供应商数据保护

- 供应商报价、成本数据加密存储
- 供应商审计结果分级（供应商自身不可见）
- 供应商间数据严格隔离

---

## 7. 审计

| 审计项 | 内容 |
|:-------|------|
| BOM 变更 | 谁/何时/改了什么/原因 |
| 标定操作 | 操作员/SN/结果/时间 |
| 检验记录 | 检验员/结果/NCMR |
| SN 生成 | 生成时间/分配目标 |
| 供应商变更 | 新增/状态变更/审核 |
